1. まずは絞り込み検索の実例

ファセット (都道府県 + 対象業種) で絞り込みながら制度を取れます。これが一番よく使われる形です。

curl -H "X-API-Key: YOUR_KEY" \
  "https://api.autonomath.ai/v1/programs/search?prefecture=東京都&target_types=smb&limit=10"

2. Claude Desktop から MCP 経由で呼ぶ

LLM クライアントから直接叩きたい場合は、claude_desktop_config.json に以下を追記するだけで、6 つのツール (programs.search / programs.get / exclusions.check ほか) がそのまま使えます。

{
  "mcpServers": {
    "autonomath": {
      "command": "npx",
      "args": ["-y", "@autonomath/mcp"],
      "env": { "AUTONOMATH_API_KEY": "YOUR_KEY" }
    }
  }
}

3. 複数制度を1リクエストで取得 (batch)

unified_id の配列を POST すれば一回で取れます。アプリの初期化時などに便利です。

curl -X POST -H "X-API-Key: YOUR_KEY" -H "Content-Type: application/json" \
  "https://api.autonomath.ai/v1/programs/batch" \
  -d '{"unified_ids":["UNI-14e57fbf79","UNI-40bc849d45","UNI-08d8284aae"]}'

よくある質問

  • 認証ヘッダ: X-API-Key: YOUR_KEY のほか、Authorization: Bearer YOUR_KEY も受け付けます。
  • レート制限: 認証済みキー (Paid) は Stripe 従量課金・上限なし (月次 UTC 0 時基準でメーター集計)。匿名は 50 req/月 per IP・JST 月初 00:00 リセット。超過時は 429 + Retry-After ヘッダを返します。
  • 請求・鍵ローテート: Stripe カスタマーポータルからいつでもカード変更・利用停止・鍵の再発行ができます。

まだ試していない場合は、/v1/ping で最小疎通確認から始められます。1リクエストだけなら10秒かかりません。
