D. 直接 curl / SDK

MCP を経由せず REST で叩く場合の最小例:

curl -H "X-API-Key: YOUR_KEY" \
  "https://api.jpcite.com/v1/programs/search?prefecture=東京都&target_type=中小企業&limit=10"

よくある詰まりどころ

  • 401 Unauthorized — ヘッダ名は X-API-Key です(Authorization: Bearer YOUR_KEY でも受けます)。Api-Key や X-Api-Token は通りません。
  • 429 Too Many Requests — Retry-After ヘッダの秒数を待ってください。Paid キーは月次課金で上限なし、匿名は 3 req/日 (JST 翌日 00:00 リセット) です。
  • FTS の単一漢字ノイズ — q=税額控除 は「税」だけ含む行も拾います。2文字以上の熟語は q="税額控除" のように引用符で囲ってください。
  • MCP が起動しない — uvx autonomath-mcp は初回 30 秒程度かかります。タイムアウトを 60s に伸ばしてください。
  • 環境変数が読まれない — Claude Desktop は config.json の env ブロックのみ読みます。シェルの export は無視されます。

それでも止まっていれば、このメールに「こう書いたがこう返る」の1行だけでも返信ください。人間が読みます。
