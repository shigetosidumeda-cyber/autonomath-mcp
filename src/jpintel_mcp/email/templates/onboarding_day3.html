MCP クライアント経由で使う

Claude Desktop など MCP 対応クライアントの設定ファイルに以下を追加すると、そのまま LLM からツールとして呼べます。

{
  "mcpServers": {
    "autonomath": {
      "command": "npx",
      "args": ["-y", "@autonomath/mcp"],
      "env": { "AUTONOMATH_API_KEY": "YOUR_KEY" }
    }
  }
}

詰まったら、このメールにそのまま返信してください。人間が読みます。
