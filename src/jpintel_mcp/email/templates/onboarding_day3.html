1. search_programs — 制度の入口

10,790 件の補助金 / 融資 / 税制 / 認定を、都道府県・対象業種・予算上限などで絞り込みます。何から始めるか迷ったらこれです。

curl -H "X-API-Key: jc_xxx" \
  "https://api.jpcite.com/v1/programs/search?prefecture=東京都&target_type=中小企業&amount_min=100&limit=10"

2. search_tax_incentives — 税制措置の専用検索

国税庁・財務省 由来の税制措置 (賃上げ促進税制 / 中小企業経営強化税制 / 事業承継税制 ほか) を横断検索します。出力は税務助言ではなく公開ルール検索です(税理士法 §52)。

curl -H "X-API-Key: jc_xxx" \
  "https://api.jpcite.com/v1/am/tax_incentives?q=賃上げ&limit=5"

3. check_invoice_registrant — 適格事業者番号の照合

法人番号 (T+13 桁) を渡すと、国税庁 適格請求書発行事業者公表サイト 由来の登録状況・氏名・本店所在地を返します。経費精算 / 仕訳の自動化に。

curl -H "X-API-Key: jc_xxx" \
  "https://api.jpcite.com/v1/invoice_registrants/T8010001213708"

4. dd_profile_am — 取引先 360° プロファイル

法人番号 1 つから、補助金採択 / 行政処分 / 適格請求書発行登録 / 業歴を 1 リクエストでまとめて取得します。デューデリジェンスの 1 次スクリーニングに。

curl -X POST -H "X-API-Key: jc_xxx" -H "Content-Type: application/json" \
  "https://api.jpcite.com/v1/am/dd_batch" \
  -d '{"houjin_bangou":["T8010001213708"]}'

5. regulatory_prep_pack — 申請前の自動チェック

unified_id を渡すと、その制度に対する排他ルール・前提認定・直近の改正点をまとめて返します。「この補助金、今この会社で出して大丈夫?」を 1 リクエストで。

MCP tool:
regulatory_prep_pack(unified_id="UNI-14e57fbf79")
