# 表示と免責

jpcite は、日本の補助金・助成金・融資制度・税制等について、一次資料へのリンクと整理済み metadata を提供するデータサービスです。

本サービスは、制度情報を探しやすくするためのものであり、受給・採択・融資実行・税務上の取扱いを保証するものではありません。申請可否や最新の要件は、必ず各制度の所管機関または資格を有する専門家に確認してください。

## jpcite が提供するもの

- 公式資料へのリンク
- 制度名、所管、対象者、地域、締切、上限額などの整理済み metadata
- API / MCP から機械的に参照しやすい response
- 出典 URL と取得時点を含む根拠情報

## jpcite が提供しないもの

- 申請書の作成代行
- 受給可否の最終判断
- 法律・税務・行政手続に関する個別助言
- 採択、受給、融資実行の保証

## 表現について

jpcite の情報を顧客提案、社内資料、AI の回答などで再利用する場合は、断定的な表現を避けてください。

避けるべき表現:

- 受給や採択を保証する表現
- 制度利用を確定事項のように見せる表現
- 審査や支給の結果を断定する表現
- 申請すれば誰でも受け取れるように読める表現

推奨する表現:

- 「対象となる可能性がある制度」
- 「公表資料上の交付上限」
- 「一次資料では次の要件が示されています」
- 「申請可否は所管機関に確認してください」

## 出典確認

制度情報は変更されることがあります。重要な判断に使う場合は、API response に含まれる `source_url` から一次資料を確認してください。
