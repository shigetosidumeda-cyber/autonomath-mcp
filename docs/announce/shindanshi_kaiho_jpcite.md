# 認定支援機関業務の月次伴走を月額 ¥3,300 で支援 ——「jpcite」recipe r07 が拓く診断士業務の生産性

中小企業診断士・認定支援機関が担う月次伴走業務において、公的情報を機械整形して artifact 化する API・MCP サーバ「jpcite」が、月額 ¥3,300 で 50 社の月次フォローを回す環境を提供している。recipe r07 は中小企業支援法・税理士法 §52 の 2 業法 fence を組み込み、診断士業務の価値創出領域を侵食しない設計を貫いている。

## 認定支援機関業務の構造的課題

認定支援機関は、中小企業の経営革新・事業承継・補助金活用・金融支援を一気通貫で支援する公的伴走業務の担い手である。中小企業庁の認定制度のもと、税理士・診断士・金融機関・行政書士など複数の専門職が認定を受けて従事している。

月次伴走業務の典型的な工程は次のとおり。

- 顧客企業のセグメント別補助金候補抽出 (月次)
- 制度融資・公的保証の maturity 棚卸し
- 経営改善計画の進捗フォロー
- 業法改正・補助金公募スケジュールの monitoring
- 認定支援機関業務報告 (中小機構提出)

50 社を担当する診断士事務所であれば、月次伴走の事前準備だけで月 80〜120 時間が消費される。診断士本人が補助スタッフを抱えていないケースが多く、人手依存が生産性の天井になっている。

## recipe r07 の入出力

jpcite は Bookyou 株式会社 (T8010001213708) が運営する公的情報 API・MCP サーバである。recipe r07「認定支援機関 月次伴走」は、顧客企業の業種・所在地・資本規模・直近採択履歴を input として、月次伴走に必要な artifact を一括出力する。

artifact の構成:

- 顧客別補助金 candidate (中小企業庁・経産省・各都道府県横断)
- 制度融資 maturity 棚卸し (公庫・保証協会・自治体)
- 経営改善計画進捗 KPI suggestion (公開ガイドラインベース)
- 直近 30 日の業法改正・公募追加 alert
- 認定支援機関業務報告 draft (公開フォーマットに沿う)

出力はすべて構造化 JSON + 出典 URL であり、Markdown / Slack / メール配信フォーマットに自動変換できる。

## 2 業法 fence の設計

認定支援機関業務は、中小企業支援法に基づく公的伴走と税理士法 §52 (税務代理・税務相談独占) の境界が常に問題となる。jpcite は次の二層で防護している。

第一に、recipe r07 の出力は「公開情報の機械整形 + 候補リスト」に限定し、特定顧客の税務判断・申告書作成・税務代理を一切行わない。これにより税理士法 §52 fence を侵食しない。

第二に、recipe r07 は中小企業支援法の建付けに沿う「経営革新等支援機関業務の補助情報」として位置付けられ、認定支援機関が自身の業務遂行のために用いることを前提とする。出力には診断士法人・税理士法人・金融機関などの認定支援機関が「自身の業務遂行に用いる」旨を ToS で明示している。

すべての API response に `X-Jpcite-Disclaimer` header と `_disclaimer` field が付与され、「公開情報の機械整形であり、税務判断・経営助言の最終責任は資格専門家にあること」を明示する。SDK / MCP server は disclaimer chain を再配布事業者にも継承させる。

## ROI 試算

50 社を月次伴走する診断士事務所のケース。

- 従来: 月次伴走事前準備 月 100 時間 × 時給換算 ¥7,000 = 月額 ¥700,000
- jpcite 導入後: API ¥3,300 + 確認・統合 30 時間 × ¥7,000 = 月額 ¥213,300

月額 ¥486,700 のキャッシュ改善、年間 ¥5.84 百万円。さらに事前準備時間が圧縮されることで、診断士本人が顧客対話・経営助言・現場ヒアリングに割ける時間が増え、顧客満足度と更新率の向上が期待できる。

## 認定支援機関業務の中核は変わらない

jpcite が自動化するのは、月次伴走業務の前処理・公開情報整理の部分である。経営助言・現場ヒアリング・改善計画策定・金融機関との折衝・補助金申請ハンズオン支援といった、認定支援機関業務の本質的な価値創出は変わらず診断士・税理士・専門家が担う。

むしろ前処理を圧縮することで、顧客一人あたりに割ける対話時間が増え、case あたりの単価上昇や付加サービスの拡張に繋がる。AI を「業務を奪う存在」ではなく「事前整理を肩代わりする存在」と位置付ければ、診断士業務の生産性は確実に向上する。
