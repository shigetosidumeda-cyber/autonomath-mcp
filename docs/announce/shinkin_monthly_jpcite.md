# 信用金庫の取引先支援業務における公的情報 retrieval を artifact API で生産性 10 倍に ——「jpcite」が拓く信用金庫渉外の月次伴走

信用金庫の渉外担当者にとって、取引先中小企業の支援業務 (補助金提案・制度融資組成・事業性評価伴走) に要する公開情報の retrieval は、月次 review で 1 担当あたり 20〜40 時間に達する。jpcite は、これらの検索作業を公開情報の機械整形 artifact として提供する API・MCP サーバである。中小企業診断士法・社会保険労務士法・税理士法 §52 fence を組み込み、士業の最終判断業務を侵食しない設計を貫いている。

## 信用金庫渉外の月次工程の現状

信用金庫の取引先支援において、渉外担当者は次の公開情報を毎月 retrieval する必要がある。

- 中小企業庁・経産省・各府省 補助金公募情報 (11,601 制度)
- 各都道府県の制度融資 (108 制度 担保・個人保証・第三者保証 三軸)
- e-Gov 法令改正 (税法・会社法・労働法・業法)
- 各府省 通達・FAQ・QA
- 採択事例 (2,286 採択公開分)
- 行政処分 (1,185 件)
- 適格事業者 (インボイス) 登録状況

これらは渉外担当者本人が各府省サイトを横断する手作業に依存しており、1 取引先あたり 30 分〜1 時間が retrieval に消費される。30〜60 社を担当する担当者であれば、月次 review だけで 20〜40 時間の検索コストが発生している。

地域金融機関による「事業性評価」と「伴走支援」は金融庁が業界基準として推進しており、検索コストの圧縮は業界共通課題となっている。

## jpcite が提供する artifact レイヤ

jpcite は、国・地方自治体・独立行政法人の公開情報を機械可読 JSON / Markdown に整形し、API と MCP (Model Context Protocol) サーバ経由で提供する Bookyou 株式会社のプロダクトである。

主な対象データセット:

- 11,601 補助金 (中小企業庁・経産省・各府省) tier S/A/B/C 整形
- 108 融資 (担保・個人保証人・第三者保証人 三軸分解)
- 9,484 法令 (e-Gov CC-BY-4.0、6,493 全文索引済)
- 2,286 採択事例
- 1,185 行政処分
- 13,801 適格事業者 (PDL v1.0)

artifact の構成は recipe 単位で提供される。信用金庫の取引先支援向け recipe としては以下が主要である。

- r01「顧問先 (取引先) 月次補助金抽出」(担当 30〜60 社の月次 fan-out)
- r02「制度融資組成 candidate 抽出」(担保・保証人三軸の組合せ最適化)
- r05「事業性評価 伴走 monitoring」(法改正 + 行政処分 + 採択履歴 横断)

API 価格は ¥3/request (税込 ¥3.30) 完全従量、daily 3 request 無料、月額固定費なし、解約 1 click。MCP server 経由であれば Claude Desktop や Cursor IDE から自然言語で artifact を引き出せる。

## 業法 fence の設計

最も重要なのは、jpcite が士業業務 (税理士法 §52 / 中小企業診断士業務 / 社会保険労務士業務) を侵食しない設計を貫いている点である。

信用金庫の渉外担当者が取引先に提供する伴走サービスは、士業領域に踏み込みやすい。jpcite はこの fence を次の三層で防護している。

第一に、artifact はあくまで「公開情報の機械整形」であり、特定取引先の事実関係を前提とした税務判断・経営助言・労務助言を提供しない。recipe 出力は常に「該当しうる候補リスト」+ 出典 URL であり、結論ではない。

第二に、すべての API response に disclaimer header (`X-Jpcite-Disclaimer: machine-formatted public information; final judgment by licensed professional`) が付与される。SDK / MCP server も同 disclaimer を必ず表示する。

第三に、信用金庫向け recipe (r01 月次補助金抽出、r05 事業性評価伴走) は、渉外担当者が「取引先紹介・士業連携の前段」として artifact を用いる前提で設計されており、士業の領域に踏み込む利用形態を ToS で禁止している。

## 節約額試算 (旧称 ROI 試算)

担当 30〜60 社、月次 review を実施する渉外担当者 5 名の支店ケースで試算する。

- 従来: 担当 50 社平均 × 月次 30 分/社 × 5 名 = 月 125 時間 × 担当者時給 ¥4,000 = 月額 ¥500,000
- jpcite 導入後: API 約 ¥4,950 (1,650 request 想定) + 担当者最終確認 5 分/社 × 250 社 = 月 20.8 時間 × ¥4,000 = 月額 ¥88,150

差額 ¥411,850/月、年間 ¥4.94 百万円のキャッシュ改善。さらに浮いた時間を新規取引先開拓・既存先深耕に再配置すれば、業務粗利の拡大が見込める。

recipe r01「取引先月次補助金抽出」は https://jpcite.com/docs/recipes/r01 で実装手順と recipe payload を公開している。
