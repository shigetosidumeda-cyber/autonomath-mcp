# 弁護士業務における公的情報 retrieval を 業法 fence 付き artifact API で 自動化する ——「jpcite」が拓く弁護士事務所の生産性

弁護士事務所において、案件 1 件あたりの 公的情報 retrieval (法令検索・許認可確認・行政処分照会・業法改正追跡) に要する時間は、ベテランで 1〜2 時間、新人で 3〜5 時間とされる。jpcite は、これらの検索作業を公開情報の機械整形 artifact として提供する API・MCP サーバである。弁護士法 §72 fence を組み込み、最終判断業務を侵食しない設計を貫いている。

## 弁護士事務所の検索作業の現状

弁護士業務において、案件着手前に必要となる公開情報の retrieval は次の通り。

- e-Gov 法令本文・施行日・改正履歴 (税法・会社法・労働法・業法 等)
- 各府省 通達・FAQ・QA (国税庁・厚労省・経産省・金融庁 等)
- 業種別の許認可保有 / 行政処分履歴 (各省庁公開 DB)
- 適格事業者 (インボイス) 登録状況 (国税庁 PDL v1.0)
- 補助金採択履歴 (中小企業庁・各都道府県)
- 入札参加資格・指名停止履歴

これらは弁護士本人 or パラリーガルが手作業で各府省サイトを横断する必要があり、ベテラン弁護士であっても案件あたり 1〜2 時間の時間が消費される。新人やパラリーガル教育期間中は 3〜5 時間に膨らむ。中堅事務所で月 30 案件を扱うケースで月 30〜100 時間相当の検索コストが発生している。

## jpcite が提供する artifact レイヤ

jpcite は、国・地方自治体・独立行政法人の公開情報を機械可読 JSON / Markdown に整形し、API と MCP (Model Context Protocol) サーバ経由で提供する Bookyou 株式会社のプロダクトである。

主な対象データセット:

- 11,601 補助金 (中小企業庁・経産省・各府省) tier S/A/B/C 整形
- 9,484 法令 (e-Gov CC-BY-4.0、6,493 全文索引済)
- 2,065 裁判例 (最高裁判所 公開分)
- 1,185 行政処分
- 13,801 適格事業者 (PDL v1.0)
- 50 税制 ruleset (措置法 + 通達 整形)
- 362 入札公示

artifact の構成は recipe 単位で提供される。弁護士業務向け recipe としては以下が主要である。

- r02「許認可・行政処分の機械整形 artifact」(弁護士法 §72 fence 内)
- r03「中小 M&A の公開情報 DD」(弁護士法 §72 + 税理士法 §52 + 金商法 §29 fence)
- r05「会社法・労働法 改正 monitoring」(直近 36 ヶ月 横断)

API 価格は ¥3/request (税込 ¥3.30) 完全従量、daily 3 request 無料、月額固定費なし、解約 1 click。MCP server 経由であれば Claude Desktop や Cursor IDE から自然言語で artifact を引き出せる。

## 弁護士法 §72 fence の設計

最も重要なのは、jpcite が「弁護士業務」を侵食しない設計を貫いている点である。

弁護士法 §72 は、弁護士でない者が報酬を得る目的で法律事件に関して鑑定・代理・仲裁・和解その他の法律事務を取り扱うこと等を禁じている。AI ベンダーが個別案件の法律判断を提供することは違法となる。jpcite はこの fence を次の三層で防護している。

第一に、artifact はあくまで「公開情報の機械整形」であり、特定案件の事実関係を前提とした法的判断・助言・代理・書面作成を提供しない。recipe 出力は常に「該当しうる候補リスト」+ 出典 URL であり、結論ではない。

第二に、すべての API response に disclaimer header (`X-Jpcite-Disclaimer: machine-formatted public information; final judgment by licensed professional`) が付与される。SDK / MCP server も同 disclaimer を必ず表示する。

第三に、弁護士業界向け recipe (r02 許認可整形、r03 M&A DD) は、弁護士が「自身の業務遂行のために」用いる前提で設計されており、無資格者が直接顧客に提供する利用形態を ToS で禁止している。

## 節約額試算 (旧称 ROI 試算)

月 30 案件を扱う中堅事務所のケースで試算する。

- 従来: 案件あたり検索 1.5 時間 × 30 件 × 弁護士時給 ¥15,000 = 月次 API fee delta
- jpcite 導入後: API 約 ¥2,970 (990 request 想定) + 弁護士最終確認 20 分/件 × 30 件 × ¥15,000 = 月次 API fee delta

月次 API fee delta、年間参考額のAPI fee delta reference。人員再配置・業務成果・専門判断の価値は本比較に含みません。

recipe r02「許認可・行政処分の機械整形 artifact」は https://jpcite.com/docs/recipes/r02 で実装手順と recipe payload を公開している。
