# 正直な能力と限界 (Honest Capabilities & Limitations)

このページは 税務会計AI が「何をするサービスで、何をしないサービスか」を 60 秒で理解できるよう、マーケティング表現を排して書かれた一次情報です。営業文句ではなく、契約前に読む仕様書として扱ってください。

---

## 1. これは何か (What this is)

税務会計AI は、日本の制度情報を一次資料 URL 付きで横断検索できる **検索インデックス API** です。

| データ種別 | 件数 |
|---|---|
| 補助金 / 助成金 | 11,684 |
| 法令 (本文収録、 全文検索対象) | 154 |
| 法令 (メタデータ catalog stubs、 法令名 resolver 用) | 9,484 |
| 判例 | 2,065 |
| 税制ルールセット | 50 |
| 適格請求書発行事業者 | 13,801 |
| 採択事例 | 2,286 |
| 行政処分記録 | 1,185 |

- 検索結果は、マッチしたレコードと**一次資料 URL** を返します
- 提供形態: REST API + MCP server
- 課金: **¥3 / req メータード**、匿名 50 req/月まで無料
- ティア無し、定額プラン無し、解約手続き不要

---

## 2. これは何ではないか (What this isn't)

以下は法令上 **有資格者の独占業務** であり、税務会計AI では提供しません。

- **法律相談** — 弁護士法 § 72 該当業務
- **税務相談** — 税理士法 § 52 該当業務
- **行政書士業務代行** — 行政書士法 § 1 該当業務
- **労務相談 / 36 協定生成代行** — 社会保険労務士法該当業務 (default off feature)
- **申請書類の最終チェック / 提出代行** — 提供しません
- **リアルタイム改正追跡** — point-in-time snapshot を保持しており、historical diff は部分的にしか持っていません
- **各レコードの正確性 100% 保証** — 一次資料の再配布であり、内容の検証はしていません

---

## 3. 典型的な使い方 (Typical use cases that work well)

以下は 税務会計AI が設計上得意としているユースケースです。

- LLM agent が地域 + 業種で日本の補助金を引く
- SaaS プラグインが該当する法令 / 規制を表示する
- 研究者が過去の採択事例を query する
- 法人番号で 行政処分歴のある法人を bulk lookup する

---

## 4. 使うべきでないケース (When NOT to use)

以下は **必ず有資格者に依頼してください**。税務会計AI は補助ツールに過ぎません。

- リアルタイム法令コンプライアンスチェック → **弁護士**
- 税務申告判断 → **税理士**
- 申請書の提出 → **行政書士**
- 監査グレードの一次資料 chain-of-custody → 提供しません (URL は返しますが、改ざん検知付き chain は提供しません)

---

## 5. 既知の限界 (Known limitations)

技術的限界を隠さず開示します。これらは**継続的に改善されますが、現時点では既知の制約**です。

- **全文検索インデックス (3-gram) の単漢字 overlap**: `税` のような単漢字 query が複数フィールドにまたがって過剰マッチする
- **制度時系列 snapshot**: 82% の row が `eligibility_hash` 空。historical diff は部分的
- **制度併用判定 matrix**: 48,815 entries のうち、`source_url` を持つのは 9% のみ (残りは heuristic 由来)
- **公開保留レコード**: 二次レビュー待ちのため検索結果から除外されています
- **法令条文**: incremental load 中 (現在 法令本文 154 件 + 法令メタデータ 9,484 件、 全文検索可能なのは 154 件のみ、 残りは法令名 / 法令番号 / e-Gov リンクの resolver として参照可能)

---

## 6. 何を保証するか / しないか (Guarantees and non-guarantees)

### 保証する (Guarantee)

- **API 稼働率** — SLA 記載の通り
- **一次資料 URL の返却** — レコードに source_url が存在する場合は必ず返す
- **¥3 / req 課金の正確性** — 計測精度を保証

### 保証しない (Non-guarantee)

- **任意のレコードが現時点で有効な法令であること** — 必ず一次資料を verify してください
- **すべての関連レコードが surface されること** — niche query では recall < 100% です
