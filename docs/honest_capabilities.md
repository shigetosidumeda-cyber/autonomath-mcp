# 能力と限界 (透明性ページ)

マーケティング表現を排した仕様書。契約前に読み、scope の境界を明確にしてください。

## 1. これは何か

日本の制度情報を一次資料 URL 付きで横断検索する **検索インデックス API**。

| データ種別 | 検索対象 件数 |
|---|---|
| 補助金 / 助成金 / 認定 | 11,684 (tier S/A/B/C) |
| 採択事例 | 2,286 |
| 融資 (担保 / 個人保証人 / 第三者保証人 三軸) | 108 |
| 行政処分 | 1,185 |
| 法令本文 (全文検索対象) | 154 |
| 法令メタデータ (resolver) | 9,484 |
| 判例 | 2,065 |
| 税制ルールセット | 50 |
| 適格請求書発行事業者 | 13,801 |

提供形態: REST API + MCP server。料金: ¥3/req 従量、匿名 3 req/日 無料。tier プラン無し、契約無し ([pricing.md](./pricing.md))。

## 2. これは何ではないか (専門業務、提供しない)

- **法律相談**
- **税務相談**
- **行政書士業務代行**
- **労務相談 / 36協定生成** — 社会保険労務士法 該当業務 (default off)
- **申請書類の最終チェック / 提出代行**
- **リアルタイム改正追跡** — ある時点での記録のみ、過去差分の追跡は部分的
- **各レコードの完全性・正確性保証** — 制度ごとの最終判断や一次資料の再配布は提供しません

## 3. 設計上得意なユースケース

- LLM agent が地域 + 業種で日本の補助金を引く
- SaaS プラグインが該当する法令 / 規制を表示
- 研究者が過去の採択事例を query
- 法人番号で 行政処分歴のある法人を bulk lookup

## 4. 使うべきでないケース (有資格者へ依頼)

- リアルタイム法令コンプライアンスチェック → 弁護士
- 税務申告判断 → 税理士
- 申請書提出 → 行政書士
- 監査グレードの一次資料 chain-of-custody → 提供しない

## 5. 既知の限界

- **全文検索の挙動** — 3-gram トークナイザーのため、単漢字 query (例: `税`) は複数フィールドに広く一致する。2 文字以上を推奨
- **制度時系列 (historical diff)** — 一部の row でのみ厳密な差分が確定。改正トラッキングは継続改善中
- **制度併用チェック** — 登録済みルールに一致する論点を返す。未登録の組み合わせは安全保証にならない
- **検索除外プール** — 検索インデックスは tier S/A/B/C のみ収録。tier C 未満は二次レビュー後に編入する運用
- **法令条文** — e-Gov 全文ロード継続中 (本文 154 件 + メタデータ 9,484 件、resolver で全件にリーチ可能)

## 6. 保証する / しない

### 保証する

- **API 稼働率** — [sla.md](./sla.md) 記載の通り
- **出典 URL 返却** — `source_url` が登録済みのレコードではレスポンスに含める
- **¥3/req 課金の正確性**

### 保証しない

- 任意のレコードが現時点で有効な法令であること
- ニッチな検索語に対する完全な recall
