jpcite Pricing — 4-tier outcome band

jpcite は 完全従量・per-call metered です。subscription / seat / 年契約 / 最低額はありません。価格は呼出 1 回 (1 billable call) の「複雑度バンド」で決まります。

4 tier の価格表

Tier 名前 1 call の価格 (税別) 典型 例

B Bundle ¥10 3–5 atomic を server-side で合成した bundle match_due_diligence_questions, cross_check_jurisdiction
C Composed ¥30 7–13 atomic を rule_tree で合成 (1 recipe walk) recipe_corporate_setup_registration, recipe_tax_year_end_adjustment

Tier A — Atomic (¥3 / call)

jpcite の baseline。1 atomic 呼出 = 1 billable unit = ¥3 (税込 ¥3.30)。匿名 IP には 3 req/日の Free quota が付き、JST 翌日 00:00 にリセットされます。

Tier B — Bundle (¥10 / call)

3–5 atomic 呼出を server-side で合成し、agent の往復回数を削減します。feedback_composable_tools_pattern 準拠。Stripe 換算 ¥3 × 4 units。

Tier C — Composed (¥30 / call)

7–13 atomic 呼出を rule_tree で合成。1 recipe walk = 1 composed call。feedback_rule_tree_branching 準拠。Stripe 換算 ¥3 × 10 units。

Tier D — Pack (¥100..¥1000 / call)

完成 pack (full deliverable)。複数 scaffold 文書 + filing windows + placeholders を 1 envelope で返却。価格は pack ごとに band 内で固定。

Payment rails

  • Stripe metered (default) — 月次自動課金、適格請求書発行事業者 T8010001213708 (Bookyou株式会社)。
  • x402 micropayment — HTTP 402 + USDC on Base / Polygon. agent 経由でカードレス。price_yen は tier から自動計算。
  • Credit Wallet — 前払い + auto-topup, 50/80/100% throttle alert.
