## §2. P1.3 — REST meta fail-closed

The pre-fix `src/jpintel_mcp/api/meta.py` lines around 152-160 and 175-180
imported hallucinated `list_autonomath_resources` / `list_autonomath_prompts`
symbols. The actual canonical exports in
`src/jpintel_mcp/mcp/autonomath_tools/{resources,prompts}.py` are
`list_resources()` and `list_prompts()`. The wrapping `except Exception:
pass` silently swallowed the `ImportError`, so `/v1/meta/resources` and
`/v1/meta/prompts` returned 0 autonomath entries from launch through
2026-05-17 — a textbook fail-open drift.

Fix (this commit):

- `meta.py` rebinds via `from ...resources import list_resources as
  list_autonomath_resources` (same for prompts) so the local identifier
  is unchanged and downstream `.extend(list_autonomath_resources())`
  continues working.
- Outer `Exception` swallow stays — it is the right posture for
  registry-import resilience, but the import line itself must point at a
  real symbol so the swallow never fires on the happy path.
- New `tests/test_meta_import_drift.py` pins 5 invariants:
  1. `autonomath_tools.resources.list_resources` exists + returns non-empty.
  2. `autonomath_tools.prompts.list_prompts` exists + returns non-empty.
  3. `api/meta.py` source carries the `... as list_autonomath_resources` /
     `... as list_autonomath_prompts` alias forms.
  4. `_load_mcp_resources()` returns at least one `autonomath://` URI.
  5. `_load_mcp_prompts()` returns at least one prompt entry.

If a future agent restores the bare hallucinated import or removes the
alias, invariant 3 trips immediately at pytest collection.

must all run after any change to the public agent-discovery surface:

`make mcp-public-discovery` (the legacy 3-step target) is intentionally
preserved as the MCP-only subset; `public-discovery` is the
LLM-ingestion + sitemap superset. Step 6 is the strictest gate
(exact-mode drift) so partial regenerations cannot silently land.

Trigger recipe — re-run after any of: tool add/remove, manifest version
bump, `llms.txt` hand-edit, sitemap source change, runtime cohort flip.
