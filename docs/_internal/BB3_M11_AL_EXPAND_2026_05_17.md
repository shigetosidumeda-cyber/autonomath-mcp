# BB3 — M11 Active-Learning expand 5 → 20 iter (deep multi-task fine-tune cycle)

last_updated: 2026-05-17

## TL;DR

Lane M11 (multi-task fine-tune of `cl-tohoku/bert-large-japanese-v2` over the jpcite corpus) was
provisionally scoped at **5 Active-Learning iterations**. BB3 (Big-Build Round 3) expands the loop
to **20 AL iters** and appends a **second distillation stage** (`distill_v2`) using the iter-20
teacher.

- AL iter cost: **~$16 / iter** on `g4dn.12xlarge` (≈ 4h × $3.91)
- 15 extra iters (6..20): **~$240**
- distill v2 (12h × $3.91): **~$47**
- **Total BB3 incremental burn: ~$287** vs original 5-iter plan
- Sits comfortably inside the **$19,490 Never-Reach** hard ceiling
- NO LLM API — OSS HuggingFace encoder + rule-based AL uncertainty sampler

## Why expand 5 → 20

The 5-iter window covers only the most-uncertain ~5,000 examples. Edge-case
coverage (rare patent classes, niche subsidy combinations, 国際課税 treaty
fragments) plateaus quickly. 15 additional uncertainty cycles each pull the
top-1,000 hardest examples from a 10K unlabeled pool, yielding ~15K more
hard-negative supervisions before distillation.

Target lift on hard examples: **+10-15% accuracy** on the 4-head suite
(MLM / NER / REL / RANK) vs the iter-5 baseline.

## Chain plan (23 stages)

```
1..20.  al_iter_{1..20}     (~4h each on g4dn.12xlarge, ~$16/iter)
21.     distill              (12h on g4dn.12xlarge, ~$47)
22.     augment              (24h on g4dn.12xlarge, ~$94)
23.     distill_v2           (12h on g4dn.12xlarge, ~$47, teacher = iter 20)
```

Per-iter Active-Learning loop:

1. Load latest checkpoint (warm-start; iter N starts from iter N-1 weights).
2. Predict on a 10K-example unlabeled pool.
3. Sort by uncertainty: entropy across the 4 task heads (MLM / NER / REL / RANK).
4. Take **top 1,000 hardest** rows.
5. Weakly supervise the picks via the **same regex / lexicon rules** used in
   `multitask_corpus_prep_2026_05_17.py` (NO LLM).
6. Re-train (warm-start) on the augmented corpus shard at
   `s3://<bucket>/finetune_corpus_multitask/al_iter_{N}/train.jsonl`.
