# 業種別 jpcite Use Case

jpcite は、日本の公的制度・税制・法令・法人情報・行政処分などを、出典 URL と取得時点付きで返す REST / MCP API です。料金は ¥3 / billable unit (税込 ¥3.30)、匿名枠は 3 req/日/IP です。

このページでは、税理士・会計士・行政書士・中小企業診断士・M&A advisor・信用金庫のような業務で、AI agent が jpcite をどこで呼び出すと有用かを整理します。金額は request 数から計算できる利用料の例であり、顧客の機会損失や売上影響を保証するものではありません。

## 基本の読み方

| 項目 | 内容 |
|---|---|
| 料金 | 1 billable unit = ¥3 税別 / ¥3.30 税込 |
| 例: 100 req | 税別 ¥300 / 税込 ¥330 |
| 例: 500 req | 税別 ¥1,500 / 税込 ¥1,650 |
| jpcite の役割 | 一次出典付きの事実検索、照合、scaffold の提供 |
| 利用者側の役割 | AI agent の回答を確認し、専門判断・申請・助言は資格者または担当者が行う |

## 1. 税理士事務所

### よくある業務

- 顧問先ごとの月次レビューで、税制改正・補助金・制度変更の該当可能性を確認する
- 賃上げ促進税制、中小企業投資促進税制、インボイス関連制度などの一次資料を素早く集める
- 顧問先別に出典 URL 付きのメモや説明資料を作る

### jpcite を呼ぶ場所

- 顧問先 1 社の制度レビュー
- 税制・補助金の候補抽出
- 出典 URL と取得時点を添えた月次メモの作成

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 顧問先 100 社を月 1 回レビュー | 100 req/月 | ¥300/月 |
| 追加で 10 社に制度候補を深掘り | 30-50 req/月 | ¥90-¥150/月 |

## 2. 会計士事務所

### よくある業務

- 研究開発税制、IT 導入補助金、会計処理の根拠条文や関連資料を確認する
- 被監査会社からの照会に対し、一次資料へのリンクを添えて論点を整理する
- 監査調書や質問リストの下書きに、公開情報の根拠を付ける

### jpcite を呼ぶ場所

- 税制 ruleset や通達リンクの検索
- 会社別の制度・採択・行政処分の確認
- 質問リストや確認メモの scaffold 作成

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 監査 1 社の公開情報確認 | 20-40 req/期 | ¥60-¥120/期 |
| 10 社分のピーク期確認 | 200-400 req/期 | ¥600-¥1,200/期 |

## 3. 行政書士

### よくある業務

- 建設業許可、産廃許可、在留資格、補助金申請の周辺制度を確認する
- 法人番号を起点に、行政処分歴や関連制度を事前に確認する
- 申請前のチェックリストや顧客説明メモの下書きを作る

### jpcite を呼ぶ場所

- 法人情報と行政処分の照合
- 業種別の補助金・制度候補の検索
- 排他ルールや前提条件の確認

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 1 案件の事前確認 | 10-20 req/件 | ¥30-¥60/件 |
| 月 25 案件の事前確認 | 250-500 req/月 | ¥750-¥1,500/月 |

## 4. 中小企業診断士

### よくある業務

- 顧問先や相談先の業種・地域・設備投資計画に合う制度を探す
- 補助金候補、認定制度、関連税制をまとめて提示する
- 採択事例や制度要件の一次資料を添えて、提案前の確認資料を作る

### jpcite を呼ぶ場所

- 顧問先一覧の月次スクリーニング
- 個社ごとの制度候補抽出
- 申請前チェックリストの scaffold 作成

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 顧問先 50 社を月 1 回スクリーニング | 50 req/月 | ¥150/月 |
| 月 10 社を個別に深掘り | 30-60 req/月 | ¥90-¥180/月 |

## 5. M&A advisor

### よくある業務

- 対象会社の法人情報、行政処分、採択履歴、適格事業者登録を確認する
- 許認可業種の公開情報 DD で、確認すべき一次資料を集める
- DD 質問リストの下書きに、公開情報の根拠を付ける

### jpcite を呼ぶ場所

- 法人番号を起点にした公開情報照合
- 行政処分・採択・制度関係の確認
- DD 質問リストの scaffold 作成

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 1 社の公開情報 DD | 30-50 req/件 | ¥90-¥150/件 |
| 月 5 社の公開情報 DD | 150-250 req/月 | ¥450-¥750/月 |

## 6. 信用金庫・金融機関の渉外担当

### よくある業務

- 取引先に関係する補助金・認定制度・融資制度を確認する
- 融資前の公開情報確認として、行政処分や法人情報を調べる
- 支店内で共有する顧客別メモに、出典 URL を添える

### jpcite を呼ぶ場所

- 取引先別の制度候補確認
- 行政処分や法人情報の照合
- 融資前 DD メモの scaffold 作成

### 利用量の例

| 使い方 | 目安 req | 税別料金 |
|---|---:|---:|
| 取引先 100 社を月 1 回確認 | 100 req/月 | ¥300/月 |
| 50 社を融資前に追加確認 | 150-250 req/月 | ¥450-¥750/月 |

## 業種別 request 量の目安

| 業種 | 典型的な使い方 | 目安 req | 税別料金 |
|---|---|---:|---:|
| 税理士 | 顧問先 100 社の月次レビュー | 100-150 req/月 | ¥300-¥450/月 |
| 会計士 | 監査 10 社のピーク期確認 | 200-400 req/期 | ¥600-¥1,200/期 |
| 行政書士 | 月 25 案件の事前確認 | 250-500 req/月 | ¥750-¥1,500/月 |
| 中小企業診断士 | 50 社スクリーニング + 個社深掘り | 80-110 req/月 | ¥240-¥330/月 |
| M&A advisor | 月 3-5 社の公開情報 DD | 150-250 req/月 | ¥450-¥750/月 |
| 信用金庫 | 取引先確認 + 融資前 DD | 250-350 req/月 | ¥750-¥1,050/月 |

## 注意事項

- 上記は request 数に基づく料金例です。実際の利用量は、検索範囲、対象社数、取得する情報量、AI agent 側の設計で変わります。
- jpcite は LLM 推論を代行しません。顧客側の Claude / ChatGPT / Cursor / Codex などが jpcite の REST / MCP API を呼び出します。
- jpcite は一次出典付きの事実検索と scaffold を提供します。税務判断、法的判断、監査意見、申請代理、許認可書面の作成などは、該当する有資格者または担当者が行ってください。
- 出典 URL、取得時点、known gaps を確認し、必要に応じて公式資料を直接確認してください。
