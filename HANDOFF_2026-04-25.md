## 1. プロジェクト全体像 (cold start 用)

### 1.1 何の repo か

- **PyPI 名**: `autonomath-mcp` / **import path**: `jpintel_mcp` (legacy, 改名禁止)

- **Operator**: Bookyou株式会社 (T8010001213708) / 代表 梅田茂利 / info@bookyou.net
- **Launch target**: 2026-05-06 (Fly.io Tokyo + Cloudflare Pages + Stripe metered)


```python
def test_subsidy_roadmap_unknown_prefecture_surfaces_input_warnings(client, roadmap_db):
```

## 10. 既知の罠 / pitfalls

これは将来の自分 (= 次の Claude) が踏まないように:

1. **memory の数字は stale 化する** ← 価格 / count / 日付は CLAUDE.md / README / DB 直接照会と相互チェック
2. **`tier='X'` quarantine を search path から除外し忘れない** ← 全 search で `tier IN ('S','A','B','C')` filter
3. **prefecture typo は silent 0-row でなく `input_warnings` で返す** ← BUG-2 pattern、全新 search/one-shot tool に必須
4. **`source_url` に aggregator 入れない** ← noukaweb / hojyokin-portal / biz.stayway 全 ban
5. **`source_fetched_at` を空 refetch で書き換えない** ← セマンティクス honesty 維持
6. **autonomath.db SHA256 を取るときは WAL checkpoint 必須** ← 実 hash drift する
7. **import path は `jpintel_mcp` のまま** ← PyPI 名 `autonomath-mcp` と分離、改名禁止
8. **subagent に Anthropic API 呼ばせない** ← 必ず subagent 本人が推論
9. **memory に書く数字 (price / count / 日付) は値が変わったら memory も update** ← 今回 ¥0.5 → ¥1 を修正したのが好例
