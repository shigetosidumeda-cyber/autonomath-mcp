# freee 審査担当者向け — 動作確認ウォークスルー

> 所要時間: 約 5 分。freee の test 事業所 + 当社が発行する demo アカウントで完結します。

## 事前準備

- freee の **test (テスト) 事業所** は不可。本アプリは public app として申請しているため、本番 (production) の事業所でログインしてください。
- 当社で freee 審査担当者用の demo 環境を発行済みです。secure_notes 経由で送付した認証情報をご利用ください。

## ステップ

### 1. アプリインストール

1. freee アプリストアの「jpcite (旧 税務会計AI)」ページから「インストール」をクリック
2. freee 認可画面が開きます (scope: `read` のみ要求)
3. 「許可する」を選択

### 2. 補助金検索

1. インストール後、freee 会計のアプリメニューから「jpcite (旧 税務会計AI)」を起動
2. ポップアップ画面が開き、上部に貴事業所名が pill (バッジ) で表示されます
3. デフォルトで「補助金」タブが選択されています
4. キーワード欄に `省エネ` と入力、都道府県欄で `東京都` を選択
5. 「検索」ボタンをクリック
6. **期待結果**: Tier S/A/B の補助金が最大 5 件、出典 URL 付きで表示される
   - 例: 「省エネルギー投資促進支援事業費補助金 (経産省)」など
7. 「出典を確認」リンクをクリックすると経産省の一次情報ページが新規タブで開く

### 3. 税制優遇検索

1. 「税制優遇」タブをクリック
2. キーワード欄に `賃上げ` と入力、「検索」
3. **期待結果**: 「中小企業向け賃上げ促進税制」など、適用期間 (例: 令和 6 年 4 月 1 日 〜 令和 9 年 3 月 31 日) 付きで一覧表示

### 4. インボイス登録番号確認

1. 「インボイス番号確認」タブをクリック
2. 入力欄に当社法人番号 `T8010001213708` を入力 (T を省いて 13 桁でも可)
3. 「確認」ボタン
4. **期待結果**: 「Bookyou株式会社」「東京都文京区小日向 2-22-1」「有効」が表示される
5. 「国税庁公表サイトで確認」リンクをクリック → 国税庁適格請求書発行事業者公表サイトに deep-link

### 5. 税理士法 §52 免責の表示確認

- 画面下部のフッターに常時表示されている黄色背景の免責文言をご確認ください
- 全 API レスポンスにも `_disclaimer` フィールドとして同梱しています (Network タブで確認可能)

### 6. ログアウト動作

1. 画面右上の「ログアウト」をクリック
2. **期待結果**: セッション cookie が削除され、`https://jpcite.com/` にリダイレクト
3. 再度プラグインを起動すると OAuth 認可画面に戻る

## 課金についての注意

- **審査期間中の従量課金は発生しません**: 審査担当者用 API key には課金免除フラグを付与済みです (Bookyou 内部運用)
- 審査終了後、本番利用者には `月 50 req 無料 / 超過 ¥3.30 (税込) /req` の従量課金が適用されます

## トラブルシューティング (審査担当者向け)

- OAuth 認可で 400 が返る場合 → state mismatch の可能性。ブラウザの cookie をクリアして再試行
- 検索が 504 で返る場合 → api.jpcite.com の一時的な高負荷。30 秒待って再試行
- 任意のエラーで `info@bookyou.net` にご連絡ください (24h 以内に返信)

## 連絡先

- 審査窓口: info@bookyou.net (Bookyou株式会社 / 代表 梅田茂利)
- 緊急時: 審査担当者専用 Slack channel (招待メールを別途送付)
