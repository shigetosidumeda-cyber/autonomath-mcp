## アプリ説明文 (1 段落 / 約 250 字)
jpcite (旧称 税務会計AI) は、freee 会計に登録された事業所情報 (法人番号・所在地) を読み取り、その事業所が利用可能な補助金・助成金・税制優遇措置・関連法令・行政処分情報・適格請求書発行事業者の登録状況を、Bookyou株式会社が運営する公的データベース (約 13,000 件の制度 + 9,400 件の法令 + 13,800 件のインボイス登録事業者) から横断検索して返します。すべての回答に出典 URL (官公庁・e-Gov・国税庁) を付与し、検索結果は freee 会計の画面内ポップアップに表示。Claude Desktop 等の AI アシスタントとも MCP プロトコルで連携可能で、自然言語での問い合わせにも対応します。

## 主要機能 (3 つの bullet)

### 1. 補助金・助成金の即時検索 (約 13,000 件 / Tier S/A/B 区分)
- freee 会計の都道府県情報から、地域・全国の補助金を Top 5 表示
- 中小企業庁・経産省・各都道府県の一次情報のみを掲載 (アグリゲータサイトは除外)
- Tier S (高採択率・大型) から Tier C (条件限定・少額) まで、tier バッジで一目で判別

### 2. 税制優遇措置の検索 (35 ruleset / 法令条文付き)
- 賃上げ促進税制・中小企業経営強化税制・DX 投資促進税制など、現行 35 措置を網羅
- 適用期間 (令和元年 4 月 1 日 〜 令和 9 年 3 月 31 日 等) を明示
- 法令条文 (租税特別措置法 etc.) との対応関係を表示

### 3. インボイス登録番号の照会 (約 13,800 事業者の delta データ)
- T + 13 桁の登録番号から、事業者名・所在地・有効/失効ステータスを即時確認
- 国税庁適格請求書発行事業者公表サイトへの deep-link 付き
- freee 会計の仕入先データから登録番号をコピペするだけで照会可能

## 想定ユース ケース (Use cases)

- **経理担当者**: 仕入先から受領したインボイスの登録番号を、freee 会計の取引画面を離れずに 1 クリックで確認したい。
- **会計事務所**: クライアントの事業所情報から、適用可能な補助金・税制優遇を網羅的に提示したい。年度ごとの制度改正にも追随したい。
- **中小企業経営者**: 自社の都道府県・業種で利用できる補助金を、申請可能性の高い順 (Tier S/A) に一覧で見たい。出典が官公庁の一次情報であることを確認したい。
- **AI アシスタント連携**: Claude Desktop や ChatGPT に「うちの今期使える税制を教えて」と自然言語で質問し、freee の事業所データを context として的確な回答を得たい (MCP プロトコル経由)。
