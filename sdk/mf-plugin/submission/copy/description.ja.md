# 税務会計AI — マネーフォワード クラウド アプリストア掲載文 (日本語)

## 1 段落 説明文

税務会計AI は、マネーフォワード クラウド会計をご利用中の事業者向けに、補助金・税制優遇・法令・判例・インボイス登録番号 を一画面から横断検索できるプラグインです。MF クラウドの画面右上から起動するポップアップ内で、お使いの事業者の都道府県や業種に応じた補助金候補、賃上げ促進税制等の優遇措置、電子帳簿保存法等の法令条文、関連判例、取引先のインボイス登録番号有効性を、すべて一次出典 (各省庁・自治体・国税庁・e-Gov・裁判所) へのリンク付きで即座に確認できます。月 50 リクエストまで無料、超過分のみ 1 リクエスト ¥3.30 (税込) の従量課金で、座席数課金や Pro/Free 等のティア課金は一切ありません。

## 3 つの特長

- **MF 画面内で完結する横断検索**: MF クラウド会計のサイドメニューからポップアップを開くだけで、補助金・税制・法令・判例・インボイス登録の 5 ジャンルを 1 つの UI から検索できます。タブを切り替えるだけで別ジャンルに移れます。
- **すべての結果に一次出典リンク**: 検索結果は各省庁・自治体・国税庁・e-Gov 法令データ・裁判所判例検索など 一次情報源への deep-link を必ず添付します。アグリゲータサイト経由の出典ロンダリングは行いません。出典が取れない情報は表示しません。
- **完全従量課金・座席課金なし**: 月 50 リクエストまで無料。超過分のみ 1 リクエスト ¥3.30 (税込) の Pay-as-you-go 課金で、Pro / Free 等のティア・座席数課金・年間最低契約金額は一切ありません。Bookyou 株式会社 (T8010001213708) が Stripe 経由で適格請求書を発行します。

## ユースケース

- **税理士・経理担当者**: 顧問先・自社で補助金が使えるか、月初 5 分で MF 会計を見ながら確認し、〆切が近いものを書き留める。
- **中小企業の経理**: 取引先から受領した請求書のインボイス登録番号 (T+13 桁) を MF 会計の請求書登録前に即時検証し、登録漏れの取引先に通知する。
- **設備投資前の事前確認**: 「省エネ 設備投資」「賃上げ」などのキーワードで、自社が要件を満たす補助金・税制優遇を事前にスクリーニングし、顧問税理士への相談材料を整理する。
- **電帳法対応の条文確認**: 「電子帳簿保存法 第7条」のように条文番号で検索し、e-Gov の最新条文 (CC-BY 4.0) を MF 画面のまま参照する。
- **類似判例の確認**: 「役員報酬 損金算入」など、税務処理で迷いやすいテーマの判例を裁判所公開データから引き当て、判断材料にする。

## 取り扱わない情報 (個人情報保護法・MF アプリポータル レビュー対応)

- 仕訳・元帳・取引明細・口座連携・残高・従業員情報・マイナンバーは **一切取得しません**。
- 取得するのは MF クラウド事業者の **表示名と tenant_uid のみ** で、これは検索クエリの都道府県スコープと利用量計測 (billing 整合) のみに使用します。
- アクセスログは API 利用量計測のため 30 日間のみ保持し、その後自動削除します。
- 取得情報は Bookyou 株式会社のサーバー (Fly.io HND リージョン) で日本国内処理を行い、AI 学習データには使用しません。
