# Code of Conduct

## 私たちの誓い

AutonoMath コミュニティに貢献する人すべてを尊重し、ハラスメントのない参加環境を維持します。年齢・性別・宗教・国籍・性的指向・障害の有無を問わず、誰もが安心して issue / PR / discussion に参加できることを目指します。

## 期待される行動

- 他者に対する敬意と配慮
- 異なる視点・経験への寛容
- 建設的なフィードバック
- 過去の判断より良い解決策が示されたら受け入れる
- コミュニティ全体の利益を優先する

## 期待されない行動

- 性的・暴力的な言動
- トローリング、侮辱、人格攻撃
- 公開・私的を問わない嫌がらせ
- 同意なしに他者の個人情報を公開すること
- 業務上不適切と一般的に判断される行為

## 適用範囲

GitHub repo (issue / PR / discussion / commit message)、AutonoMath の公式チャネル、関連イベントすべてに適用されます。

## 報告と対応

問題行動を目撃または受けた場合、`info@bookyou.net` に直接ご連絡ください。Bookyou 株式会社 (T8010001213708) 代表 (梅田茂利) が一次対応します。

報告内容は秘匿し、報告者のプライバシーを保護します。

対応:

1. 警告 (一次)
2. 一時的な参加禁止
3. 永久 ban

## 帰属

このドキュメントは [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を参考に作成しています。
