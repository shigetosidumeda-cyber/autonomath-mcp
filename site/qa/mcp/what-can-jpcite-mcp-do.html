jpcite MCPで何ができる?

最終確認日: 2026年4月30日 / 出典取得: 2026年4月30日 / jpcite

※一次資料は頻繁に改訂されます。最新は出典の公式ページをご確認ください。

回答 (TL;DR)

Claude Desktop / Cursor / Cline から96 toolsを呼び、日本の制度・法令・判例・税制を検索できる。

詳細 Q&A

jpcite MCPは何をするサーバーですか?
日本の補助金・融資・税制・認定・法令・判例・行政処分・適格請求書発行事業者を、AIクライアントから tool call で検索する MCP server です。
どのAIクライアントで使えますか?
MCP stdio に対応した Claude Desktop、Cursor、Cline、Continue などで使えます。ChatGPTやCustom GPTではREST/OpenAPI経由の組み込みが主経路になります。
ツール数はいくつですか?
公開ファクトシート上の最新値では、standard configuration で 96 MCP tools です。制度検索、制度詳細、batch detail、排他ルール、採択事例、法令、税制、判例、provenance lookup などに分かれます。
匿名で試せますか?
はい。匿名は 3 req/日 per IP まで登録不要で試せます。本番利用はAPI keyを発行し、従量課金対象単位ごとに ¥3 税別 (税込 ¥3.30) で使います。
LLMが直接Web検索する場合との違いは?
Web検索はページ単位の候補を返します。jpcite MCPは制度ID単位の構造化レコード、一次資料URL、取得時刻、排他ルール、schemaを返すため、AI agent が後続処理に渡しやすい形になります。

API で取得

このトピックに関連する制度・税務 ruleset・法令データは REST API および MCP で取得できます。

curl -H "X-API-Key: YOUR_API_KEY" \
 "https://api.jpcite.com/v1/programs/search?q=jpcite%20MCP%20Claude%20Desktop"

MCP クライアント (Claude Desktop / Cursor / Cline など) では search_programs(q="jpcite MCP Claude Desktop")、ChatGPT Custom GPT では OpenAPI Actions 経由で同等の REST endpoint を呼べます。詳細は API reference 参照。
