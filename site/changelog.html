横断検索の機能拡充

  • 制度・採択事例・融資・行政処分・法令・判例・税制を 1 つの API で横断検索。
  • MCP プロトコル 2025-06-18 対応。 Claude Desktop / Cursor から直接呼び出し可能。
  • API レスポンスに 出典 URL と 取得時刻を全件付与。
  • カタカナ・ひらがな の表記揺れを自動展開 (例: モノづくり / ものづくり、 インボイス / 適格請求書)。
  • 都道府県名 の入力誤りは無音 0 件返却ではなく、 警告を返すよう改善。
