本番 API です。 リクエストは即時 api.jpcite.com に届き、 匿名クォータ (3/日) を消費します。 リセットは JST 翌日 00:00。 残量は下のインジケータで確認できます。
3 回の無料ライブ検証: Evidence Packet を返すエンドポイント (GET /v1/intelligence/precomputed/query) を選択済みです。 デフォルトで source_tokens_basis=pdf_pages · source_pdf_pages=30 · input_token_price_jpy_per_1m=300 が入っているため、 送信すると source_url · source_fetched_at · known_gaps · compression.input_context_reduction_rate が返ります。 2 回目は source_pdf_pages を変えて差を確認、 3 回目は自社の質問に差し替えてください。
