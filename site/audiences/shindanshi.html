中小企業診断士向け

月次 saved search 50 顧問先 fan-out、eligibility chain 適格性連鎖、renewal forecast、補完制度 探索、申請 kit scaffold を Claude Desktop 経由で 1 query。¥3/billable unit 完全従量、anonymous 3 req/日 free per IP。

業法 fence

行政書士法 §1 + 弁護士法 §72 fence (重要): jpcite は申請書面の作成・提出代理・法律判断を一切行いません。本サービスは事実検索 + scaffold (cover letter draft + 必要書類 checklist) + 一次 URL 提供のみ、最終申請書面は診断士本人の判断で行政書士法 §1 範囲内で作成してください。

Cost saving calculator (月次 30 顧問 × 10 req: 純 LLM vs jpcite ¥3/req)

baseline 注記: 数値は中堅診断士 30 顧問 × 月 10 req の想定値 (推定値)。「純 LLM」は外部 web search + token 課金で同じ公開情報 (顧問先別 keyword search・eligibility chain・renewal forecast 素材・補完制度・申請 kit 素材) を fetch した場合、「jpcite ¥3/req」は jpcite MCP 1-call 経路で同じ事実検索を回した場合の差分。詳細仮定は docs/canonical/cost_saving_examples.md case 4 を参照。

MCP install (Claude Desktop)

Claude Desktop の claude_desktop_config.json に下記を追加してください。

{
  "mcpServers": {
    "jpcite": {
      "command": "uvx",
      "args": ["autonomath-mcp"]
    }
  }
}

paid usage を有効化する場合は env に API key を追加。anonymous 3 req/日 free per IP は API key なしで利用可 (JST 翌日 00:00 リセット)。顧問先ごとの fan-out は X-Client-Tag header (saved_searches.profile_ids 連動) で属性付与可能。
