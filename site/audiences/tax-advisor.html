税理士法 第 52 条 (税理士業務の制限)

jpintel-mcp (税務会計AI) は税務代理・税務書類作成・税務相談を行うものではありません。

本サービスは公開されている税制優遇制度の情報提供 (情報処理) のみを行います。

個別具体的な税務判断は、必ず税理士にご相談ください。

なぜ 税務会計AI か

一次資料 99%以上

国税庁・財務省・e-Gov

tax_rulesets は国税庁原典、laws は e-Gov CC-BY、措置法・通達は条項単位で参照。集約サイト (二次情報源) は使わない。顧問先への説明根拠として URL を提示できる。

排他ルール 181 本

特例の併用失格を事前検出

賃上げ促進税制 + 中小企業投資促進税制 + DX 投資促進税制 等の併用可否を 181 本の排他/前提ルールで構造化。「気づかないまま併用申告 → 後日否認」を回避できる。

改正アラート

電帳法・インボイス・改正電通法

関心条項を登録すると、改正・通達発出・施行日を email で通知。月次ダイジェストは無料、即時 通知は ¥500/月。確定申告期の事故を予防する。

主要 use case

1. 顧問先からの「今年使える特例は?」

業種・規模・投資予定額を入れると、事前確認 エンドポイントが上位 5 件 (賃上げ促進・中小企業投資促進・DX 投資・特別償却) を返す。Claude Desktop で MCP ツール search_tax_incentives を呼び、根拠条項 + 適用期限 + 排他ルールを併記した提案書を生成。

2. 措置法 検索 + 通達リンク

措置法 42 条の 12 の 5 (賃上げ促進税制) のような条項を検索し、最新改正履歴を取得。search_by_law で関連制度 (補助金・優遇税制) を逆引きできる。

3. 電帳法・インボイス 改正キャッチ

関心条項を登録するとアラートが届く。例: 電子帳簿保存法 7 条 (電子取引データ保存) の改正、インボイス 2 割特例の延長、適格請求書の経過措置等。原典 URL 付き。

4. 確定申告期の集客 (Widget)

事務所サイトに <script> 1 行で「使える特例検索」ボックスを埋込。Standard 月 ¥10,000 (10,000 リクエスト 含む、ロゴ付) または ホワイトラベル 月 ¥30,000 (30,000 リクエスト 含む、ロゴ非表示)。来訪者がその場で自社向け特例を発見 → 申告依頼につながる導線。

5. 顧問先の補助金併用 cross-check

顧問先が補助金 + 税制特例を併用する場合、combined_compliance_check で「ものづくり補助金 + 賃上げ促進税制」の組合せが排他ルールに抵触しないかを確認。

Python (商用 ¥3/リクエスト)

import httpx
H = {"X-API-Key": "sk_..."}
# 1. 税務 ruleset (国税庁原典) を検索
r = httpx.get(
  "https://api.zeimu-kaikei.ai/v1/tax_rulesets/search",
  params={"q": "電帳法 7 条", "limit": 10},
  headers=H)
for row in r.json()["items"]:
    print(row["name"], row["source_url"])

# 2. 税制特例 (DX 投資促進・賃上げ促進等) を抽出
r2 = httpx.get(
  "https://api.zeimu-kaikei.ai/v1/am/tax_incentives",
  params={"limit": 10}, headers=H)
for row in r2.json()["items"]:
    print(row["name"], row.get("sunset_at"))

tax_rulesets/search は jpintel コア (35 件)、am/tax_incentives は autonomath 由来。APIキー は dashboard.html から発行。

よくある質問 (税理士)

顧問先 80 社 (法人 60 / 個人 20) でどれくらいの req 数になりますか?

1 社あたり月 5-10 件の問合せ → 月 400-800 リクエスト → ¥1,200-¥2,400 (税別) が目安。3 月の確定申告期は跳ね上がるため、¥3-5,000/月 を上限想定に。

扱える税法はどこまで?

tax_rulesets は 2026-04 時点 35 件 (インボイス・電帳法等中心) で継続拡張中。 laws テーブルは 法令本文 154 件 + 法令メタデータ 9,484 件 (本文ロード継続中、 法令名 resolver は 9,484 件全件で稼働、 全文検索可能なのは 154 件)。詳細は API リファレンス。

通達・質疑応答事例まで参照できますか?

現時点で参照可能なのは法令本文 (e-Gov CC-BY) と税務 ruleset (国税庁原典)。通達・質疑応答事例は 構造 構築済み・データロード準備中。

改正アラートはどの程度詳細ですか?

法令名・改正条項・施行日・関連制度 ID を email + RSS で配信。月次ダイジェスト無料、即時通知 ¥500/月。本文は要約のみ、原典 URL を付与。

顧問先からの相談を士業 affiliate で受けられますか?

はい。専門家 登録は無料、成約時のみ ¥3,000 (税別)。プロフィールに専門分野・地域・料金表を載せ、中小企業 経営者からの問合せが LINE bot 経由で届きます。

守秘義務 (税理士法 38 条) との整合は?

税務会計AI は検索クエリを学習データに使用しません。推論ログは課金照合のみに利用。顧客固有の事業情報を入力する場合は事務所内ポリシーに従ってください。

jGrants や経産省サイトを直接見ればいいのでは?

jGrants は申請ポータルです。「自分の顧客に該当する制度を網羅的に発見する」「複数の制度の併用可否を判定する」「他省庁・地方自治体の制度まで横串で検索する」ことはできません。税務会計AI は 13,578 件 (jGrants 掲載 + 2,286 採択事例 + 108 融資 + 1,185 行政処分 + 法令本文 154 + 法令メタデータ 9,484) を 1 つの API で照会でき、181 の exclusion / prerequisite ルールで併用可否も機械判定します。事務所内で「この顧客に対象の制度を 30 秒で全件出す」運用ができます。

顧問先への営業材料として使えるか?

はい。「あなたの会社で使える補助金候補を 税務会計AI で全件抽出すると ¥X,XXX 万円分の上限がありました」というレポートを毎月顧問先に出すことで、関与度向上と追加業務獲得に使えます。1 顧客あたり数 req で済むため、月 50 リクエスト 無料枠で 5–10 顧客分は完全無料、それ以上は ¥3/リクエスト。

TKC / 弥生 / freee と何が違うのか?

申告ソフトは「税額計算 + 申告書出力」に特化しており、「顧問先が使える補助金 / 認定 / 融資の発見」はカバーしません。税務会計AI は申告ソフトの代替ではなく、補完ツール (顧問関与の前段リサーチ + 改正アラート + 過去採択事例) です。月顧問料 ¥30,000 の顧問先に「今期使えそうな補助金 3 件、申告期前に検討を」を毎月送るだけで関与度が上がり、追加業務 (補助金申請補助 / 経営相談) の獲得に繋がります。

関連 利用者層 は?

行政書士 (許認可 + 補助金書類)、AI 開発者 (事務所内 RAG 自作)、VC / M&A 専門家 (DD enforcement check)、中小企業経営者 (顧問先のセルフ調査) と use case が重なる。5 利用者層 全体を比較。

税理士向けリソース

  • 中小企業 経営者ページ (顧問先の理解) — 顧問先がどんな目線で 税務会計AI を使っているかを把握。LINE bot 経由のセルフ調査 → 税理士相談という流入経路。
  • 行政書士ページ (協業相手) — 顧問先の許認可案件で行政書士と協業する際の参考。事業承継・M&A での税務 + 許認可承継 DD で重なる。
  • REST API: /v1/tax_rulesets/search (35 件) — 賃上げ促進税制・所得拡大促進・経営強化法等の優遇税制を国税庁原典から照会。条件判定 + 適用可否を機械チェック。
  • REST API: /v1/laws/search (本文 154 件 + メタデータ 9,484 件、 本文ロード継続中) — e-Gov CC-BY 法令データを検索。 全文検索が可能なのは現時点で 154 件、 残りは法令名 / 法令番号 / e-Gov リンクの resolver として参照可能。
  • 改正アラート登録 — 法令名・改正条項・施行日を email + RSS で配信。月次ダイジェスト無料、即時通知 ¥500/月。本文要約 + 原典 URL 付与。
  • 士業 affiliate に登録 — LINE bot 経由で 中小企業 から相談を受ける側になる。登録無料、1 紹介成立 ¥3,000 (税別) 専門家 受取。
  • 5 分で開始 (クイックスタート) — API キー発行から最初の検索までの最短ルート。月 50 リクエスト 無料枠で 5-10 顧客分は完全無料運用可能。
