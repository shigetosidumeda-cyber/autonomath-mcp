製造業特化 jpcite

製造業特化の補助金候補抽出 + 国税庁通達 (法人税基本通達) + 採択事例を 1 API で。製造業コンサル / 中小企業中央会 (製造部会) 担当者 / kintone-製造特化 plug-in 開発者向け。エンドユーザー (ものづくり中小企業 経営者 個人) ではなく、その上流に立つ「経由チャネル」を主対象に再設計しています。

対象

  • 製造業特化 補助金コンサル / 中小企業診断士 / 認定支援機関 — ものづくり / 設備投資 / 省エネ / GX / 事業再構築 / IT 導入 顧問先の補助金候補と通達根拠を 1 リクエストでまとめて返す
  • 都道府県中小企業中央会 (製造部会) / 各種工業組合 担当者 — 組合員企業向けの補助金 widget や会報誌コンテンツを jpcite から自動更新
  • kintone-製造特化 plug-in / 生産管理 SaaS / 在庫管理 SaaS 連携開発者 — 製造業向け業務アプリに「使える補助金候補」プラグインを差し込む

主要価値

  • JSIC E fence (製造業) — ものづくり・製造・設備投資・省エネ・GX・脱炭素・事業再構築・IT 導入・DX キーワード union で 456 件 の S/A/B/C tier program を絞り込み (snapshot 2026-05-07 時点)
  • 法基通 (法人税基本通達) のセクション固定参照 — 各補助金候補に対する課税関係 (圧縮記帳 / 国庫補助金等の益金不算入 / 圧縮限度額) のセクション ID 付き引用、50 件の tax_rulesets (研究開発税制 措置法42-4 / IT導入補助金 会計処理 含む) を併記
  • 経済産業省 行政処分 cohort — 経済産業省関連の処分 56 件 (ministry='経済産業省') を houjin_bangou で横断検索、与信審査・反社チェックの補助情報として併記可
  • JSIC E 採択事例 590 件 (snapshot 2026-05-07 時点、JSIC E 配下の case_studies 総数。E24 金属製品 / E26 生産用機械 / E09 食料品 を含む全細目) — 案件提案書に「同業の採択パターン」として添付できる一次資料リンク付き
  • 各補助金候補・各通達・各裁決事例・各処分・各採択事例に 一次資料 source_url を併記 (官庁・自治体・公庫などの一次資料に限定)
  • 通常 1 call = 1 課金単位、税込 ¥3.30 の完全従量。匿名枠は 3 リクエスト/日まで無料 (匿名 IP ベース、JST 翌日 00:00 リセット)

経由チャネルでの使い方

  • コンサル登録 — 認定支援機関の方へ。顧問先製造業者の補助金候補を pack_manufacturing 1 コールでまとめて取得 → 提案書に通達 ID 付きで添付。outreach 用の API 利用説明・ドキュメント直リンクは下部 CTA から。
  • 業種団体 widget 設置 — 都道府県中小企業中央会 (製造部会) や各種工業組合の会員サイト / 会報誌 Web 版に、jpcite の補助金候補 widget を埋込。組合員企業からは中央会経由で見える形。
  • kintone 製造特化 plug-in 開発 — 生産管理アプリ・在庫管理アプリで pack_manufacturing を呼び出し、案件登録時に「使える補助金候補」を併記。実装例は integration guide に掲載。

API 実例

curl "https://api.jpcite.com/v1/am/pack_manufacturing?prefecture=大阪府&employee_count=50" \
  -H "X-API-Key: $JPCITE_API_KEY"

corpus 開示 (2026-05-07 snapshot): JSIC E 関連 program 456 件 / JSIC E 採択事例 590 件 / 経済産業省 行政処分 56 件 / tax_rulesets 50 件。本サービスは情報検索です。個別具体的な税務・法律判断は資格を有する専門家にご相談ください (税理士法 §52 / 公認会計士法 §47条の2 / 行政書士法 §1の2・§19)。本ページは LLM による生成・要約は行っておらず、SQLite + 一次資料 URL のみで構成されています。
