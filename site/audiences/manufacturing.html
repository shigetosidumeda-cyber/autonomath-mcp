3 つの代表クエリ

  prefecture="愛知県",
  employee_count=50,
  revenue_yen=500_000_000
)

# REST
curl -H "X-API-Key: sk_..." \
  "https://api.zeimu-kaikei.ai/v1/am/pack_manufacturing?prefecture=愛知県&employee_count=50"

1 リクエスト ¥3。top 10 補助金 + 5 裁決事例 + 3 通達。

# 検索系で深掘り

  industry_jsic="E",
  tier=["S","A"],
  limit=20

免責事項 (税理士法 §52 / 公認会計士法 §47条の2)

本ページおよび pack_manufacturing tool の出力は、jpintel programs (製造業 fence) + nta_saiketsu + nta_tsutatsu_index 一次資料の機械的 aggregation です。 税務助言 (税理士法 §52) ・監査調書 (公認会計士法 §47条の2) ・申請代理 (行政書士法 §1) ・経営判断 (中小企業診断士の経営助言) の代替ではありません。 業種マッピングは JSIC E + 名称キーワード fence による heuristic で、各 program の適合可否は申請要領を一次資料 (source_url) で必ずご確認ください。 裁決事例・通達は公表時点の解釈であり、改正により現在の取扱が変更されている可能性があります。最終判断は資格を有する税理士・公認会計士・行政書士へご相談ください。
