この page は誰向けか

補助金 専門家 の実態は 9 割が中小企業診断士・FP・無資格の経営コンサル・認定経営革新等支援機関 (税理士・銀行・コンサル法人を含む) です。 行政書士は少数派 (申請代理が独占業務とはいえ、補助金実務全体での割合は小さい)。 本ページはその「大半を占める 9 割」 向けに書かれています — 申請書類の代理作成は行政書士独占ですが、 制度の事前リサーチ・絞込・排他チェックは士業独占ではなく、 全 専門家 の本業の中心です。

行政書士の方は 行政書士向けページ をご参照ください (許認可案件と組合せた use case を中心に整理)。

3 つの価値提案

1. 一次資料 100%

集約サイト排除 → 情報精度の確保

noukaweb / hojyokin-portal / biz.stayway などの集約サイトは 不採用 (内部運用ルールで source_url から ban)。 全制度が省庁・都道府県・日本政策金融公庫・公社などの一次資料 URL に直接紐付き、 専門家 が顧問先に提示しても 「孫引きだった」 「制度が実は終了していた」 リスクを排除します。

2. 排他ルール 181 本

重複申請の事前検出

「補助金 A と B は併用不可」「補助金 C は事前に認定 D が必要」 など公的機関 publicly stated な関係を 181 ルール (125 exclude + 17 prerequisite + 15 absolute + 24 other) で構造化。 program_ids を投げると衝突候補が返り、 「気づかず併用申請 → 後日失格」 を事前回避できます。

3. MCP / Slack Webhook 通知 通知 で agent 自動化

saved searches + Webhook 通知 通知

API/MCP に加え、 saved searches + Webhook 通知 通知 で 「東京都 × ものづくり × 30 日以内」 を保存し、 新規該当が出たら Slack / Claude agent に自動通知。 200 顧問先 × 月 10 query を 人手ゼロ で回せます。

採択率公表値の透明性

採択率公表値: 69 制度 / 10,790 中 (0.64%)

公的機関が round ごとの採択率を明示公表している制度は 10,790 中わずか 69 制度 (0.64%) です。 残り 99.36% は 「採択率公表データなし」 と honest に明示し、 推測値は出しません。 当社が 0.64% のカバレッジを 隠さず明示 するのは、 営業現場で 「採択率○%」 を盛って提示することが誤情報リスクと隣り合わせだからです。

採択率データが取得できる主な制度の例 (2026-04-25 時点):

  • ものづくり補助金: 65 round 公開、最低 16.97% / 最高 67.15% / 平均 48.29%
  • 事業再構築補助金: 48 round 公開、最低 22.52% / 最高 59.32% / 平均 39.55%
  • 小規模事業者持続化補助金: 46 round 公開、最低 38.72% / 最高 69.78% / 平均 54.88%
  • 事業承継・M&A補助金: 33 round 公開、最低 46.39% / 最高 73.06% / 平均 54.52%
  • IT導入補助金: 31 round 公開、最低 52.04% / 最高 80.40% / 平均 64.54%

残り 10,721 制度 (99.36%) は採択率公表値が存在しないため、 当社の per-program 制度 page には 「採択率公表データなし」 とのみ表示します。 「公開情報がないものを推測で埋める」 ことはしません。

主要 use case (補助金コンサルタント目線)

1. 顧問先 200 社の月次スクリーニング

各顧問先のプロファイル (業種 / 都道府県 / 設備投資意向) を保存し、 月初に saved searches で一括 query。 新規該当が Slack に流れ、 専門家 は確認だけで提案メールに進めます。

2. ものづくり / 事業再構築の認定支援機関業務

認定支援機関の関与が要件の制度 (ものづくり / 事業再構築) で、 関連制度抽出 + 排他チェック + 採択率公表値 (利用可能な制度のみ) を 1 リクエスト で提示。 提案書のエビデンス連結を自動化。

3. Slack / Claude / GPT から agent 発火

Claude Desktop / Cursor で get_program / search_programs / check_exclusions を MCP ツール として呼ぶ。 専門家 自身が prompt で 「○○業の県補助金は」 と聞くだけで、 一次資料リンクつきで返答。

4. 認定支援機関の DD (取引先デューデリ)

顧問先候補の法人番号で行政処分歴 1,185 件 + 適格請求書登録 13,801 件を横断 query。 受託前の DD を 30 秒で完了。 関連: VC / M&A 専門家 向けページ。

よくある質問 (補助金コンサルタント)

補助金コンサルタントとは具体的に誰ですか?

実態として 補助金 専門家 の約 9 割は 中小企業診断士・FP・無資格の経営コンサルタント・認定経営革新等支援機関 (税理士・銀行・コンサル法人等) であり、 行政書士は少数派です。 当ページはその大半を占める層に向けて作られています。 行政書士向けは 専用ページ を参照。

navit ¥1,000/月 と比較してなぜ高いのですか?

件数 (147,976 案件) は navit が圧勝です。 当社の差別化は API/MCP / Slack Webhook 通知 通知 / 排他ルール 181 本 / 一次資料 100% の 4 点で、 顧問先 200 社規模で agent から自動 query を発火する想定の運営に向きます。 「人が Web UI で 1 件ずつ検索する」 運用なら navit を推奨します。

顧問先 200 社で月コストはどれくらいですか?

顧問先 1 社 × 月 10 query × ¥3 = ¥30 / 顧問先 / 月。 200 社で ¥6,000 (税別) / 月 + 認定支援機関側の事前リサーチ ¥600 で計 ¥6,600 / 月 (税別、 税込 ¥7,260)。 月額固定 0、 最低利用なし。

排他ルール 181 本とは?

「補助金 A と補助金 B は併用不可」「補助金 C は事前に認定 D が必要」 などの公的機関 publicly stated な関係を構造化した 181 ルール (125 exclude + 17 prerequisite + 15 absolute + 24 other)。 重複申請による失格を事前検出します。

採択率は表示されますか?

公表値が存在する 69 制度 (10,790 中、 0.64%) のみ表示します。 ものづくり補助金は 65 round 公開分を保持しており、 直近 65 round の採択率は最低 16.97% / 最高 67.15% / 平均 48.29% です。 公表値がない制度は 「採択率公表データなし」 と honest に明示し、 推測値は出しません。

認定経営革新等支援機関の登録業務にも使えますか?

はい。 ものづくり補助金・事業再構築補助金は認定支援機関の関与が要件です。 税務会計AI で関連制度を抽出し、 提案書ドラフトに 1 次資料リンクと採択率公表値 (利用可能な場合) を併記して顧問先に提示できます。

税務助言は提供しますか?

提供しません。 税理士法 §52 により税務代理・税務書類作成・税務相談は税理士の独占業務であり、 税務会計AI はそれらを代行しません。 当サービスは 一次資料リンクと制度メタデータの抽出 のみを提供します。 個別の税務判断は税理士・社労士・中小企業診断士等の有資格者にご相談ください。

関連 利用者層 は?
