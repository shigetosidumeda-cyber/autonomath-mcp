建設業特化 jpcite

建設業特化の補助金候補抽出 + 国税庁通達 (法人税基本通達 / 消費税基本通達) + 採択事例を 1 API で。建設業コンサル / 全国建設業協会 担当者 / ANDPAD・kintone 等 業務 SaaS 連携開発者向け。エンドユーザー (建設業経営者 個人) ではなく、その上流に立つ「経由チャネル」を主対象に再設計しています。

対象

  • 建設業特化 補助金コンサル / 中小企業診断士 / 認定支援機関 — 顧問先建設業者の補助金候補と通達根拠を 1 リクエストでまとめて返す
  • 全国建設業協会 / 都道府県建設業協会 / 中央会 (建設部会) 担当者 — 会員企業向けの補助金 widget や会報誌コンテンツを jpcite から自動更新
  • ANDPAD / kintone / SPIDERPLUS / 写蔵 等 建設業務 SaaS 連携開発者 — 工事台帳・工程管理アプリに「使える補助金候補」プラグインを差し込む

主要価値

  • JSIC D fence (建設業) — 建設・建築・住宅・耐震・改修・空き家・工事・下請 キーワード union で 285 件 の S/A/B/C tier program を絞り込み (snapshot 2026-04-30 時点)
  • 法基通 (法人税基本通達) + 消基通 (消費税基本通達) のセクション固定参照 — 各補助金候補に対する課税関係 (圧縮記帳 / 国庫補助金等の益金不算入 / 消費税の不課税) のセクション ID 付き引用
  • 国土交通省 行政処分 cohort — 建設業法・宅建業法等の処分 79 件 (ministry='国土交通省') を houjin_bangou で横断検索、与信審査・反社チェックの補助情報として併記可
  • JSIC D 採択事例 43 件 (D06 総合工事業 28 / D08 設備工事業 10 / D 親 5) — 案件提案書に「同業の採択パターン」として添付できる一次資料リンク付き
  • 各補助金候補・各通達・各裁決事例・各処分・各採択事例に 一次資料 source_url を併記 (官庁・自治体・公庫などの一次資料に限定)
  • 1 リクエスト ¥3 (税込 ¥3.30) 完全従量、3 リクエスト/日まで無料 (匿名 IP ベース、JST 翌日 00:00 リセット)

経由チャネルでの使い方

  • コンサル登録 — 認定支援機関の方へ。顧問先建設業者の補助金候補を pack_construction 1 コールでまとめて取得 → 提案書に通達 ID 付きで添付。outreach 用の API 利用説明・ドキュメント直リンクは下部 CTA から。
  • 業種団体 widget 設置 — 全国建設業協会・都道府県建設業協会・中央会 (建設部会) の会員サイト / 会報誌 Web 版に、jpcite の補助金候補 widget を埋込。会員企業からは協会経由で見える形。
  • ANDPAD / kintone integration 開発 — 工事台帳プラグインや kintone カスタマイズで pack_construction を呼び出し、案件登録時に「使える補助金候補」を併記。実装例は integration guide に掲載。

API 実例

curl "https://api.jpcite.com/v1/am/pack_construction?prefecture=東京都&employee_count=30" \
  -H "X-API-Key: $JPCITE_API_KEY"

corpus 開示 (2026-04-30 snapshot): JSIC D 関連 program 285 件 / JSIC D 採択事例 43 件 / 国土交通省 行政処分 79 件。本サービスは情報検索です。個別具体的な税務・法律判断は資格を有する専門家にご相談ください (税理士法 §52 / 公認会計士法 §47条の2 / 行政書士法 §1 / 建設業法 §3)。本ページは LLM による生成・要約は行っておらず、SQLite + 一次資料 URL のみで構成されています。

API キー発行 → コンサル登録 (認定支援機関 outreach) → 業種団体 widget 設置 (全建協 / 中央会経由) → ANDPAD / kintone integration 開発 → 料金
