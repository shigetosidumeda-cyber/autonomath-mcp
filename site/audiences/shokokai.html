商工会・商工会議所向け 補助金検索 — Evidence Packet で 顧問先別 alert

全国商工会連合会 (1,535 商工会) + 日本商工会議所 (514 商工会議所) + 経営指導員 + 全国中小企業団体中央会 + 認定経営革新等支援機関ロールの担当者向け。 巡回先 SMB 1 社あたり、 補助金 (持続化補助金 様式4 / 商工会枠 / 商工会議所枠) + 融資 (マル経推薦) + 共済 + 認定 + エキスパートバンク を 30 秒で集約。 顧問先別の Evidence Packet (一次資料 URL + 排他ルール根拠) を Slack / email に alert。 営業 cold call 一切なし、 ¥3/billable unit 完全従量。

対象

  • 商工会・商工会議所 経営指導員 — 町内会単位で地域 SMB を巡回し、 補助金 / 融資 / 共済 / 認定を案内する一次窓口担当者
  • 全国商工会連合会 / 日本商工会議所 / 47 都道府県連合会 職員 — 中央 / 県連レベルでの会員企業向けコンテンツ・widget・会報誌の自動更新
  • 全国中小企業団体中央会 — ものづくり補助金 / 事業承継・引継ぎ補助金等の指定支援機関ロール
  • 認定経営革新等支援機関ロールの商工会員 — 経営革新計画認定 / 事業性評価の周辺制度横断

主要価値

  • 持続化補助金 商工会枠 / 商工会議所枠 — 様式4 (事業支援計画書) 発行窓口と、 商工会員向けの直接 link で keyword 検索を 1 リクエストに集約
  • マル経推薦の連動 — 商工会議所マル経融資 (経営改善資金) の推薦書発行フロー周辺制度を check_exclusions で他融資との併給判定
  • エキスパートバンク (専門家派遣) — 商工会の専門家派遣制度を corpus 化、 商工会員企業の支援メニューに追加可
  • 共済 / 福利厚生 — 特定退職金共済 / 商工会経営者休業補償 / 小規模企業共済 (中小機構) を 1 リクエスト横断
  • 47 都道府県 商工会連合会 directory — 各県独自の補助金 / 助成金を直接参照、 prefecture page から 1 タップで到達
  • Evidence Packet の Slack / email alert — 保存検索 + Webhook で 顧問先別 monthly screening を自動化、 各 alert に一次資料 URL + 排他ルール根拠を併記
  • 通常 1 call = 1 課金単位、税込 ¥3.30 の完全従量。匿名枠は 3 リクエスト/日まで無料 (匿名 IP ベース、 JST 翌日 00:00 リセット)

経営指導員の Day-in-a-Life シナリオ

  1. 09:00 巡回前準備 — 当日訪問予定の商工会員 3 社を X-Client-Tag 別に 1 batch lookup、 各社の業種 + 都道府県 + 直近の決算傾向から該当しそうな補助金 / 融資 / 共済 / 認定を Evidence Packet として印刷
  2. 10:30-15:00 巡回 — 紙 packet を商工会員に手渡しながら、 様式4 発行が必要な持続化補助金候補を即座に判定、 マル経推薦の併給可否を check_exclusions で確認
  3. 16:00 帰所後 — 翌月以降の保存検索を Slack / email Webhook に登録、 顧問先別 alert を継続化、 巡回ノートに一次資料 URL を貼付
  4. 月末 — 顧問先別の利用実績を X-Client-Tag 経由で集計、 商工会本部 / 県連への報告書に転記

サンプル query (curl)

# 北海道の小規模事業者向けに 持続化補助金 + 商工会枠 制度を keyword 横断
curl "https://api.jpcite.com/v1/programs/search?prefecture=北海道&q=持続化補助金" \
  -H "X-API-Key: $JPCITE_API_KEY" \
  -H "X-Client-Tag: client-hokkaido-smb-001"

corpus 開示: programs 11,601 件 (S/A/B/C tier) + 採択事例 2,286 件 + 融資 108 件 (担保 / 個人保証人 / 第三者保証人 三軸) + 行政処分 1,185 件 + 法令 9,484 件 (メタデータと参照導線) + 通達 / 裁決 (国税庁) corpus 連携。 jpcite は LLM API を呼び出さず、 SQLite + 一次資料 URL のみで構成 (商工会員相談記録の機密性は呼出側に保持されます)。 本サービスは情報検索です。 個別具体的な税務 / 法律判断は資格を有する専門家にご相談ください (税理士法 §52 / 行政書士法 §1の2・§19)。
