公認会計士事務所向け

研究開発税制 措置法 lookup、IT 導入補助金 会計処理 dual check、被監査会社 DD question deck、行政処分歴 filter、監査調書 template scaffold を Claude Desktop 経由で 1 query。¥3/billable unit 完全従量、anonymous 3 req/日 free per IP。

業法 fence

公認会計士法 §47条の2 fence (重要): jpcite は監査意見の表明・税務代理を一切行いません。本サービスは事実検索 + scaffold (template draft) + 一次 URL 提供のみ、最終調書・監査意見書は会計士本人の専権事項です。

MCP install (Claude Desktop)

Claude Desktop の claude_desktop_config.json に下記を追加してください。

{
  "mcpServers": {
    "jpcite": {
      "command": "uvx",
      "args": ["autonomath-mcp"]
    }
  }
}

paid usage を有効化する場合は env に API key を追加。anonymous 3 req/日 free per IP は API key なしで利用可 (JST 翌日 00:00 リセット)。
