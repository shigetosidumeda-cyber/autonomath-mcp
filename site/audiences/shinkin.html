信用金庫向け 補助金検索 — Evidence Packet で 顧問先別 alert

全国 254 信用金庫 + 信金中央金庫 + 信用組合 + 認定経営革新等支援機関ロールの信金担当者向け。 取引先 SMB の業種 × 都道府県 × 設備投資計画から、 補助金 / 融資 / 税制特例 / 商工会議所マル経 / 47 信用保証協会連動を 1 リクエストで抽出。 顧問先別の Evidence Packet (一次資料 URL + 排他ルール根拠) を Slack / email に alert。 営業 cold call 一切なし、 ¥3/req 完全従量。

対象

  • 信用金庫 (全国 254 行) / 信用組合 取引先担当 / 営業推進部 / 経営支援部 — 取引先 SMB の bank statement / 決算書 / 設備投資計画から補助金 / 融資の対象となりそうな案件を月次検出
  • 認定経営革新等支援機関ロールの信金 — 経営革新計画 (中小企業等経営強化法) 認定や事業性評価の周辺制度を顧客に案内
  • 信金中央金庫 / 全国信用金庫協会 / 地域・中小企業研究所 (SCBRI) 関係者 — 中央機関の調査資料・公開データの downstream 活用

主要価値

  • 地域金融機関 推薦が要件の制度の横断検索 — 信用金庫推薦・地域金融機関推薦が要件となる補助金 / 融資制度を program_facts から fact 単位で抽出
  • 商工会議所マル経推薦の連動 — マル経融資 (経営改善資金、 商工会議所推薦が要件) の周辺制度を check_exclusions で他融資との併給判定
  • 47 信用保証協会 corpus — 全国 47 拠点の directory + 関連保証制度を収載、 prefecture page から 1 タップで到達
  • 3 軸 融資 enum との接続 — loan_collateral / loan_personal_guarantor / loan_third_party_guarantor (担保 / 個人保証人 / 第三者保証人) は信金の稟議資料項目とそのまま一致
  • Evidence Packet の Slack / email alert — 保存検索 + Webhook で 顧問先別 monthly screening を自動化、 各 alert に一次資料 URL + 排他ルール根拠を併記
  • 1 リクエスト ¥3 (税込 ¥3.30) 完全従量、 3 リクエスト/日まで無料 (匿名 IP ベース、 JST 翌日 00:00 リセット)

経営支援部 担当者の典型 use case

  • 取引先審査・与信モニタリング — 取引先 1 社の業種 + 都道府県 + 設備投資計画から、 該当しそうな国 / 都道府県 / 市区町村 / 公庫 / 信用保証協会の制度を 30 秒で集約
  • マッチング提案 — マニュアル巡回 (中小企業庁 / 経産省 / 農水省 portal) を 1 リクエストに置換、 一次資料 URL 付きで顧客提案書に転記
  • 稟議資料の補強 — 3 軸融資 enum で稟議書の担保 / 保証人 欄をそのまま埋められる、 該当制度の出典 URL を引用根拠として添付

サンプル query (curl)

# 大阪府 製造業 取引先 1 社向けに 信金推薦が要件の融資 + 補助金候補を集約
curl "https://api.jpcite.com/v1/loan-programs/search?q=設備投資&collateral_required=not_required" \
  -H "X-API-Key: $JPCITE_API_KEY" \
  -H "X-Client-Tag: client-osaka-mfg-001"

# 信金担当者の手元 Claude Desktop / Cursor / Cline で MCP として呼ぶ
uvx autonomath-mcp --src=audience_shinkin

一次資料 (信金 / 中央会)

  • 全国信用金庫協会 — 254 信金 facing
  • 信金中央金庫 — 中央機関 (中銀機能)
  • 信金中央金庫 地域・中小企業研究所 (SCBRI) — 中小企業向け公開調査資料の配布チャネル
  • 全国信用組合中央協会 — 信用組合の上部団体
  • 中小企業庁 経営革新等支援機関 認定制度 — 認定経営革新等支援機関のロール定義
  • 全国信用保証協会連合会 — 47 信用保証協会の上部団体
  • 日本政策金融公庫 マル経融資 案内 — 商工会議所推薦による経営改善資金

corpus 開示: programs 11,601 件 (S/A/B/C tier) + 案件 採択事例 2,286 件 + 融資 108 件 (担保 / 個人保証人 / 第三者保証人 三軸) + 行政処分 1,185 件 + 法令 9,484 件 (うち本文索引 6,493 件) + 通達 / 裁決 (国税庁) corpus 連携。 jpcite は LLM API を呼び出さず、 SQLite + 一次資料 URL のみで構成 (信金内部の与信判断データは jpcite サーバを経由しません)。 本サービスは情報検索です。 個別具体的な税務 / 法律 / 与信判断は資格を有する専門家・所内稟議プロセスにご確認ください。
