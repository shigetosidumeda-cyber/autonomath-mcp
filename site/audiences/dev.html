主要 use case

1. Claude Desktop で日本制度 RAG

config に MCP server 追加するだけで、Claude が制度・採択・融資・法令を自律的に検索。「東京都の補助金で建設業向け、500 万円規模」のような自然言語クエリを内部的に search_programs + check_exclusion に分解する。

2. Cursor / Copilot 補助金 SaaS 開発

Cursor で MCP 接続、Cursor agent が AutonoMath tools を直接呼んで型定義 + 実装を生成。OpenAPI spec import で型安全な client コードも自動生成可能。

3. GPTs Actions / Dify / LangChain

OpenAPI 3.1 spec を ChatGPT GPTs の Action として import → ChatGPT 上で日本制度を検索する GPT が即作れる。Dify / LangChain も REST tool として登録するだけ。

4. 自社 SaaS の compliance feature

会計 SaaS / 営業 SaaS / 法務 SaaS に「使える補助金」「インボイス対応」「行政処分チェック」機能を追加。¥3/req metered で end-user 数に比例、固定費なし。White-label OK。

5. enterprise RAG への組込み

社内 Confluence + AutonoMath を 1 つの RAG に統合。combined_compliance_check / trace_program_to_law で制度 → 法令 → 通達まで連鎖検索。

よくある質問 (developer)

MCP の transport は stdio だけですか?

stdio が primary transport (Claude Desktop / Cursor 標準)。SSE / WebSocket は 2026 後半対応予定。REST API も同等機能で公開しているので、HTTP しか使えない環境はそちらを利用してください。

72 tools の内訳は?

39 コア (programs + case_studies + loan_programs + enforcement + get_usage_status + 7 one-shot discovery + 拡張: laws / tax_rulesets / court_decisions / bids / invoice_registrants + cross-dataset glue) と 33 autonomath (entity-fact DB 由来: 17 V1 + 4 V4 universal + 5 Phase A + 7 lifecycle/abstract/prerequisite/graph_traverse/snapshot/rule_engine)。36協定 pair は AUTONOMATH_36_KYOTEI_ENABLED gate で off (有効化で 74 tools)。詳細は /docs/mcp-tools/ で全 tool schema 公開。

Claude Desktop / Cursor / ChatGPT 全部動きますか?

Claude Desktop は MCP native サポート、即動作。Cursor は MCP 設定可。ChatGPT は GPTs の Action 経由で REST API を呼ぶ形。サンプル設定は quickstart 参照。

rate limit は?

匿名 (key なし) は 50 req/月 per IP (JST 月初リセット)、認証ありは hard cap なし (UTC 月次)。スパイク時も 429 にしない設計。同時接続は MCP/HTTP 含めて事実上無制限。

OpenAPI spec はどこ?

/docs/openapi/v1.json で 3.1 準拠の spec を公開。FastAPI から自動生成、Stoplight / Postman / Insomnia 等で import 可能。

PyPI / npm パッケージは?

PyPI: autonomath-mcp (uvx で即実行可)、内部 import path は jpintel_mcp (legacy)。npm 版は当面提供予定なし。MCP は stdio なので uvx / pipx での起動が標準。

自社 SaaS に組み込んだ場合の課金は?

貴社の API key 1 本で全 req を走らせれば metered ¥3/req。エンドユーザーへの転売は禁止していませんが、AutonoMath ブランドを表示する義務もありません (white-label OK)。

省庁サイトを自分で scrape すれば無料では?

技術的には可能ですが、現実は: (1) 47 都道府県 × 4 ministries × 数百自治体 で robots.txt / 認証 / PDF / Word / 画像化 PDF / 複合文書が混在し、最初の 1 制度を取るのに 2-4 時間。13,578 制度を維持するには月 100 時間以上。(2) 排他ルール 181 本は人間が一次資料を読まないと推論できない (LLM では幻覚する)。(3) 一次資料が更新されると source_url + fetched_at の追跡が必要。50 req/月の Free 枠で十分なら無料、超えたら ¥3/req — ご自身の時給で割り戻して判断してください。

関連 audience は?

同じ MCP/REST を別の切り口で利用している peers: VC / M&A advisor (DD 自動化)、税理士 (税制 ruleset 照会)、行政書士 (許認可 batch 照会)、SMB 経営者 (LINE bot 経由の最終消費者)。5 audience 全体を比較。
