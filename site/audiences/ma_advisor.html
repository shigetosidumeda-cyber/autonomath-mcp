M&Aアドバイザー向け

法人 360° (3-axis scoring)、DD deck (60 question × 7 category)、補助金返還義務 chain、業法 fence 検証、事業承継 制度 mapping を Claude Desktop 経由で 1 query。¥3/billable unit 完全従量、anonymous 3 req/日 free per IP。

業法 fence

M&A 仲介業者登録制度 + 弁護士法 §72 fence (重要): jpcite はM&A仲介の代行・法律判断・税務代理を一切行いません。本サービスは事実検索 + DD scaffold + 一次 URL 提供のみ、最終 DD 判断・契約交渉・締結はアドバイザー本人および有資格者 (弁護士・会計士・税理士) の専権事項です。中小企業庁 M&A仲介業者登録制度・自主規制ルールに反する代行は一切行いません。

MCP install (Claude Desktop)

Claude Desktop の claude_desktop_config.json に下記を追加してください。

{
  "mcpServers": {
    "jpcite": {
      "command": "uvx",
      "args": ["autonomath-mcp"]
    }
  }
}

paid usage を有効化する場合は env に API key を追加。anonymous 3 req/日 free per IP は API key なしで利用可 (JST 翌日 00:00 リセット)。案件ごとの fan-out は X-Client-Tag header (案件 ID 連動) で属性付与可能。
