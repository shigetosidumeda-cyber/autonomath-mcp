司法書士向け制度・登記情報検索を Claude Desktop で

商業登記前の houjin 360°・不動産登記 jurisdiction cross-check・簡裁訴訟代理前の法人実態 verify・後見業務 制度 mapping を Claude Desktop 経由で 1 query。jpcite が司法書士法 §3 fence を尊重しながら事実検索のみ提供します。¥3/billable unit 完全従量、anonymous 3 req/日 free。

司法書士法 §3 fence (重要): jpcite は登記・供託・簡裁訴訟代理 (司法書士法 §3 司法書士独占業務) を一切代行しません。本サービスは scaffold + 一次 URL 提供のみ、申請書面 generation 不可。登記申請書類の作成・提出・代理出頭は有資格司法書士の専権事項です。

4 つの活用ケース

1. 商業登記前 houjin 360°

受任候補の前歴・enforcement (fine / grant_refund / subsidy_exclude)・業務範囲 (定款目的・JSIC) を 1-call で。

example query: 「法人番号 1234567890123 の 過去 5 年の代表者変更履歴と enforcement 履歴」

推奨 tool: intel_houjin_full + check_enforcement_am

// MCP sample call (¥3 × 2 = ¥6)
intel_houjin_full({"houjin_bangou": "1234567890123"})
check_enforcement_am({"houjin_bangou": "1234567890123"})

2. 不動産登記前 cross_check_jurisdiction

本店 (法務局) vs 営業所 (NTA invoice) vs 不動産所在地 の 3 軸不一致検出、登記申請先確定手前の sanity check。

example query: 「法人 X の本店 vs 営業所 vs 不動産所在地 jurisdiction 確認」

推奨 tool: cross_check_jurisdiction

// MCP sample call (¥3)
cross_check_jurisdiction({"houjin_bangou": "1234567890123"})

3. 簡裁訴訟代理前 法人実態 verify

相手方 (140 万円以下) の公開情報、行政処分、補助金除外情報などを時系列で確認。訴額判定と代理可否は司法書士本人判断。

example query: 「法人 X の 直近 与信 + 倒産 risk + 反社チェック 相当」

推奨 tool: shihoshoshi_dd_pack_am (新 tool)

// MCP sample call (¥3)
shihoshoshi_dd_pack_am({"houjin_bangou": "1234567890123"})

4. 後見業務 制度 mapping

成年後見・任意後見・関連補助金 (中核機関整備 / 市民後見人養成) timeline と民法条文 を横断検索。

example query: 「成年後見 + 補助金 関連制度 一覧」

推奨 tool: search_programs + get_law_article_am (民法 §7-§876)

// MCP sample call (¥3 × 2 = ¥6)
search_programs({"q": "成年後見", "limit": 10})
get_law_article_am({"law_id": "minpou", "article_range": "7-876"})

よくある質問

Q1. 商業登記書類を jpcite が作成してくれますか?

いいえ。jpcite は事実検索のみで、書類は一切作成しません。登記申請書類の作成・提出は司法書士法 §3 司法書士独占業務であり、有資格者の専権事項です。jpcite は商業登記情報・enforcement 履歴・jurisdiction 不一致等の事実 surface と一次 URL 提供に留まります。

Q2. 成年後見業務支援は可能ですか?

成年後見・任意後見・関連補助金等の制度情報の検索と一次 URL 引用は可能です。後見人選任申立書面の作成や後見人候補者の判断は jpcite では一切行いません。司法書士本人の業務範囲です。

Q3. 反社チェックの代替になりますか?

法人の行政処分履歴や補助金除外情報の事実検索は可能ですが、個人信用情報や反社データベースは一切扱いません (APPI 配慮)。反社チェック相当の判断は司法書士本人が責任を持って行ってください。

Q4. pricing は?

¥3 / 課金単位 完全従量 (税込 ¥3.30)。anonymous は IP ベースで 3 req/日 free、JST 翌日 00:00 リセット。tier・seat・年間最低額は一切ありません。

Q5. 他の sensitive cohort (税理士・公認会計士・補助金 consultant) との違いは?

各 cohort 専用の persona prompt と tool routing が分かれています。司法書士向けは登記・供託・簡裁訴訟代理 (司法書士法 §3) を fence、税理士向けは税務代理・税務相談 (税理士法 §52) を fence、公認会計士向けは監査・会計参与 (公認会計士法 §47条の2) を fence、補助金 consultant 向けは行政書士法 §1の2・§19 業務範囲を fence します。
