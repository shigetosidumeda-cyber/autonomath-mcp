Claude / Cursor / ChatGPT / Codex に jpcite を接続する

jpcite は回答生成前に一次資料 URL・取得時刻・known gaps・8 業法 fence を返す evidence layer です。接続後は Evidence Packet、会社フォルダ、M&A DD / 取引先公開情報チェック、月次レビュー、保存検索通知を普段の AI agent から呼び出せます。
