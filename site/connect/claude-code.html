Claude Code から jpcite を 30 秒で接続

claude mcp add jpcite -- uvx autonomath-mcp 1 行で 139 tool 解禁。anonymous は 3 req/IP/日まで無料、paid は ¥3/req metered。

5 分接続手順

  1. uvx をインストール: curl -LsSf https://astral.sh/uv/install.sh | sh を実行し、シェルを再起動。
  2. jpcite を MCP server に登録: claude mcp add jpcite -- uvx autonomath-mcp (PyPI 名は legacy 維持で autonomath-mcp、ブランドは jpcite)。
  3. 接続確認: claude mcp list | grep jpcite が緑表示なら成功。
  4. (任意) API key を設定: 3 req/日上限を外す場合は export JPCITE_API_KEY=sk_...。発行は /pricing.html#api-paid から。
  5. 動作確認: 次節の verification コマンドで 139 tool が見えるか確認。

copy-paste snippet

~/.claude.json に直接書く場合は以下を MCP servers ブロックに追加。

{
  "mcpServers": {
    "jpcite": {
      "command": "uvx",
      "args": ["autonomath-mcp"],
      "env": {
        "JPCITE_API_KEY": "sk_xxxxx"
      }
    }
  }
}

API key 不要 (anon free 3 req/IP/日) の場合は env ブロック自体を省略可能。

動作確認

下記コマンドで 139 が返れば接続完了。

claude mcp call jpcite list_tools 2>&1 | jq '.tools | length'

期待値: 139 (default gates 有効時。AUTONOMATH_36_KYOTEI_ENABLED=false がデフォルトで、競艇 tools は含まれません)。

FAQ

anonymous と paid の違いは?

anon は IP ベースで 3 req/日 (AnonIpLimit)、API key 無し。paid は ¥3/req metered で上限なし、JPCITE_API_KEY 経由でカウント。

API key はどう発行する?

/pricing.html#api-paid から Stripe Checkout (consent は custom_text.submit.message で表示) で発行。発行直後にダッシュボードで sk_... を表示。

7 業法 fence とは?

jpcite は税理士法§52・弁護士法§72 等の独占業務に抵触する個別助言を返しません。/legal-fence.html 参照。
