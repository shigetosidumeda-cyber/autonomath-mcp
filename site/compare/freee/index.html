当社を選ぶケース

  • API + MCP。LLM Agent / 自社プロダクトに直接組み込める 96 tools。freee 助成金 AI の外部 API は 2026-04 時点で公開情報なし。
  • 判例 + 行政処分 + 法令 + 適格請求書の横断検索。freee 助成金 AI は助成金中心で、これらは対象外。

freee 助成金AI が向いている

移行 / 併用ガイド
