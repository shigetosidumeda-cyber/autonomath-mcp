ブラウザから 1 click で日本の制度を引く

税理士・コンサル・記者が Web 巡回中に出会った 法人番号 (13桁) や 制度名 を、選択 → 1 click で jpcite に照会できる bookmarklet と Chrome 拡張です。
匿名 3 req/日 無料 (IPベース、JST 翌日 00:00 リセット)。API キー利用時は従量課金対象単位ごとに ¥3 税抜 (税込 ¥3.30)。

1. ブックマークレット (今すぐ使える)

下のボタンをブラウザのブックマークバーに ドラッグ&ドロップ してください。
以降、任意のページでテキストを選択 → ブックマークをクリックで jpcite に照会します。 選択無しの場合は入力プロンプトが出ます。

jpcite で照会 ← ブックマークバーへドラッグ&ドロップ
ソースコード (JavaScript 1 line) を表示 
javascript:(function(){var s=((getSelection()+'')||prompt('jpcite 検索:')||'').trim(),d=s.replace(/\D/g,''),b='https://api.jpcite.com/v1/';s&&open(/^T?\d{13}$/i.test(s)?b+'houjin/'+d:b+'programs/search?q='+encodeURIComponent(s),'_blank')})()

動作: 選択テキスト → 13桁数字 (T プレフィックス可) なら /v1/houjin/{n}、 それ以外なら /v1/programs/search?q=<encoded> を新タブで開く。 外部 LLM API は呼び出さず、選択した文字列で jpcite の検索ページを開きます。

使い方 3 step

  1. ブラウザのブックマークバーを表示 (Cmd+Shift+B / Ctrl+Shift+B)。
  2. 上の「jpcite で照会」ボタンをブックマークバーへドラッグ&ドロップ。
  3. 任意の Web ページで法人番号 / 制度名を選択 → ブックマークをクリック。

2. Chrome 拡張 (右クリック menu + popup form)

bookmarklet より一歩深い体験。13桁数字を hover でハイライト、 右クリック menu「jpcite で照会」、ツールバーアイコンから popup form。

  • 権限は contextMenus のみ (拡張は fetch せず、ブラウザにタブを開かせるだけ — bookmarklet と同等のプライバシー)。
  • 外部 LLM API は呼び出しません。
  • 料金は同じ。匿名 3 req/日 無料、API キー利用時は従量課金対象単位ごとに ¥3 税抜。
Chrome 拡張を手動インストール

Chrome 拡張の手動インストール手順

  1. 下の「unpacked 読み込み」手順で拡張フォルダを Chrome に追加します。
  2. ツールバーに jpcite アイコンが現れます。
  3. 任意のページで 13桁数字を hover → ハイライト → クリックで照会。

公開前 / 開発者向け unpacked 読み込み

  1. GitHub から sdk/chrome-extension/ を clone。
  2. chrome://extensions/ を開き、右上「デベロッパーモード」 ON。
  3. 「パッケージ化されていない拡張機能を読み込む」→ sdk/chrome-extension/ を指定。

3. ブックマークレット vs Chrome 拡張 — 比較

項目 ブックマークレット Chrome 拡張
配布形態 本ページから drag drop (即使える) Chrome Web Store (公開には ¥750 + Google アカ要)
起動 UX テキスト選択 → ブックマーククリック (2 step) 右クリック menu / 13桁 hover クリック / popup form (1 step)
13桁の自動検出 無し (選択時のみ判定) 有り (hover ハイライト + click)
対応 ブラウザ Chrome / Edge / Firefox / Safari 全部 Chrome / Edge (Chromium 系)
権限要求 無し (ページ DOM のみ) contextMenus のみ (host_permissions 空)
同期 (PC 間) ブラウザのブックマーク同期で OK Chrome アカウント同期で OK
配布 SEO 本ページのみ Chrome Web Store 検索流入が追加
推奨 まずはこちら。 インストール不要ですぐ使えます。 右クリックや hover 検出を使いたい利用者向けです。

4. 想定 user

税理士・会計士・行政書士 (顧問先 fan-out)
国税庁 法人番号公表サイト・gBizINFO・金融機関の与信書類で 13桁を見つけた瞬間に jpcite で /v1/houjin/<n> を叩く。採択履歴・行政処分・適格請求書登録状況が 1 click で揃う。
補助金 / 経営コンサル
METI / 中小企業庁 / 都道府県の制度ニュースを巡回中、制度名を選択 → 即 jpcite で類似制度・併給排他ルール・申請ラウンドを照会。
記者・調査会社・M&A デューデリ
企業 IR / 報道記事に出てくる 13桁法人番号を hover でハイライトしてくれる。クリックで採択額・補助金履歴・enforcement (行政処分) を即横断確認。
SMB 経営者 / 個人事業主
商工会・信金のメルマガで気になる制度名を選択 → そのまま jpcite で詳細。匿名 3 req/日でほぼ用が足りる。

5. 何が短くなるか

本ツールは、ブラウザ上の調査作業から jpcite の検索へすばやく移動するための補助機能です。 コピー、別タブ移動、検索 URL 入力、法人番号の整形を省き、一次資料付きの確認に進みやすくします。

場面 通常の手順 このツール
法人番号を調べる 13桁をコピーし、検索ページを開き、貼り付ける 選択してクリック。13桁なら自動で法人情報照会へ送ります。
制度名を確認する 制度名をコピーし、jpcite で検索し直す 選択語をそのまま制度検索に渡します。
AI に渡す前の根拠確認 Web 検索結果や PDF を読み直して出典を探す jpcite の構造化レスポンスで、出典 URL と取得時刻を先に確認できます。
