通知チャネル

現行の通知はメールから開始できます。相談パック更新・保存検索・締切前リマインドを、検索や相談票とは別の通知チャネルとして扱います。

メール通知を受け付けています。Evidence Packet / API を今すぐ試す場合は Playground または Pricing をご確認ください。

通知で受け取れるもの

相談パック更新通知

根拠・known gaps・質問票の更新

顧問先メモや相談パックに新しい候補、締切、確認事項が出た時に、一次資料付きの確認導線へ戻れる通知を受け取れます。

保存検索・締切リマインド

候補制度の見落とし防止

保存した検索条件や候補制度について、応募開始・締切前の確認を促す通知を受け取れます。

外出先の確認導線

詳細は Web / Docs / 専門家レビューへ

LINE 上で結論を断定せず、必要な根拠、未確認事項、次に確認すべき画面へ短く戻す導線にします。

本利用

有料通知フローは条件を事前表示

保存検索・締切リマインド

API / MCP / Playground の料金は Pricing に掲載

  • 通知はリマインド・確認導線として扱います
  • 有料通知フローを有効にする前に条件を画面で表示します
  • Pricing で現行の API 利用条件を確認できます

通知を受け取る

保存検索・締切リマインドの通知をメールで受け取れます。メール以外の情報は取得しません。解除はワンクリック。

コンプライアンス

通知配信と不正利用防止のため、メールアドレスなど必要最小限の識別子を保存します。詳細は プライバシーポリシー。
