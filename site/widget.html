あなたのサイトに
補助金検索を 1 行で

税理士事務所・商工会議所・中小企業支援サイトの HP に <script> タグ 1 行で埋め込める補助金検索 widget。フィルタ UI・結果リスト・一次資料リンクを自動で生成し、 お客様は「自社サイトでそのまま検索できる」体験を得られます。

<script async src="https://jpcite.com/widget/jpcite.js"></script> <div data-jpcite-widget data-key="wgt_live_YOUR_KEY"></div>

なぜ widget が使われるか

組み込みは 1 行

script タグを貼って div を置くだけ。React / Vue に依存せず、 ビルド不要で既存の HP にそのまま差せます。

取得時刻を表示

jpcite の公開データを参照し、一次資料リンク、取得時刻、 締切候補を widget に反映します。

Origin 制限で安全

キーは登録した origin からのみ使用可能。広告のクリックボムや 第三者サイトでの無断流用を防ぎます。

UI は自社サイトに馴染む

light / dark テーマ + CSS オーバーライド対応。既存サイトの 配色・フォントに合わせて違和感なく埋め込めます。

一次資料リンクで信頼性

全制度に中央省庁・自治体・日本政策金融公庫等の公式 URL を リンク。来訪者は「ちゃんとした根拠」を直接確認できます。

セルフサーブ・完全従量

サイト運営者の wgt_live_... key 利用分だけを ¥3/課金単位で従量課金。サイト訪問者へ jpcite から直接請求しません。

料金プラン

Widget は公開API/Playgroundの匿名評価枠とは別です。 発行された wgt_live_... key の利用分をサイト運営者に請求します。

申し込みフォーム

下記を記入して「決済に進む」を押すと Stripe Checkout に遷移します。 決済完了後、wgt_live_... キーが管理ダッシュボード経由で発行されます。

wgt_live_... はブラウザ用の widget key です。 HTML に置く前提の key で、登録した origin からの /v1/widget/* 呼び出しだけに使います。サーバー/API 用の am_... key とは別物です。 課金はサイト訪問者ではなく、申し込んだ widget owner の利用分として計上されます。

公開API/Playgroundの匿名評価枠とは別です。Widget は発行済み wgt_live_... key の利用分を widget owner に請求します。

Origin は https://example.com のような scheme + host + 任意の port です。 path、query、末尾 slash は不可です。https://*.example.com はサブドメイン用で、 https://example.com 自体は含みません。本番、www、staging は必要に応じて別行で登録してください。

最低利用期間・解約違約金はありません。Stripe Customer Portal から いつでもキャンセル可能。消費税 10% は Stripe が自動計算、 適格請求書は Stripe から発行されます。登録情報は特商法表記をご確認ください。
