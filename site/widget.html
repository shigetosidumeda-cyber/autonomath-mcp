なぜ widget が使われるか

組み込みは script と div

script タグを読み込み、設置場所に div を置くだけ。React / Vue に依存せず、 ビルド不要で既存の HP にそのまま差せます。

取得時刻を表示

jpcite の公開データを参照し、一次資料リンク、取得時刻、 締切候補を widget に反映します。

設置サイトURLで安全

キーは登録したサイトURLからのみ使用可能。広告のクリックボムや 第三者サイトでの無断流用を防ぎます。

UI は自社サイトに馴染む

light / dark テーマ + CSS オーバーライド対応。既存サイトの 配色・フォントに合わせて違和感なく埋め込めます。

一次資料リンクで信頼性

主要な公開レコードでは、中央省庁・自治体・日本政策金融公庫等の 公式 URL を可能な範囲で表示します。出典 URL がない行や確認範囲外の 項目は、known gaps や再確認対象として扱います。

セルフサーブ・完全従量

サイト運営者のブラウザ用キーで正常処理された検索リクエストを 1 件 ¥3 税別 (税込 ¥3.30) で従量課金。サイト訪問者へ jpcite から直接請求しません。

申し込みフォーム

下記を記入して「決済に進む」を押すと Stripe Checkout に遷移します。 決済完了後、このブラウザにブラウザ用キーと埋め込みコードを表示します。

ここで発行されるキーは、サイト埋め込み専用のブラウザ用キーです。 HTML に置く前提の key で、登録したサイトURLからの widget 呼び出しだけに使います。 サーバー/API 用の jc_... key とは別物です。 課金はサイト訪問者ではなく、申し込んだ widget owner の key で正常処理された検索リクエストとして計上されます。

公開API/Playgroundの匿名評価枠とは別です。Widget は発行済みブラウザ用キーで正常処理された検索リクエストを widget owner に請求します。

ここに登録したサイトだけが widget を呼び出せます。https://example.com のように入力してください。 path、query、末尾 slash は不要です。https://*.example.com はサブドメイン用で、 https://example.com 自体は含みません。本番、www、管理用サブドメインは必要に応じて別行で登録してください。

最低利用期間・解約違約金はありません。Stripe Customer Portal から いつでもキャンセル可能。消費税 10% は Stripe が自動計算、 適格請求書は Stripe から発行されます。登録情報は特商法表記をご確認ください。
