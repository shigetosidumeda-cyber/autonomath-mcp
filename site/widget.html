あなたのサイトに
補助金検索を 1 行で

税理士事務所・商工会議所・中小企業支援サイトの HP に <script> タグ 1 行で埋め込める補助金検索 widget。フィルタ UI・結果リスト・一次資料リンクを自動で生成し、 お客様は「自社サイトでそのまま検索できる」体験を得られます。

<script async src="https://jpcite.com/widget/autonomath.js"></script> <div data-autonomath-widget data-key="wgt_live_YOUR_KEY"></div>

※ 上記は表示例です。許可された origin を登録した本番キーでは お客様のサイト上で正常に結果が表示されます。 詳細は ドキュメント をご覧ください。

なぜ widget が使われるか

組み込みは 1 行

script タグを貼って div を置くだけ。React / Vue に依存せず、 ビルド不要で既存の HP にそのまま差せます。

検索結果は常に最新

jpcite 本体の 11,684 制度 DB を参照。毎晩更新される 一次資料リンクと締切を widget にそのまま反映します。

Origin 制限で安全

キーは登録した URL からのみ使用可能。広告のクリックボムや 第三者サイトでの無断流用を防ぎます。

UI は自社サイトに馴染む

light / dark テーマ + CSS オーバーライド対応。既存サイトの 配色・フォントに合わせて違和感なく埋め込めます。

一次資料リンクで信頼性

全制度に中央省庁・自治体・日本政策金融公庫等の公式 URL を リンク。来訪者は「ちゃんとした根拠」を直接確認できます。

セルフサーブ・月額定額

Business 以上で included 10,000 リクエスト/月。お問い合わせ不要で Stripe Checkout 完了後、すぐキー発行されます。

料金プラン

完全従量の API (税込 ¥3.30/リクエスト) とは別プランです。サイト埋め込みのニーズに合わせて 月額ベースで設計しています。

Business

¥10,000 / 月 (税別)

10,000 リクエスト/月 含む、超過分 税込 ¥3.30/リクエスト

  • 税理士事務所・支援機関の HP 向け
  • 10,000 リクエスト/月 が標準利用分として含まれる
  • 超過は税込 ¥3.30/リクエスト を Stripe usage-based で自動請求
  • 「powered by jpcite」フッターを表示
  • 複数 origin (本番 + staging 等) 登録可
  • 適格請求書 を Stripe が自動発行

ホワイトラベル

¥30,000 / 月 (税別)

100,000 リクエスト/月 fair use、ブランド表示なし

  • 自社ブランドで提供する商工会議所・金融機関向け
  • 「powered by jpcite」フッターを非表示に
  • 100,000 リクエスト/月 (fair use) まで追加課金なし
  • Business と同じ origin 制限・一次資料リンク
  • 月 100,000 リクエスト を大幅超過した場合のみ個別連絡
  • 適格請求書 を Stripe が自動発行
