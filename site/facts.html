ファクトシート — 統計の出典

jpcite のサイト・ドキュメント・営業資料に登場する数値の基準ページです。各値は /v1/meta API を一次出典とし、更新日 (JST) を明記しています。サイト内の各数字は <span data-fact="ID"> でマークアップされ、本ページの <dt id="fact-ID"> に 1:1 で対応します。

最終更新: 2026-04-30 JST · 一次出典: GET /v1/meta · ライセンス: 利用規約 準拠 (各データセットの元ライセンスは出典欄を参照)

引用方法

記者・研究者・パートナーがサイトの数値を引用する場合、以下のいずれかの形式を推奨します:

  • URL アンカー: https://jpcite.com/facts.html#fact-searchable_programs_total を直接リンク
  • API 直叩き: curl https://api.jpcite.com/v1/meta (匿名 3 リクエスト/日 per IP 無料)
  • テキスト引用: 「jpcite ファクトシート (2026-04-26 JST 更新) より」

よくある質問

サイトに表示されている数字 (例: 11,601 制度) の出典は?

本ファクトシート (https://jpcite.com/facts.html) を基準にしています。各値は /v1/meta API を一次出典とし、JST 更新日を併記しています。サイト内の各数字は <span data-fact="..."> でマークアップされており、ファクトシート上の <dt id="fact-..."> と 1:1 対応します。

数字はどの頻度で更新されますか?

/v1/meta は 60 秒キャッシュ付きでライブ集計されます。本ファクトシートはデータ取り込みサイクル (週次〜月次) と料金改訂のタイミングで再生成し、最終検証日を記載します。

Tier S/A/B/C の意味は?

実データ充足度に基づく品質ラベルです。Tier S/A/B/C は検索対象、検索対象外の制度は検索対象外。詳細は /docs/exclusions/ を参照。
