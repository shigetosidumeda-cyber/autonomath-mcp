ファクトシート — 統計の出典

AutonoMath のサイト・ドキュメント・営業資料に登場する数値の正本です。各値は /v1/meta API を一次出典とし、取得日 (JST) を明記しています。サイト内の各数字は <span data-fact="ID"> でマークアップされ、本ページの <dt id="fact-ID"> に 1:1 で対応します。

最終更新: 2026-04-26 JST · 一次出典: GET /v1/meta · ライセンス: 利用規約 準拠 (各 sub-dataset の元ライセンスは出典欄を参照)

引用方法

記者・研究者・パートナーがサイトの数値を引用する場合、以下のいずれかの形式を推奨します:

  • URL アンカー: https://autonomath.ai/facts.html#fact-searchable_programs_total を直接リンク
  • API 直叩き: curl https://api.autonomath.ai/v1/meta (匿名 50 req/月 per IP 無料)
  • テキスト引用: 「AutonoMath ファクトシート (2026-04-26 JST 取得) より」

よくある質問

サイトに表示されている数字 (例: 10,790 制度) の出典は?

本ファクトシート (https://autonomath.ai/facts.html) が正本です。各値は /v1/meta API を一次出典とし、JST 取得日を併記しています。サイト内の各数字は <span data-fact="..."> でマークアップされており、ファクトシート上の <dt id="fact-..."> と 1:1 対応します。

数字はどの頻度で更新されますか?

/v1/meta は 60 秒キャッシュ付きでライブ集計されます。本ファクトシートは ingest cadence (週次〜月次) と料金改訂のタイミングで再生成し、Last verified 列に最新の手動クロスチェック日を記載します。

Tier S/A/B/C の意味は?

実データ充足度に基づく品質ラベルです。Tier S/A/B/C は検索対象、Tier X は quarantine (検索除外)。詳細は /docs/exclusions/ を参照。
