## 選び方

用途別の素直な推奨です。jpcite が常に最適ではありません。

中小企業 / 経営者

### 人が数件見るだけなら無料サイトで十分

自社で 1〜数件の補助金を探すだけなら、まず ミラサポ plus と J-Net21 (どちらも中小機構運営・無料・公的) が出発点になります。

一方で、AI・顧問先管理・CRM・監査メモへ組み込むなら、HTML を読むより jpcite の API/MCP が向いています。一次資料URL、取得時刻、採択事例、行政処分、法令メタ、known gaps を同じ形式で返せるため、会社フォルダや相談前プレ診断を ¥3/課金単位で自動化できます。

税理士 / 行政書士 / 中小企業診断士

### 顧客横断で扱うなら一次資料連結が必須

顧問先 10 社以上を扱う士業の方は、二次情報サイトの古い金額や終了済み制度をそのまま使うと事故になります。一次資料 URL と取得時刻を確認できる仕組みが必要です。

jpcite は、主要公開レコードの出典 URL + 取得時刻、e-Gov 法令へのリンク、行政処分 1,185 件を API/MCP から参照できます。Claude Desktop / Cursor / Cline は MCP、ChatGPT Custom GPT は OpenAPI Actions で同じ根拠データを取得できます。

開発者 / AI Agent 構築

### API/MCP を重視する場合

主要な補助金検索サイト・支援ポータルの多くは、公開 API や MCP サーバーを提供していません (2026-04-26 時点)。AI から呼ぶには HTML スクレイピング + 個別 TOS 確認が必要です。

jpcite は REST + MCP + OpenAPI 仕様書 + 151 機能を公開しています。匿名 3 リクエスト/日で評価でき、本番は 税別 ¥3 / 税込 ¥3.30 / 課金単位 完全従量、SDK 不要です。
