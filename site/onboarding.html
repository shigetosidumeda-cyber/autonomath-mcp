¥0 から始めて 4 step で API を呼ぶ

登録なしで匿名 3 req/日/IP 試せます。気に入ったらサインイン → 課金設定 → API/MCP 呼び出しへ進みます。月額固定なし、契約期間なし、解約手数料なし。

step 1 / 4

1匿名 3 req/日/IP を無料で試す

登録不要。playground で実際の REST 応答を 3 回まで試せます。JST 翌日 00:00 に枠がリセットされ、自然なペースで使い続けられます。 /v1/openapi.json と /healthz はいつでも free quota の外で公開しています。

確認ポイント: まず 3 req で、出典 URL、取得時点、known gaps、料金単位の見え方を確認してください。

3 req 無料で試す → サインインから始める

2GitHub OAuth または magic link で 1 click サインイン

GitHub アカウント連携または email magic link を選べます。Stripe アカウント・カード入力は後段で必要なときだけ作成、ここでは 無料枠の拡張と利用量ダッシュボードが解放されるだけです。

迷子ゼロ要素: ダッシュボード上部に常に「次の step は X」と「残り req / 課金設定」を表示。30 秒操作なしで hint modal が出ます。

サインインする → 先に topup を見る

3Stripe 従量課金と上限を確認する

主経路は Stripe 従量課金です。利用可能な Credit Wallet / x402 などの追加 API rail はダッシュボード表示に従って確認します。月次上限、X-Cost-Cap-JPY、Idempotency-Key を使うと、広い batch や fan-out でも費用を実行前に止められます。

  • Stripe 従量課金: 月末締め・適格請求書自動発行 (T8010001213708)
  • Credit Wallet: 表示されている場合は前払い topup と spending alert で小さく開始
  • x402: 表示されている場合は agent-to-agent 実行での自動支払い経路
  • 月次上限: ダッシュボードで上限到達時に API を停止
  • request 上限: X-Cost-Cap-JPY で高額実行を直前に停止

確認ポイント: まず小さい request 数で試し、広い実行は事前見積もりと上限を入れてから動かしてください。

課金設定を確認する → 先に API キーを見る

4API / MCP を呼ぶ

ダッシュボードで API キーを発行するか、MCP server URL を Claude Desktop / Cursor / Continue に登録するだけです。1 呼び出し = 税込 ¥3.30。 X-Cost-Cap-JPY で予算上限、Idempotency-Key で二重課金抑制、X-Client-Tag で顧客別原価集計。

費用の目安: 顧問先 100 社月次レビュー想定では、1,800 req × ¥3.30 = 税込 ¥5,940 / 月。実際の req 数は対象件数・取得項目・batch 設計で変わります。

API キーを発行 → MCP 接続ガイドを読む
4 step を一行で: 1. 試す → 2. サインイン → 3. 課金設定 → 4. API 呼出。 迷ったら下部の feedback widget から info@bookyou.net へ 1 click 送信できます。

よくある質問

登録なしでどこまで試せますか?

同一 IP から 3 req/日まで実際の本番 API を呼べます。/v1/openapi.json と /healthz は free quota の外で常時公開。

課金設定では何を確認すべきですか?

まず月次上限と request 単位の上限を確認してください。標準は Stripe 従量課金で、通常の API/MCP 呼び出しは 1 billable unit = ¥3 税別です。

解約は?

ダッシュボードから Stripe Customer Portal で 1 click キャンセル。当月利用分のみ ¥3/billable unit で精算、解約違約金・最低利用期間なし。

料金が変動することはありますか?

通常検索・詳細は固定 ¥3/billable unit。batch / export は事前見積もり (POST /v1/cost/preview) で units と金額を表示してから実行できます。
