¥0 から始めて 4 step で API を呼ぶ

登録なしで匿名 3 req/日 試せます。気に入ったら 1 click サインイン → 3 ルートから topup → API/MCP を呼び出すだけ。月額固定なし、契約期間なし、解約手数料なし。

step 1 / 4

1匿名 3 req/日を無料で試す

登録不要。playground で実際の REST 応答を 3 回まで試せます。JST 翌日 00:00 に枠がリセットされ、自然なペースで使い続けられます。 /v1/openapi.json と /healthz はいつでも free quota の外で公開しています。

節約額の目安: 1 案件取りこぼし回避で ¥36-120万 / 年 (顧問契約解除リスク回避、historical ROI 1,000-3,600 倍 表現)。

3 req 無料で試す → サインインから始める

2GitHub OAuth または magic link で 1 click サインイン

GitHub アカウント連携または email magic link を選べます。Stripe アカウント・カード入力は後段で必要なときだけ作成、ここでは 無料枠の拡張と利用量ダッシュボードが解放されるだけです。

迷子ゼロ要素: ダッシュボード上部に常に「次の step は X」と「残り req / topup 状況」を表示。30 秒操作なしで hint modal が出ます。

サインインする → 先に topup ルートを比較

33 ルートから topup を選ぶ

Credit Wallet (¥前払い + auto-topup default ON)、x402 USDC (API キー不要・2 秒決済)、Stripe 自動課金の 3 ルートを並列に用意。1 プロトコル絞り禁止で運用環境を選びません。

  • Credit Wallet: ¥1,000 単位の前払い、auto-topup default ON、spending alert 50%/80%/100%
  • x402 USDC: HTTP 402 challenge → USDC ($0.001 から)、API キー不要
  • Stripe 自動課金: 月末締め・適格請求書自動発行 (T8010001213708)

節約額の目安: auto-topup で 0 分中断。x402 なら登録ステップを完全 skip して即課金可能。

topup する → 先に API キーを見る

4API / MCP を呼ぶ

ダッシュボードで API キーを発行するか、MCP server URL を Claude Desktop / Cursor / Continue に登録するだけです。1 呼び出し = 税込 ¥3.30。 X-Cost-Cap-JPY で予算上限、Idempotency-Key で二重課金抑制、X-Client-Tag で顧客別原価集計。

節約額の目安: 顧問先 100 社月次レビュー = ¥5,940 / 月 (1,800 req × ¥3.30、業法 fence 自動付与)。

API キーを発行 → MCP 接続ガイドを読む
4 step を一行で: 1. 試す → 2. サインイン → 3. topup → 4. API 呼出。 迷ったら下部の feedback widget から info@bookyou.net へ 1 click 送信できます。

よくある質問

登録なしでどこまで試せますか?

同一 IP から 3 req/日まで実際の本番 API を呼べます。/v1/openapi.json と /healthz は free quota の外で常時公開。

3 つの topup ルートのどれを選ぶべきですか?

個人/小規模なら Credit Wallet、AI agent 間 micro-payment なら x402 USDC、月末締め請求書が必要なら Stripe 自動課金。どれを選んでも単価 ¥3/req は変わりません。

解約は?

ダッシュボードから Stripe Customer Portal で 1 click キャンセル。当月利用分のみ ¥3/req で精算、解約違約金・最低利用期間なし。

料金が変動することはありますか?

通常検索・詳細は固定 ¥3/billable unit。batch / export は事前見積もり (POST /v1/cost/preview) で units と金額を表示してから実行できます。
