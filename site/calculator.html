ROI 試算

顧問先数 / 月次 確認 件数 / 1 件あたり削減時間 / 平均時間報酬 を入力すると、 jpcite API (¥3/リクエスト 税別、税込 ¥3.30) の月額コストと、 時間削減 + 機会発見の期待値を ブラウザ内で 計算します。 入力値はブラウザから外に出ません (送信なし、 LLM 呼び出しなし)。

入力

税理士事務所が抱える法人 / 個人クライアント数。 社
補助金 / 助成金 / 税制 / 法令改正 を 1 顧問先あたり 月に何回 確認 するか。 件 / 顧問先 / 月
人手 確認 (制度文書 + 一次資料の突合) を AI に置き換えた時の短縮分。 分 / 件
税理士 / 担当者の請求単価または機会費用 (時給換算)。 円 / 時間
確認 した制度のうち 本来適用できたが見落としていた 確率。 業界平均は 1〜5%、 ここでは 2% を初期値。 %
見落としを 1 件捕捉した時の顧問先側の交付決定額 平均値。 補助金 1 件 ¥1〜3M が標準、 初期値は保守的に ¥500,000。 円 / 件
