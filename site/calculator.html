カバレッジ試算

jpcite API は ¥3/リクエスト 税別 (税込 ¥3.30) の従量料金です。 顧問先数 / 月次 確認 件数 を入力すると、 月額 API コストと、 1 リクエストで実際に join される複数 DB ・出典本数・鮮度日数 を ブラウザ内で 提示します。 入力値はブラウザから外に出ません (送信なし、 LLM 呼び出しなし)。

※ 本ツールは「トークン削減 / コスト x% 削減」を保証するものではありません。 削減効果はモデル・プロンプト・キャッシュ・クエリ分布で大きく変動するため、 測定値の読み方は context savings の説明 をご参照ください。 ここでは jpcite が 1 リクエストで何を join するか のみを開示します。

本試算は情報提供目的の参考値です。 「1 リクエストで join される DB / 出典本数 / 鮮度日数」 はカタログ値であり、 個別クエリの返却件数は対象制度・地域・キーワードにより変動します。 トークン削減・AI コスト全体の増減はモデル・プロンプト・キャッシュ・クエリ分布で大きく変動するため、 当社ベンチ範囲外への外挿はできません ( context savings の説明 参照)。 jpcite は税理士法 §52 / §72 の遵守のため、 個別税務助言は提供せず、 一次資料の検索・整理に限定します。 税理士業務 / 会計士業務 の判断は必ず資格者にご相談ください。
