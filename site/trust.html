外部 LLM API を呼ばない設計

検証済み

jpcite のサーバー側処理は外部 LLM API を呼び出さず、構造化データと一次資料メタデータを返します。利用者側の AI クライアントは、その根拠を使って回答を組み立てます。

  • 回答生成の前に、一次資料 URL・取得時刻・根拠項目を取得できます
  • 長文資料を AI に渡す前の evidence prefetch レイヤーとして使えます
