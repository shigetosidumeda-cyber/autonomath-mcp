# chushin-shigaichi-no_2

# 中心市街地の活性化に関する法律第五十四条に規定する業務に係る食品等持続的供給推進機構に関する省令
法令番号 平成10年農林水産省令第63号 施行日 2025-10-01 最終改正 2025-09-18 e-Gov 法令 ID 410M50000200063 ステータス active

目次

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日）

（施行期日）第一条この省令は、卸売市場法及び食品流通構造改善促進法の一部を改正する法律（以下「改正法」という。）の施行の日（平成三十年十月二十二日）から施行する。
