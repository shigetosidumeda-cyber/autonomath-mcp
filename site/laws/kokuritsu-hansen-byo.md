# 国立ハンセン病療養所名誉所長の称号の授与に関する省令
法令番号 昭和39年厚生省令第19号 施行日 2010-04-01 最終改正 2010-03-31 e-Gov 法令 ID 339M50000100019 ステータス active

## 第1条 （施行期日）

（施行期日）第一条この省令は、公布の日から施行する。ただし、附則第八条から第十八条までの規定は、平成十六年四月一日から施行する。

## 第1_附2条 （施行期日）

（施行期日）第一条この省令は、平成二十二年四月一日から施行する。
