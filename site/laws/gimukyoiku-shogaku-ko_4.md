# gimukyoiku-shogaku-ko_4

# 義務教育諸学校の教科用図書の無償に関する法律 
法令番号 昭和37年法律第60号 施行日 1962-03-31 最終改正 1962-03-31 所管 mext カテゴリ 教育 e-Gov 法令 ID 337AC0000000060 ステータス active 

目次 

- [1 （趣旨） ](#art-1)

## 第1条 （趣旨） 

（趣旨）第一条義務教育諸学校の教科用図書は、無償とする。２前項に規定する措置に関し必要な事項は、別に法律で定める。
