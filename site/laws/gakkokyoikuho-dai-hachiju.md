# gakkokyoikuho-dai-hachiju

# 学校教育法第八十九条の規定を適用しない者を定める省令 
法令番号 平成11年文部省令第38号 施行日 2007-12-26 最終改正 2007-12-25 所管 mext カテゴリ 教育 e-Gov 法令 ID 411M50000080038 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この省令は、内閣法の一部を改正する法律（平成十一年法律第八十八号）の施行の日（平成十三年一月六日）から施行する。
