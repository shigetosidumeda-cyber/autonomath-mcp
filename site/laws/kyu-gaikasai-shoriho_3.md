# kyu-gaikasai-shoriho_3

# 旧外貨債処理法による借換済外貨債の証券の一部の有効化等に関する法律施行令に規定する書類の様式及び記載事項を定める省令 
法令番号 昭和27年大蔵省令第36号 施行日 2020-12-25 最終改正 2020-12-25 所管 fsa カテゴリ 金融 e-Gov 法令 ID 327M50000040036 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この省令は、公布の日から施行する。 

## 第2条 （経過措置） 

（経過措置）第二条４この省令（前条ただし書に掲げる規定にあっては、当該規定）の施行の際、現に存する改正前の様式又は書式による用紙は、当分の間、これを取り繕い使用することができる。
