# noyaku-torishimariho-daisan

# 農薬取締法第三条第一項の登録を要しない場合を定める省令 
法令番号 平成15年農林水産省・環境省令第2号 施行日 2018-12-01 最終改正 2018-11-30 所管 mhlw e-Gov 法令 ID 415M60001200002 ステータス active 

目次 

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日） 

（施行期日）第一条この省令は、農薬取締法の一部を改正する法律の施行の日（平成三十年十二月一日）から施行する。
