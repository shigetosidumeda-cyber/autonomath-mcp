# noyaku-torishimariho-ni

# 農薬取締法に基づく農薬の使用の禁止に関する規定の適用を受けない場合を定める省令 
法令番号 平成15年農林水産省・環境省令第1号 施行日 2018-12-01 最終改正 2018-11-30 所管 mhlw e-Gov 法令 ID 415M60001200001 ステータス active 

目次 

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日） 

（施行期日）第一条この省令は、農薬取締法の一部を改正する法律の施行の日（平成三十年十二月一日）から施行する。
