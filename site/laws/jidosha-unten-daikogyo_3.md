# jidosha-unten-daikogyo_3

# 自動車運転代行業の業務の適正化に関する法律の施行に伴う道路交通法施行規則の規定の読替えに関する内閣府令 
法令番号 平成14年内閣府令第35号 施行日 2023-04-01 最終改正 2022-12-23 所管 mlit カテゴリ 建設 e-Gov 法令 ID 414M60000002035 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この府令は、道路交通法の一部を改正する法律（以下「改正法」という。）の施行の日（平成十九年九月十九日）から施行する。
