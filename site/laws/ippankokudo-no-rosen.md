# ippankokudo-no-rosen

# 一般国道の路線を指定する政令
法令番号 昭和40年政令第58号 施行日 2004-04-01 最終改正 2004-03-19 e-Gov 法令 ID 340CO0000000058 ステータス active

目次

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日）

（施行期日）第一条この政令は、公布の日から施行する。ただし、附則第九条から第四十四条までの規定は、平成十六年四月一日から施行する。
