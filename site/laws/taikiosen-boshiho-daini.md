# taikiosen-boshiho-daini

# 大気汚染防止法第二条第十七項の自動車及び原動機付自転車を定める省令
法令番号 昭和43年運輸省令第58号 施行日 2021-04-01 最終改正 2020-10-15 e-Gov 法令 ID 343M50000800058 ステータス active

## 第1条 第一条

第一条大気汚染防止法（昭和四十三年法律第九十七号。以下「法」という。）第二条第十七項の環境省令で定める自動車は、道路運送車両法施行規則（昭和二十六年運輸省令第七十四号）第二条に規定する普通自動車、小型自動車、軽自動車、大型特殊自動車及び小型特殊自動車であつて、ガソリン、軽油又は液化石油ガス（プロパン・ガス又はブタン・ガスを主成分とする液化ガスをいう。）を燃料とするものとする。

## 第1_附2条 （施行期日）

（施行期日）第一条この省令は、大気汚染防止法の一部を改正する法律（次条において「改正法」という。）の施行の日（令和三年四月一日）から施行する。

## 第2条 第二条

第二条法第二条第十七項の環境省令で定める原動機付自転車は、ガソリンを燃料とする原動機付自転車とする。
