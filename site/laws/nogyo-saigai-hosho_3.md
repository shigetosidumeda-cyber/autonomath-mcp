# nogyo-saigai-hosho_3

# 農業災害補償法による果樹共済の共済目的たる果樹を指定する政令
法令番号 昭和50年政令第37号 施行日 2018-04-01 最終改正 2017-10-25 所管 maff カテゴリ 農業 e-Gov 法令 ID 350CO0000000037 ステータス repealed

目次

- [1 （施行期日） ](#art-1)
- [2 （都道府県農業共済保険審査会規程等の廃止） ](#art-2)

## 第1条 （施行期日）

（施行期日）第一条この政令は、平成三十年四月一日から施行する。

## 第2条 （都道府県農業共済保険審査会規程等の廃止）

（都道府県農業共済保険審査会規程等の廃止）第二条次に掲げる勅令及び政令は、廃止する。一及び二略三農業災害補償法による果樹共済の共済目的たる果樹を指定する政令（昭和五十年政令第三十七号）
