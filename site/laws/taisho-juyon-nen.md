# taisho-juyon-nen

# 大正十四年逓信省令第八十七号（仏蘭西国船舶ノ検査ニ関スル件）
法令番号 大正14年逓信省令第87号 施行日 1945-05-19 最終改正 1945-05-19 カテゴリ 運輸 e-Gov 法令 ID 214M10001000087 ステータス active

目次

## 第1条 第一条

第一条仏蘭西国（其ノ殖民地及管治地域ヲ含ム以下同シ）ノ船舶ノ検査ヲ行フ場合ニ於テ仏蘭西国政府ノ発給シタル航海許可書ヲ受有スル船舶ニ対シテハ大体ニ於テ耐航性ヲ有シ船員及旅客ニ危険ヲ及ホス虞ナシト認ムルトキハ該証書ノ目的トスル船体、機関及属具ニ関スル照査ヲ行ハス且如何ナル場合ニ於テモ仏蘭西国ノ法規ニ定ムル以外ノ条件ヲ課セス

## 第2条 第二条

第二条日本ノ港ニ於テ移住民若ハ三等旅客五十人以上又ハ移住民及三等旅客五十人以上ヲ搭載シテ近海航路外ノ港又ハ運輸大臣ノ定ムル地方ニ運送セムトスル仏蘭西国船舶ニ対シテハ前条ノ規定ニ拘ラス糧食、飲料水及衛生状態ニ関スル検査ハ之ヲ行フ
