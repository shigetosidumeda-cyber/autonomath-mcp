# chiiki-koyo-kaihatsu_2

# 地域雇用開発促進法第五条第六項及び第六条第六項の審議会を定める政令 
法令番号 平成13年政令第319号 施行日 2011-08-30 最終改正 2011-08-30 所管 mhlw カテゴリ 労働 e-Gov 法令 ID 413CO0000000319 ステータス active 

目次 

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日） 

（施行期日）第一条この政令は、雇用対策法及び地域雇用開発促進法の一部を改正する法律の施行の日（平成十九年八月四日）から施行する。
