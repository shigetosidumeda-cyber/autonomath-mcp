# 船舶登記の嘱託職員を指定する省令 
法令番号 平成12年厚生省令第135号 施行日 2016-04-01 最終改正 2016-04-01 カテゴリ 運輸 e-Gov 法令 ID 412M50000100135 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この省令は、公布の日から施行する。ただし、附則第八条から第十八条までの規定は、平成十六年四月一日から施行する。
