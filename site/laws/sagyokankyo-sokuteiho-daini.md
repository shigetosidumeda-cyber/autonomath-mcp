# sagyokankyo-sokuteiho-daini

# 作業環境測定法第二十条第二項に規定する指定試験機関の指定に関する省令 
法令番号 平成13年厚生労働省令第70号 施行日 2013-06-12 最終改正 2013-06-12 所管 moe e-Gov 法令 ID 413M60000100070 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この省令は、一般社団法人及び一般財団法人に関する法律の施行の日（平成二十年十二月一日）から施行する。
