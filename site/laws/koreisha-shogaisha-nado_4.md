# koreisha-shogaisha-nado_4

# 高齢者、障害者等の移動等の円滑化の促進に関する法律施行令第二十条に規定する標識に関する省令
法令番号 平成18年国土交通省令第113号 施行日 2025-06-01 最終改正 2024-11-21 e-Gov 法令 ID 418M60000800113 ステータス active

目次

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日）

（施行期日）第一条この省令は、高齢者、障害者等の移動等の円滑化の促進に関する法律施行令の一部を改正する政令の施行の日（令和七年六月一日）から施行する。
