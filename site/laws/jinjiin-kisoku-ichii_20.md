# 人事院規則一―一（規則の分類） 
法令番号 昭和24年人事院規則一―一 施行日 2014-05-30 最終改正 2014-05-29 e-Gov 法令 ID 324RJNJ01001000 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この規則は、国家公務員法等の一部を改正する法律（平成二十六年法律第二十二号）の施行の日から施行する。
