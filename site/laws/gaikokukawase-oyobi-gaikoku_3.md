# gaikokukawase-oyobi-gaikoku_3

# 外国為替及び外国貿易法第六十八条第二項に規定する証票の様式を定める省令
法令番号 昭和44年通商産業省令第25号 施行日 2025-06-01 最終改正 2025-05-23 e-Gov 法令 ID 344M50000400025 ステータス active

目次

- [1 （施行期日） ](#art-1)
- [2 （経過措置） ](#art-2)
- [3 第三条 ](#art-3)

## 第1条 （施行期日）

（施行期日）第一条この省令は、平成二十二年四月一日から施行する。

## 第2条 （経過措置）

（経過措置）第二条この省令の施行前にした行為に対する罰則の適用については、なお従前の例による。

## 第3条 第三条

第三条２この省令の施行の際に現にあるこの省令による改正前の外国為替及び外国貿易法第六十八条第二項に規定する証票の様式を定める省令様式により使用されている書類は、この省令による改正後の外国為替及び外国貿易法第六十八条第二項に規定する証票の様式を定める省令様式によるものとみなす。
