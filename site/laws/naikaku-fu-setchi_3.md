# 内閣府設置法第四条第一項第二十四号に規定する北方地域の範囲を定める政令
法令番号 昭和34年政令第33号 施行日 2016-04-01 最終改正 2016-03-31 e-Gov 法令 ID 334CO0000000033 ステータス active

目次

## 第1条 （施行期日）

（施行期日）第一条この政令は、沖縄開発庁設置法の施行の日（昭和四十七年五月十五日）から施行する。

## 第1_附2条 （施行期日）

（施行期日）第一条この政令は、内閣法の一部を改正する法律の施行の日（平成十三年一月六日）から施行する。
