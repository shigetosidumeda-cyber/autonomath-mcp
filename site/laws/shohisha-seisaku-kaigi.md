# shohisha-seisaku-kaigi

# 消費者政策会議令 
法令番号 昭和43年政令第249号 施行日 2011-07-01 最終改正 2011-06-29 所管 caa カテゴリ 消費者 e-Gov 法令 ID 343CO0000000249 ステータス active

## 第1条 （会長） 

（会長）第一条会長は、会務を総理する。２会長に事故があるときは、会長があらかじめ指名する委員が、その職務を代理する。 

## 第1_附2条 （施行期日） 

（施行期日）第一条この政令は、内閣法の一部を改正する法律の施行の日（平成十三年一月六日）から施行する。 

## 第2条 （庶務） 

（庶務）第二条消費者政策会議の庶務は、消費者庁消費者政策課において処理する。 

## 第3条 （雑則） 

（雑則）第三条前二条に定めるもののほか、議事の手続その他消費者政策会議の運営に関し必要な事項は、会長が消費者政策会議に諮つて定める。
