# doshu-sei-tokubetsu_3

# 道州制特別区域推進本部令
法令番号 平成19年政令第12号 施行日 2007-01-26 最終改正 2007-01-24 e-Gov 法令 ID 419CO0000000012 ステータス active

## 第1条 （参与会議）

（参与会議）第一条道州制特別区域推進本部に、参与会議を置く。２参与会議は、広域行政の推進（道州制特別区域における広域行政の推進に関する法律第三条第一項に規定する広域行政の推進をいう。）のために講ぜられる施策に係る重要事項について審議し、道州制特別区域推進本部長に意見を述べる。３参与会議は、参与五人以内をもって組織する。４参与は、優れた識見を有する者のうちから、内閣総理大臣が任命する。

## 第2条 （参与の任期等）

（参与の任期等）第二条参与の任期は、二年とする。ただし、補欠の参与の任期は、前任者の残任期間とする。２参与は、再任されることができる。３参与は、非常勤とする。

## 第3条 （道州制特別区域推進本部の運営）

（道州制特別区域推進本部の運営）第三条この政令に定めるもののほか、道州制特別区域推進本部の運営に関し必要な事項は、道州制特別区域推進本部長が道州制特別区域推進本部に諮って定める。
