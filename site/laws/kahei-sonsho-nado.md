# 貨幣損傷等取締法 
法令番号 昭和22年法律第148号 施行日 2025-06-01 最終改正 2022-06-17 e-Gov 法令 ID 322AC0000000148 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この法律は、昭和六十三年四月一日から施行する。 

## 第16条 （罰則の適用に関する経過措置） 

（罰則の適用に関する経過措置）第十六条この法律の施行前にした行為に対する罰則の適用については、なお従前の例による。
