# kokki-oyobi-kokka

# 国旗及び国歌に関する法律 
法令番号 平成11年法律第127号 施行日 1999-08-13 最終改正 1999-08-13 e-Gov 法令 ID 411AC0000000127 ステータス active 

目次

- [2 （国歌） ](#art-2)

## 第1条 （国旗） 

（国旗）第一条国旗は、日章旗とする。２日章旗の制式は、別記第一のとおりとする。 

## 第2条 （国歌） 

（国歌）第二条国歌は、君が代とする。２君が代の歌詞及び楽曲は、別記第二のとおりとする。
