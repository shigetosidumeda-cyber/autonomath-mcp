# 農薬取締法第二十五条第一項の農林水産省令・環境省令で定める農薬を定める省令
法令番号 平成15年農林水産省・環境省令第4号 施行日 2018-12-01 最終改正 2018-11-30 所管 mhlw カテゴリ 農業 e-Gov 法令 ID 415M60001200004 ステータス active

## 第1条 （施行期日）

（施行期日）第一条この省令は、農薬取締法の一部を改正する法律の施行の日（平成三十年十二月一日）から施行する。
