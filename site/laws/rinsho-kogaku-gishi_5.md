# 臨床工学技士法第十七条第一項に規定する指定試験機関を指定する省令 
法令番号 平成13年厚生労働省令第91号 施行日 2018-07-09 最終改正 2018-07-09 e-Gov 法令 ID 413M60000100091 ステータス active

## 第1条 （施行期日） 

（施行期日）第一条この省令は、一般社団法人及び一般財団法人に関する法律の施行の日（平成二十年十二月一日）から施行する。
