# an-ma-matsusaaji_5

# あん摩マツサージ指圧師、はり師、きゆう師等に関する法律第三条の四第一項及び第三条の二十三第一項に規定する指定試験機関及び指定登録機関を指定する省令 
法令番号 平成13年厚生労働省令第88号 施行日 2018-12-11 最終改正 2018-12-11 e-Gov 法令 ID 413M60000100088 ステータス active 

目次 

- [1 （施行期日） ](#art-1)

## 第1条 （施行期日） 

（施行期日）第一条この省令は、一般社団法人及び一般財団法人に関する法律の施行の日（平成二十年十二月一日）から施行する。
