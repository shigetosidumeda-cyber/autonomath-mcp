集約サイト採用ポリシー

税務会計AI は noukaweb / hojyokin-portal / biz.stayway 等の二次情報集約サイトを source_url に採用していません。

過去に二次情報サイトを出典とすると、原典との乖離・古い金額・既に終了した制度・存在しない要件などをそのまま AI に渡してしまう事故がありました。利用者から見れば「税務会計AI が嘘をついた」になり、最悪の場合は受給見込み違いや不適切申請につながり、これは重大な誤情報リスクです。

そのため 税務会計AI は source_url に集約サイトを入れない方針を、コード側 (除外リスト) と運用側 (レビュー) の両方で固めています。出典は省庁・自治体・日本政策金融公庫・国税庁・e-Gov 等の一次資料に限定し、99% 以上の行に source_url + fetched_at を付与しています (例外は CMS が存在しない 12 件の小規模自治体のみ)。

ライセンス略称の解説

  • CC BY 4.0 (Creative Commons Attribution 4.0 International) — 出典明示で再配布・改変・商用利用可。e-Gov 法令データはこのライセンスで提供されています。
  • PDL v1.0 (Public Data License v1.0、政府データ向け) — 出典明示で API による下流配信可。国税庁の適格請求書発行事業者公表データはこのライセンスです。税務会計AI は 2026-04-24 に TOS 直接確認で ¥3/リクエスト API 下流提供 OK を確定済み。
  • 政府標準利用規約 v2.0 — 各省庁・地方公共団体が公開する公的データの一般的な利用条件 (CC BY 4.0 互換に整理された 2014 年改訂版)。出典明示・改変箇所明示で複製・公衆送信・翻訳・変形・翻案・商用利用可。利用規約 PDF 参照。
  • public domain — 著作権の保護対象外、または公開判決・通達のように出典明示のみで自由に利用可。
  • proprietary (公開情報のみ収録) — 当該機関が著作権を留保するが、公開ページに掲載された情報のみを収録し、出典 URL を明示することで二次情報リスクを回避しています。JST など TOS で再配布制限のあるデータについては、当社 API 経由でのデータ受領は「限定利用」とし、お客様による商用再配布 (二次商用配信) は認められません。商用配信ライセンスについては launch 直前に license 再評価を行います。
  • license unknown — 出典 URL の domain 抽出失敗または規則 unmatched により license 未確定 (805 rows、2026-04-26 時点)。API レスポンスでは license: "unknown" として明示し、再配布 / 商用利用前に info@bookyou.net までお問い合わせください。

出典の機械可読アクセス

各 programs / case_studies / loan_programs / enforcement_cases / laws 等のレコードには source_url (一次資料 URL) と fetched_at (取得日時) が付与されており、API レスポンスでも MCP ツール の戻り値でも参照できます。API リファレンス および MCP ツール リファレンス を参照してください。

個別の制度・判例・入札等について「出典が古い」「リンク切れ」「原典と齟齬がある」等を発見された場合は、info@bookyou.net までお知らせください。dispute (利用者からの指摘) として運営者が一次資料と再突合し、48 時間以内 (営業日) に判断します。
