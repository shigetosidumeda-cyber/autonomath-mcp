ライセンス略称

  • CC BY 4.0 — 出典明示で再配布・改変・商用利用可。
  • PDL v1.0 — 政府データ向け Public Data License。出典明示で API による下流配信可。
  • 政府標準利用規約 v2.0 — 出典明示・改変箇所明示で複製・公衆送信・商用利用可 (利用規約 PDF)。
  • パブリックドメイン — 著作権の保護対象外。出典明示のみで自由に利用可。
  • 独自整理 (公開情報のみ収録) — 当該機関が著作権を留保。公開情報のみ収録し出典 URL を明示。利用規約に商用再配布制限がある場合は当社 API 経由のデータも限定利用となります。
