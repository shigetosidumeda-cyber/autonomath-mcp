例題プロンプト

Claude Desktop / Cursor / ChatGPT に MCP 接続済みの 税務会計AI をつないだら、そのまま貼れる 20 の例題。5,400+ の補助金・融資・税制・認定制度を横断して、1 分で棚卸しできます。

詳しいセットアップは getting-started、Claude Desktop の設定は MCP 設定セクション を参照してください。

使い方:
  1. 各カードの「コピー」ボタンでプロンプト全文をコピー
  2. Claude Desktop / Cursor / ChatGPT Plus のチャット欄に貼り付け
  3. 地名・品目・従業員数などを自分のケースに置き換えて送信
