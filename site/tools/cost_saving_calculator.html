jpcite API fee delta calculator

「普通に Claude/GPT を使う」場合の external API fee (token + Anthropic web search / OpenAI web search) と jpcite fee (¥3/billable unit) の API 料金差額を即時計算します。既定 baseline は Claude Sonnet 4.5、Anthropic web search、USD/JPY=150、jpcite ¥3。数値は stated baseline の参考比較で、provider pricing は 2026-05-13 に公式 pricing page を再確認済みです。詳細な前提は API fee delta examples を参照。

jpcite 側の fee は、Evidence Packet または Company Folder Brief が返す source_url、source_fetched_at、source_receipts、known_gaps、copy_paste_parts を LLM 回答前に使う前提です。費用差だけではなく、何を AI に渡せるかは Evidence Packet、Company Folder Brief、API fields で確認できます。
