富山県 補助金・税制 主要10件

中部・富山県 の事業者が活用しやすい tier S/A の主要制度を、補助金額の大きい順に上位10件まで集約しました。 各制度名は AutonoMath の制度詳細ページに、出典は一次資料 URL に直接リンクしています。

主要制度（tier S/A）

  • AMED 創薬ベンチャーエコシステム強化事業 tier S ・ 公的支援制度 ・ 最大 300,000万円 一次資料を開く
  • 経営環境変化対応資金（セーフティネット貸付） tier S ・ 融資 (政策金融) ・ 最大 72,000万円 一次資料を開く
  • 危機対応後経営安定資金（セーフティネット貸付） tier S ・ 融資 (政策金融) ・ 最大 72,000万円 一次資料を開く
  • 中小企業成長加速化補助金 tier S ・ 補助金・交付金 ・ 最大 50,000万円 一次資料を開く
  • 経営所得安定対策（畑作物の直接支払交付金・米の直接支払交付金等） tier S ・ 補助金・交付金 ・ 最大 35,000万円 一次資料を開く
  • AMED 橋渡し研究プログラム (preF/シーズF/シーズB/シーズC) tier S ・ 公的支援制度 ・ 最大 20,000万円 一次資料を開く
  • 農林水産業みらい基金 tier S ・ 補助金・交付金 ・ 最大 10,000万円 一次資料を開く
  • 企業版ふるさと納税 tier S ・ 公的支援制度 ・ 10万円 〜 10,000万円 一次資料を開く
  • みよし市 農業振興事業補助金 地場産業振興 tier S ・ 補助金・交付金 ・ 最大 10,000万円 一次資料を開く
  • スマート農業技術の開発・実証プロジェクト tier S ・ 補助金・交付金 ・ 1,000万円 〜 10,000万円 一次資料を開く
