よくある質問

月の最低利用額は?

ありません。 使った分だけ。 0 リクエスト の月は ¥0、 100 リクエスト の月は税込 ¥330 (税抜 ¥300 + 消費税 10%)。 月初 50 リクエスト までは無料枠です。

解約違約金は?

ありません。 Stripe Customer Portal でいつでも cancel できます。 解約後は匿名 50 リクエスト/月 の無料枠に自動で戻ります。

税込・税抜どちら?

表示は税込 ¥3.30/リクエスト。 内訳は税抜 ¥3 + 消費税 10%。 Stripe Tax が請求書に消費税を自動計算 / 明細化します。

invoice (適格請求書) もらえる?

はい、 全 invoice に登録番号 T8010001213708 (Bookyou株式会社) を印字。 仕入税額控除の要件を満たした適格請求書を Stripe Tax が自動発行します。

予算上限を設定できる?

はい、 dashboard で月額 cap を任意の円額で設定可能 (POST /v1/me/cap 経由でも可)。 設定額を超えると即座に 503 + cap_reached で自動停止し、 JST 月初 00:00 にリセットします。 デフォルトは無制限。

どれを選べばいい?

AI 開発者は API + MCP。スマホで使いたい個人 / 経営者は LINE bot。法令改正キャッチアップは アラート。自社サイト集客は Widget。士業・認定支援機関は 専門家 登録。迷ったら 比較ページ の決定木。

複数同時に使えますか?

はい。1 アカウントで 5 インターフェース全てを購読可能。例: API (¥3/リクエスト) + アラート (¥500/月) + 専門家 登録 の組合せなど。Stripe 1 請求書にまとめて明細化します。

途中で利用するインターフェースを変更できる?

はい。Stripe Customer Portal から各インターフェースの追加・切替・解約ができます。日割りは自動計算。インターフェースの組合せは自由で、上下のランクはありません。

使わない月も請求される?

従量制 (API / Widget 超過分) は 0 リクエスト なら ¥0。月額固定 (LINE Plus / アラート 即時 / Widget) は解約しない限り毎月請求されます。

契約書 / 個別見積は?

完全セルフサーブで運用しており、個別契約・個別見積はしていません。公開料金がそのまま全ユーザーに適用されます。

データ提供範囲は?

2026-04 時点で補助金・融資・税制・認定制度 14,472 件 (検索対象 11,684)、採択事例 2,286、融資 108 (担保・個人保証人・第三者保証人の三軸分解)、行政処分 1,185。全件一次資料 URL + fetched_at 付き。排他ルール 181 本。4-dataset 拡張 (法令 / 判例 / 入札 / 税務ruleset / 国税庁適格事業者) も含む。すべての数値は ファクトシート に出典付きで掲載。
