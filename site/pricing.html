よくある質問

tier はないんですか?

ありません。 ¥3/req 完全従量のみ。 Free / Starter / Pro / Enterprise といった階層は一切設けていません。 全顧客に同じ単価。

大量に使うとディスカウント?

ありません。 月 100 req でも 100 万 req でも全顧客 ¥3/req (税込 ¥3.30)。 営業交渉や個別見積もしません。 公開単価が全顧客に一律適用されます。

signup 必須ですか?

いいえ。 IP-based で 50 req/月 まで anonymous OK。 50 req を超える場合のみ API キー発行 + Stripe カード登録が必要です (JST 月初 00:00 リセット)。

解約手続き?

ダッシュボードで API key 削除のみ、 違約金なし。 Stripe Customer Portal からカードを外せば即時匿名 50 req/月 の無料枠に戻ります。 通知期間 / 引止め電話 / 解約手数料 一切ありません。

Stripe 自動課金?

はい、 月末締めで翌月 1 日に登録カードへ自動請求。 適格請求書発行 (T8010001213708 / Bookyou株式会社) を Stripe Tax が自動発行し、 仕入税額控除の要件を満たします。

法人 / 個人事業主 どちらも?

はい、 課税事業者 / 免税事業者 / 法人 / 個人事業主 いずれも同じ ¥3/req。 適格請求書は全顧客に発行 (T8010001213708)、 仕入税額控除を取りたい課税事業者にも対応済み。

予算上限を設定できる?

はい、 dashboard で月額 cap を任意の円額で設定可能 (POST /v1/me/cap 経由でも可)。 設定額を超えると即座に 503 + cap_reached で自動停止し、 JST 月初 00:00 にリセットします。 デフォルトは無制限。

1 リクエストの定義は?

1 HTTP もしくは 1 MCP tool 呼び出し = 1 req。 結果が 100 件返っても 1 req カウント。 /v1/healthz・/v1/me/* 等のメタ系は無料カウント外。
