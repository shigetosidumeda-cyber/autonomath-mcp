よくある質問

どれを選べばいい?

AI 開発者は API + MCP。スマホで使いたい個人 / 経営者は LINE bot。法令改正キャッチアップは アラート。自社サイト集客は Widget。士業・認定支援機関は advisor 登録。迷ったら 比較ページ の決定木。

複数同時に使えますか?

はい。1 アカウントで 5 インターフェース全てを購読可能。例: API (¥3/req) + アラート (¥500/月) + advisor 登録 の組合せなど。Stripe 1 請求書にまとめて明細化します。

途中で利用するインターフェースを変更できる?

はい。Stripe Customer Portal から各インターフェースの追加・切替・解約ができます。日割りは自動計算。インターフェースの組合せは自由で、上下のランクはありません。

使わない月も請求される?

従量制 (API / Widget overage) は 0 req なら ¥0。月額固定 (LINE Plus / アラート Real-time / Widget) は解約しない限り毎月請求されます。

契約書 / 個別見積は?

完全セルフサーブで運用しており、個別契約・個別見積はしていません。公開料金がそのまま全ユーザーに適用されます。

データ提供範囲は?

2026-04 時点で補助金・融資・税制・認定制度 13,578 件 (検索対象 10,790)、採択事例 2,286、融資 108 (担保・個人保証人・第三者保証人の三軸分解)、行政処分 1,185。全件一次資料 URL + fetched_at 付き。排他ルール 181 本。4-dataset 拡張 (法令 / 判例 / 入札 / 税務ruleset / 国税庁適格事業者) も含む。すべての数値は ファクトシート に出典付きで掲載。
