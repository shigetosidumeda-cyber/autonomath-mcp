FAQ

Are there tiers?

No. ¥3/req fully metered only — no Free / Starter / Pro / Enterprise. Every customer pays the same unit rate.

Volume discount for high usage?

No. 100 req or 1,000,000 req per month — every customer pays ¥3/req (¥3.30 tax inclusive). No sales negotiation, no custom quotes.

Is signup required?

No. The first 50 req/month per IP are free without signup (resets at JST first-of-month 00:00). Beyond 50, issue an API key + register a card in Stripe.

Cancellation procedure?

Just delete the API key in the dashboard, or remove the card in Stripe Customer Portal. No cancellation fee, no notice period, no retention call. Reverts to anonymous 50 req/month immediately.

Stripe auto-billing? Qualified Japanese invoice?

Yes. End-of-month metered usage is auto-charged on day 1 of the following month. Stripe Tax issues a qualified Japanese invoice (registration number , Bookyou株式会社) that meets the input-tax-credit requirements.

Corporations and sole proprietors?

Both. Same ¥3/req for taxable / tax-exempt entities, corporations, and sole proprietors. Every invoice carries — taxable entities can claim the input-tax credit.

Can I set a monthly spend cap?

Yes. Set monthly_cap_yen from the dashboard (or POST /v1/me/cap). Once hit, the API returns 503 with cap_reached until the JST first-of-month 00:00 reset. Default is uncapped.

How is 1 request defined?

1 HTTP or 1 MCP tool call = 1 req. A response with 100 hits still counts as 1 req. /v1/healthz and /v1/me/* meta endpoints are uncounted.
