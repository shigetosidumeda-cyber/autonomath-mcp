FAQ

Which interface should I pick?

AI developers: API + MCP. Individuals or business owners on a phone: LINE bot. Compliance & statute tracking: alerts. Embed search on your own site: widget. Licensed Japanese advisors / 認定支援機関: advisor listing. Unsure? See the decision tree on the comparison page.

Can I use multiple interfaces at once?

Yes — a single account can subscribe to all 5 interfaces. Example: API (¥3/req) + Real-time alerts (¥500/month) + advisor listing. Stripe consolidates everything onto one itemised invoice.

Can I change interfaces mid-cycle?

Yes. Add, switch, or cancel any interface from Stripe Customer Portal. Pro-rating is automatic. Interfaces combine freely — there are no upper or lower tiers.

Am I billed in months I don't use it?

Metered components (API / widget overage) bill ¥0 at 0 req. Fixed monthly plans (LINE metered overflow, Real-time alerts, widget Standard / Whitelabel) bill until cancelled.

What happens if I hit the API anonymous monthly allowance?

The API returns 429 with Retry-After set to the seconds until JST first-of-month 00:00. Subscribe to Metered (¥3/req tax exclusive, ¥3.30 tax inclusive) and the cap is removed immediately — pay only for what you use.

Volume discount for high usage?

Not at launch. A single rate keeps the pricing page readable. If you are consistently above 1 M req/month and want a custom rate, email us — but the public price applies to everyone by default.

Can I issue multiple API keys?

Yes — one per environment is common. Rotate or revoke from the billing portal; each key bills against your single metered subscription.

Custom contracts or quotes?

Operations are fully self-serve; we do not negotiate individual contracts or custom quotes. The published pricing applies to every customer equally.

Data coverage?

As of April 2026: 13,578 Japanese public programs (subsidies, loans, tax incentives, certifications; 10,790 searchable), 2,286 adoption case studies, 108 loan programs with three-axis risk decomposition (collateral / personal guarantor / third-party guarantor), 1,185 enforcement actions, 181 exclusion + prerequisite rules. Every row carries a primary-source URL + fetched_at timestamp; aggregators are banned. The 4-dataset expansion (statutes / case law / public bids / tax rulesets / NTA qualified-issuer registry) is also included. All figures are sourced on the fact sheet.
