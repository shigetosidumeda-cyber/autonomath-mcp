Cross-references

Audiences overlap. Pick combinations that fit your role.

Cases that move between the two through client referrals. Tax advisors specialize in special measures and filings; administrative scriveners handle permits and subsidy paperwork. Both tax advisor and administrative scrivener pages map the overlap.

SMB → professional advisor

SMB owners discover programs via the LINE bot, then connect to a matched professional when needed. Free for the user; the advisor pays ¥3,000 per closed introduction. SMB → home.

VC ↔ AI developer

DD pipelines are usually built as in-house RAG. The VC defines requirements; the developer wires up MCP / REST. Combine the VC and developer pages.

Tax advisor → AI developer

Firms building an internal RAG plug the MCP server straight into Claude Desktop or Cursor. Tax advisor → developer.
