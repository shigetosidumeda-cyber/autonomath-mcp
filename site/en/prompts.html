Example prompts

Once AutonoMath is connected to Claude Desktop, Cursor, or ChatGPT over MCP, paste any of these 20 examples directly. Each one sweeps across 5,400+ subsidies, loans, tax measures, and certifications — and finishes in about a minute.

For setup, see getting-started. The Claude Desktop configuration lives in the MCP setup section.

About the example prompts: the prompts below are written in Japanese to match the underlying corpus (Japanese public-program data from METI / MAFF / SME Agency / Japan Finance Corporation, etc.). LLMs like Claude / GPT handle Japanese-language input natively — paste them as-is and you will get Japanese responses. The AutonoMath API and MCP tools accept queries in any language, but always return source data in Japanese to preserve fidelity to the primary documents. To work in English, simply prefix your message with "Reply in English." and the LLM will translate the result for you.
How to use:
  1. Click the "Copy" button on any card to grab the full prompt.
  2. Paste it into the Claude Desktop / Cursor / ChatGPT Plus chat box.
  3. Replace placeholders (location, crop, headcount, etc.) with your own situation and send.
