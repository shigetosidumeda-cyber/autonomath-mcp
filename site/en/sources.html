Data sources, attribution & licenses

AutonoMath ingests Japanese public-program data from 14 primary sources. This page discloses the source agency, record count, license terms, and the exact attribution string each source requires — so tax accountants (税理士), administrative scriveners (行政書士), labor & social security attorneys (社労士), SaaS developers, and downstream LLM agents can perform due diligence and design correct attribution for any further redistribution.

Counts in the table below are a snapshot as of 2026-04-26. Live counts continuously refreshed; see the fact sheet and changelog for the latest values.

Aggregator policy

AutonoMath does not use second-tier aggregators such as noukaweb, hojyokin-portal, or biz.stayway in source_url.

Past incidents proved that citing aggregator pages let stale amounts, expired programs, and fabricated eligibility requirements flow straight into AI agent responses. From the user's perspective, it looks like AutonoMath lied; in the worst case, it leads to incorrect benefit projections or improper applications — which is fraud risk by another name.

So AutonoMath enforces an aggregator ban on source_url in both code (banlist) and operations (manual review). source_url values are restricted to primary sources — ministries, prefectures, Japan Finance Corporation, NTA, e-Gov, etc. Over 99% of rows carry both source_url and fetched_at; the remaining ~12 rows belong to small municipalities that do not run a public CMS.

License legend

  • CC BY 4.0 (Creative Commons Attribution 4.0 International) — redistribution, modification, and commercial use permitted with attribution. e-Gov laws are licensed under CC BY 4.0.
  • PDL v1.0 (Public Data License v1.0 for Japanese government data) — downstream API redistribution permitted with attribution. The NTA Qualified Invoice Issuer registry is under PDL v1.0; on 2026-04-24 we directly verified the terms of use and confirmed downstream redistribution as a paid (¥3/req) API is permitted.
  • Government Standard Terms of Use v1.0 — the standard public-data terms used by Japanese ministries and local governments. Reproduction, public transmission, translation, transformation, adaptation, and commercial use are all permitted with attribution and disclosure of any modifications.
  • public domain — not subject to copyright (e.g. published court judgments, NTA notices); free to use with attribution.
  • proprietary (publicly-disclosed information only) — the source agency retains copyright, but AutonoMath only ingests information already publicly published, with source_url attribution. Commercial-redistribution licensing will be re-evaluated immediately before launch.

Machine-readable attribution

Every row in programs, case_studies, loan_programs, enforcement_cases, laws, etc. carries source_url (primary-source URL) and fetched_at (fetch timestamp). Both are returned by every API and MCP tool response. See the API reference and the MCP tools reference.

If you find a stale source URL, a broken link, or a discrepancy between the data and the primary source, please email info@bookyou.net. The dispute will be re-checked against the primary source and resolved within 48 hours (business days).

Right to be forgotten

For removal requests against the Qualified Invoice Issuer registry data ("the registration was withdrawn but is still indexed", "I am a sole proprietor and revoked the public listing of my trade name", etc.), email info@bookyou.net with your corporate number / registration number / trade name. Records are removed or hidden within 48 hours (business days). See the privacy policy (JP).
