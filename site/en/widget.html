Add Japanese-subsidy search
to your site in one line

A drop-in JavaScript widget for advisor firms, chambers of commerce, and SMB-support portals. Paste a <script> tag plus a single <div>; filters, results, and primary-source links render automatically — no build step, no React/Vue dependency.

<script async src="https://jpcite.com/widget/autonomath.js"></script> <div data-autonomath-widget data-key="wgt_live_YOUR_KEY"></div>

Live demo (production JS, language=en)

The above is a demo render. With a production key whose origin is allowlisted, live results render on your site. See the widget demo page for additional themes (light / dark / programmatic).

Why teams ship the widget

One-line install

Just a script tag and a div. Works in plain HTML — no React/Vue, no build pipeline. Drops into existing CMS pages.

Always-fresh results

Backed by jpcite's 14,472-program database, refreshed nightly. Primary-source URLs and deadlines flow straight through.

Origin-locked keys

Each key is bound to allowlisted origins, blocking click-bombs and unauthorized third-party reuse.

Theme-aware UI

Light / dark themes plus CSS overrides. Match your existing palette and typography without iframe wrappers.

Cite-able results

Every program links to its primary source — METI, MAFF, JFC, or the relevant prefecture. Visitors verify the basis themselves.

Self-serve, flat monthly

Business and Whitelabel both ship via Stripe Checkout. Keys are issued automatically — no sales call, no onboarding gate.
