Add a prescreen intake
to your site

A drop-in JavaScript widget for advisor firms, chambers of commerce, and SMB-support portals. Paste a <script> tag plus a single <div>; candidate programs, primary-source links, fetched timestamps, and prescreen questions render automatically — no build step, no React/Vue dependency.

<script async src="https://jpcite.com/widget/jpcite.js"></script> <div data-jpcite-widget data-key="[issued-widget-key]"></div>

With a production key registered to your site URL, live results render on your site. See the widget demo page for additional themes (static light / dark / programmatic examples).

Why teams ship the widget

Script plus div install

Just a script tag and a div. Works in plain HTML — no React/Vue, no build pipeline. Drops into existing CMS pages.

Fetch timestamps shown

Backed by jpcite's 14,472 total programs, including 11,601 searchable rows. Source URLs, fetch timestamps, and deadline candidates flow through.

Limited to your site URLs

Each key is bound to registered site URLs, blocking click-bombs and unauthorized third-party reuse.

Theme-aware UI

Light / dark themes plus CSS overrides. Match your existing palette and typography without iframe wrappers.

Cite-able results

Every program links to its primary source — METI, MAFF, JFC, or the relevant prefecture. Visitors verify the basis themselves.

Self-serve metered billing
