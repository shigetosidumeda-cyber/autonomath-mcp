API / MCP で取得

都道府県を絞り込んだ検索は API / MCP の prefecture パラメータでも実行できます。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.{{ DOMAIN }}/v1/programs?prefecture=東京都&tier=S,A"

詳細は API reference および MCP tools をご参照ください。

本ページは AutonoMath が一次情報を集約・構造化したインデックスであり、法的助言・税務助言・申請代行を構成するものではありません。制度の最新内容は所管官公庁・自治体の一次情報および有資格者の助言で必ず確認してください。
