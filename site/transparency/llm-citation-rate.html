LLM citation rate — cascade tipping panel

jpcite が Claude / Perplexity / ChatGPT / Gemini の回答内でどの程度 引用されるかを確認する透明性ページです。初回の公開用集計はまだ 確定していないため、このページは暫定表示として noindex にしています。 数値が確定するまでは、測定方法と公開予定の項目のみを掲載します。

データ未確定 ・ 表示用の集計データがない状態で テンプレート変数や仮の率を公開しないため、現在の q、LLM 別 rate、 過去 trend は非表示にしています。公開時は測定期間、sample 数、 判定基準、known gaps を同じページに併記します。

Sampling protocol (公開仕様)

  • 頻度 ・ 月次で手動確認
  • 対象 ・ Claude / Perplexity / ChatGPT / Gemini の 4 LLM web UI
  • query 数 ・ 公開時に実数を併記
  • query 設計 ・ 業務領域、一般質問、比較質問が偏らないように分散
  • 判定 ・ jpcite.com / api.jpcite.com / 明示 citation の目視確認
  • 記録 ・ 測定月、LLM 名、query group、citation 判定、確認者メモ
  • LLM API 呼出 ・ 公開用測定は web UI の目視確認のみ
  • query rotate ・ 固定 query と入れ替え query を分け、同一設問への過適合を避ける
