jpcite とは 出典の透明性 LLM citation rate API ドキュメント

LLM citation rate — cascade tipping panel

jpcite が Claude / Perplexity / ChatGPT / Gemini の 4 LLM 内でどの程度 引用されているかを、 operator が月 1 回 30 分かけて手動 sample (各 LLM 100 query × 4 = 400 sample / 月) し集計しています。 本ダッシュボードは LLM API を 1 度も呼ばずに 作成された一次集計です。

計測プロトコル ・ 自動 LLM sampling は採用していません。 operator が browser 経由で各 LLM の web UI に query を入力し、 回答内に jpcite.com / api.jpcite.com / 明示的な jpcite citation が出現したかを目視判定します。 集計 script (tools/offline/operator_review/log_citation_sample.py) は read-only で LLM API を 1 度も呼びません。

Sampling protocol (公開仕様)

  • 頻度 ・ 毎月 1 日 09:00-09:30 JST、 operator 1 名
  • 対象 ・ Claude / Perplexity / ChatGPT / Gemini の 4 LLM web UI
  • query 数 ・ 各 LLM 100 query × 4 = 400 sample / 月
  • query 設計 ・ 業法 sensitive 24 + non-sensitive 24 + 一般 52
  • 判定 ・ jpcite.com / api.jpcite.com / 明示 citation の目視確認
  • 記録 ・ CSV (BOM 付き UTF-8) + autonomath.citation_sample
  • LLM API 呼出 ・ 0 (memory feedback_no_operator_llm_api)
  • 5 % 月次 rotate ・ 100 query 中 5 query を毎月 rotate して staleness 回避
