運用方針: jpcite は auto-translate を一切しません。 コーパスに乗る訳文は全て 一次資料 (e-Gov 日本法令外国語訳 CC-BY 4.0 / 各官庁 EN sub-path / JETRO zh-cn/zh-tw/kr 等) からの fetch のみです。 取れなかった law / program は manual review queue に積まれ、担当者が一次資料 URL を確認した行のみ次回更新で反映されます。 LLM API は呼びません。

Language coverage

Manual review queue 入口

review queue 経路: EN/ZH/KO の一次資料 mirror を確認した後、 取れなかった entry を manual review queue に追記します。 担当者が一次資料 URL を確認した行だけを次回更新で反映します。 担当者が確認しない限り 公開コーパスには反映されない ので、 誤訳・自動翻訳のリスクはゼロです。
Queue Scope 処理 備考
Program EN review program の EN mirror が取れない entry Manual review 担当者が一次資料 URL を確認する
Law ZH review law の ZH mirror が取れない entry Manual review 外務省 / METI CN / JETRO zh-cn/zh-tw に当たって駄目だった law
Law KO review law の KO mirror が取れない entry Manual review JETRO kr / MOFA 韓国 / JNTO kr に当たって駄目だった law

REST API

プログラム的に取得する場合は GET /v1/translation/status?lang=en / ?lang=zh / ?lang=ko / ?lang=all を叩いてください。 1 req = 1 unit (¥3) でメータリングされます。 

$ curl https://api.jpcite.com/v1/translation/status?lang=en
{
  "languages": {
    "en": {
      "laws": {"total": 6493, "filled": ..., "coverage_pct": ...},
      "articles": {"total": 353278, "filled": ..., "coverage_pct": ...},
      "programs": {"total": 11601, "filled": ..., "coverage_pct": ...},
      "review_queue_pending": ...,
      "last_refreshed_at": "..."
    }
  },
  "generated_at": "...",
  "disclaimer": "..."
}
データソース: public law corpus + program corpus + manual review queue. クロール頻度: weekly update + monthly audit. 著作権: 各 1 次資料の license を尊重 (e-Gov JLT は CC-BY 4.0, 官庁 EN サイトは政府著作物原則, JETRO public-domain). 本ダッシュボードは自動更新されます。
