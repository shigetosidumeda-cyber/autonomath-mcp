採取手法

Bench は管理された評価セッションで実行し、公開ページでは集計済みメトリクスのみ表示します。 本番サービスから外部 LLM API を直接呼び出して評価する設計ではありません。

Current set includes 4 -latest aliases: claude-opus-4-7-latest / gemini-2-5-flash-latest / gpt-5-latest / deepseek-v4。 未公開時は同 adapter 経由で dated model id に切り戻し、latest_fallback_used=true flag を row に記録。
