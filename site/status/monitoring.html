4. 監視 alert feed (ATOM)

SLA breach / scheduled-update failure / deploy issue / endpoint 5xx などを ATOM feed で publish します。 Feedly / Inoreader / RSS reader 等で subscribe してください。 Critical alert は必要に応じて運用通知にも連携されます。

  • /status/feed.atom — 監視 alert ATOM feed
  • /v1/status/alerts — 現在 active な alert (JSON)
