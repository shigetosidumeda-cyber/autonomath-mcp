KPI 定義

各 KPI の定義は公開メトリクス向けに要約しています。

  • Activity rows — 公開可能な範囲で丸めた request activity index
  • Agent Retention (D30) — authenticated callers active D30 ago and today (fraction)
  • TTFU (seconds) — median seconds from credential issue to first usage
  • Usage Variance (p90/p50) — 7d request-count concentration
  • AI Mention Share — weekly average citation rate
  • Justification Strength (0-100) — description and source URL coverage
