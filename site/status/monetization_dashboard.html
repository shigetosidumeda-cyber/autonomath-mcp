KPI 定義

各 KPI の数式は memory feedback_agent_new_kpis_8 参照。

  • ASR (24h, JPY) — エージェント UA リクエスト × ¥3.30
  • ARC (rows) — last 30d req × 365/30 (rolling-annualized)
  • Cost-to-Serve (JPY/call) — sum(cost_ledger.amount_jpy) / req count, 30d
  • Agent Retention (D30) — api_keys active D30 ago AND today (fraction)
  • TTFP (seconds) — median seconds from api_key issue → first usage
  • Spending Variance (p90/p50) — req count per api_key, 7d
  • AI Mention Share — aeo_citation_bench weekly avg cite rate
  • Justification Strength (0-100) — desc length + source url coverage on S+A programs
