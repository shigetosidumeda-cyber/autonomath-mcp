Revenue share の仕組み

全 partner 共通のメカニズム。詳細は partner_referral_mechanism.md 参照。

referral 仕組みの一覧
項目 内容
ユーザー単価 ¥3 / req (税別、税込 ¥3.30)。referral 経由でも同額 — discount は永久 NG。
referral 還元率 完全従量 売上の 10% (Anthropic Directory のみ 0%)。
payout 方法 Stripe Connect Transfer または銀行振込。最低金額なし。
cycle 月末日 23:59 JST 締め、翌月末日 17:00 JST 払出。
request カウント 有料 APIキー 経由の billable success request のみ。匿名 3 リクエスト/日 free と 4xx は対象外。
適格請求書 jpcite (Bookyou株式会社 ) が振込通知書として発行。
違約金 / 最低期間 なし。1 ヶ月 0 リクエスト なら ¥0 払出 (transfer event スキップ)。

計算例

freee 紹介の仕組み 経由の user が 1 ヶ月に 20,000 リクエスト を使用:

request_count = 20,000
charged_jpy = 20,000 × ¥3 = ¥60,000 (税別)
share_percent = 10
share_jpy = floor(¥60,000 × 0.10) = ¥6,000 ← freee に支払
税務会計AI net = ¥54,000 (税別)

Partnership 原則

  • Self-serve only. 営業電話 / 個別契約 / 紙の覚書 / Slack Connect は永久 NG。Stripe 標準 ToS のみで運用。
  • ¥3/リクエスト 完全均一. partnership 経由でも discount は適用されません — referral fee は jpcite 側コストとして処理。
  • Logo は法務 OK 後のみ. 現在の placeholder は仮表示。partner 各社の許諾を受けた後に差替えます。
  • Pass-through データ. partner 経由ユーザーの会計 / HR / レコードデータを jpcite サーバーに保存しません。query parameter として受け取り即破棄。
  • Solo ops. jpcite 運営は梅田茂利 1 名。SLA / DPA 個別交渉なし、24/7 phone サポートなし。docs / メールでセルフサービス対応。
  • Organic only. 広告 / 営業活動なし。partnership は inbound self-serve のみ。「Featured Partner」呼称は許諾後のみ使用 (ランキング根拠を明示)。
