Revenue share の仕組み

全 partner 共通のメカニズム。料金の正本は Pricing を参照してください。

referral 仕組みの一覧
項目 内容
ユーザー単価 ¥3 / unit (税別、税込 ¥3.30)。referral 経由でも同額 — 個別割引は提供していません。
referral 還元率 完全従量 売上の 10% (Anthropic Directory のみ 0%)。
payout 方法 Stripe Connect Transfer または銀行振込。最低金額なし。
cycle 月末日 23:59 JST 締め、翌月末日 17:00 JST 払出。
request カウント 有料 APIキー 経由の billable success request のみ。匿名 3 リクエスト/日 free と 4xx は対象外。
適格請求書 jpcite (Bookyou株式会社) が適格請求書情報を含む振込通知書として発行。
違約金 / 最低期間 なし。1 ヶ月 0 リクエスト なら ¥0 払出 (transfer event スキップ)。

計算例

freee 紹介の仕組み 経由の user が 1 ヶ月に 20,000 リクエスト を使用:

request_count = 20,000
charged_jpy = 20,000 × ¥3 = ¥60,000 (税別)
share_percent = 10
share_jpy = floor(¥60,000 × 0.10) = ¥6,000 ← freee に支払
jpcite net = ¥54,000 (税別)

Partnership 原則

  • Self-serve only. Partnerships are handled through self-service docs and email; separate enterprise contracts are not offered at this time.
  • ¥3/課金単位 完全均一. partnership 経由でも discount は適用されません — referral fee は jpcite 側コストとして処理。
  • ブランド表示は許諾後のみ. 企業名は連携候補の説明であり、明示されていない限り提携・推奨を意味しません。
  • Pass-through データ. partner 経由ユーザーの会計 / HR / レコードデータを jpcite サーバーに保存しません。query parameter として受け取り即破棄。
  • Support. Support is handled through documentation and email. Standard terms apply unless otherwise published.
  • Organic only. 広告 / 営業活動なし。partnership は inbound self-serve のみ。「Featured Partner」呼称は許諾後のみ使用 (ランキング根拠を明示)。
