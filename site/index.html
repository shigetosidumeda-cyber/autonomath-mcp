制度プレスクリーン — 3 つの質問で 上位 5 件を提示

都道府県・事業形態・予定投資額を入れると、税務会計AI の 11,684 制度から適合度の高い 上位 5 件を表示します（無料・登録不要・匿名 50 リクエスト/月/IP）。/v1/programs/prescreen を 1 回呼ぶだけ。

「全国」を選ぶと地域条件を外して全制度から検索します。都道府県を選ぶと、その県専用制度＋全国制度の両方が候補になります。
「指定なし」だと両方を含む制度のみヒット。「法人」「個人事業主」を選ぶと専用制度＋共通制度を表示。
設備・システム導入などの予定投資額（万円単位、空欄可）。例: 800 = 800 万円。投資額に応じた上限額の制度を上位に並べます。

回答は AI ではなく、制度データベースから規則ベースでマッチングします (prefecture + target_types + amount_fit)。出典 URL は一次資料のみ。送信内容は保存しません。レート制限のため IP ハッシュのみ記録 (JST 月初リセット)。

ChatGPT に聞けば済む話では?

短答: いいえ。3 つの理由で fact-grade な制度判定には使えません。

学習日が古い (cutoff problem)

時間軸が合わない

ChatGPT/Claude の学習データは数ヶ月〜1年前で固定。補助金は「公募締切=昨日」のような時間軸で動くため、AI 単独では「今申請可能か」が分からない。税務会計AI は一次資料の fetched_at を全行に持つ。

出典の捏造 (hallucination risk)

誤判定 = 受給機会の損失

AI は「存在しない補助金 ID」や「実在しない URL」を生成する事例が報告されている。¥300〜¥3,000万 の制度誤判定は申請失敗・依頼者トラブルの主因。税務会計AI は全 14,472 件に source_url + fetched_at を付与し、二次集約は出典禁止 (景表法対応)。

構造化されていない (no compatibility check)

併用可否は推論で決まらない

AI は「補助金 A と補助金 B は併用可?」「同一事業者の他案件との重複は?」を構造的に判定できない。税務会計AI は 181 の exclusion / prerequisite ルールで併用可否を機械判定する。

つまり: AI + 税務会計AI = 速度 (AI) × 正確性 (一次資料 + ルール判定)。税務会計AI は AI の代替ではなく、AI が信頼できる出力を出すための事実層。

なぜ 税務会計AI か

集約サイトを採用しない

情報精度の確保

noukaweb / 補助金ポータル / biz.stayway などの集約サイトは、「掲載しているのに既に終了している」「上限金額が古い」「存在しない要件を追記している」といった事例が複数報告されており、これを根拠に申請して採択されなかった場合、依頼者から依頼者からの信頼喪失や苦情につながる事業者リスクがあります。

税務会計AI は方針として、source_url に集約サイト URL を一切登録しません (禁止)。出典は省庁 (METI / MAFF / 中企庁) / 都道府県 / 政府系金融 (JFC / SMRJ / NEDO) / 商工会の一次 URL のみ。14,472 件の 99%+ で一次 URL を保有し、残る 12 件 (小規模自治体) も明示的に「一次 URL 不足」とフラグしています。

全行が source_url + fetched_at を持つため、「なぜこの補助金がヒットしたか」を顧客 / 監督官庁 / 訴訟相手に客観的に提示できます。これは集約サイトを根拠にした場合には作れない監査ログです。

一次資料 100%

毎日 cron で出典 URL の生存確認

全 programs 行に source_url + fetched_at を付与。出典取得日は透明に開示 (「最終更新」のような誤誘導はしない)。nightly liveness scan で URL が死んでいないかを監視。 法改正の検知は cron で差分走査する仕組みを実装済み (履歴差分の蓄積は初回フル走査後)。

排他ルールを内部化

併用失格を事前検出

181 本の exclusion / prerequisite / absolute を構造化。「AI が気づかないまま併用申請 → 後日失格」という事故を、program_ids を投げた時点で検出する。

cross-dataset glue

1 リクエスト で制度→法令→判例→処分

拡張データセット: 法令 (e-Gov CC-BY、 法令本文 154 件 + 法令メタデータ 9,484 件・本文ロード継続中) + 税務 ruleset (インボイス/電帳法・研究開発税制 等、50 件) + 国税庁適格事業者 (PDL v1.0、13,801 件 delta) をライブ提供。判例 (2,065 件) も live。 さらに autonomath entity-fact DB (503,930 entities / 6.12M facts / 177,381 relations) を 50 tools で公開。89 MCP ツール に含む trace_program_to_law / combined_compliance_check / search_tax_incentives / related_programs で横断的に解決できる。
