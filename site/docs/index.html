API ドキュメント

jpcite は AI agent に日本の制度・法令・判例・補助金を 一次資料 URL 付き で答えさせるための API / MCP server です。¥3/req 完全従量、匿名 3 req/日 無料、月額固定なし。

主なリファレンス

  • OpenAPI v0.4.0 (REST)

    306 paths。検索 / 詳細 / Webhook / Wallet / x402 を含む全 REST 仕様。

  • OpenAPI Agent (slim)

    GPT-3.0 / Custom GPT 向け slim 版。tool count を抑えた agent profile。

  • MCP server.json

    MCP capabilities (tools / resources / prompts / sampling / transports)。

  • llms.txt / llms-full.txt

    LLM 向け要約。citation guidance + companion .md token budget。

  • オンボーディング (4 step)

    匿名無料 → OAuth → topup → 呼出。階層プラン無し、最低利用期間無し。

  • 料金

    ¥3/billable unit (税込 ¥3.30)。Stripe / Wallet / x402 の 3 ルート。
