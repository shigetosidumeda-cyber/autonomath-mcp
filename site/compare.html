選び方

用途別の素直な推奨です。AutonoMath が常に最適ではありません。

SMB / 経営者

まず無料の公的サイトを試す

自社で 1〜数件の補助金を探すだけなら、まずは ミラサポ plus と J-Net21 (どちらも中小機構運営・無料・公的) が出発点として十分です。

公募要項 PDF を AI に読ませて要件適合判定したい / 採択事例を industry でフィルタしたい / 顧問税理士・士業に渡す前のスクリーニングをしたい場合は AutonoMath の API ¥3/req (匿名 50 req/月 無料) もご検討ください。

税理士 / 行政書士 / 中小企業診断士

顧客横断で扱うなら一次資料連結が必須

顧問先 10 社以上を扱う士業の方は、二次情報サイトの古い金額や終了済み制度をそのまま使うと事故になります (景表法・善管注意義務リスク)。一次資料 URL と取得日時が全行に紐づく仕組みが必要です。

AutonoMath なら全行に source_url + fetched_at、e-Gov 法令本文への直リンク、行政処分 1,185 件 (取引相手の業者調査) が一気通貫。MCP 経由で Claude / ChatGPT に直接渡せます。

開発者 / LLM Agent 構築

API/MCP が必須なら選択肢は実質一つ

noukaweb、hojyokin-portal、biz.stayway、J-Net21、ミラサポ plus はいずれも公開 API・MCP サーバーを提供していません (2026-04-26 時点)。AI agent から呼ぶには HTML スクレイピング + 個別 TOS 確認が必要です。

AutonoMath は REST + MCP (protocol 2025-06-18) + OpenAPI 3.1 + 72 tools が launch 時点で揃います。匿名 50 req/月で評価でき、本番は ¥3/req 完全従量、SDK 不要。
