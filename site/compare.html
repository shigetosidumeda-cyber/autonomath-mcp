jpcite と他の補助金検索サービスの比較

「補助金検索 API」「公的制度データ API」「ミラサポ plus API」等で辿り着いた方向けに、jpcite と主要な補助金・助成金検索サービスを 13 軸で並べた中立比較表です。各社の公開情報のみで構成し、未公開項目は「公開情報非公表」と表記しています。月次で見直します。

比較対象: jpcite (本サイト) / 補助金検索サイト / 公的支援ポータル / 記事型データベース / J-Net21 / ミラサポ plus / 帝国データバンク (TDB) / 東京商工リサーチ (TSR)。詳細な 2 軸比較は TDB 個別ページ · TSR 個別ページ。

住み分けの率直な開示: TDB / TSR のコア商品は信用調査評点・代表者経歴・親族関係で、jpcite は提供しません (この軸は TDB / TSR の独占領域)。一方、当社のコアは制度 DB (補助金・融資・税制・認定)・法令・判例・行政処分・適格請求書発行事業者の機械可読 API / MCPで、TDB / TSR の対象外領域です。「どちらか」ではなく「目的別の併用」が現実解です。

選び方

用途別の素直な推奨です。jpcite が常に最適ではありません。

中小企業 / 経営者

人が数件見るだけなら無料サイトで十分

自社で 1〜数件の補助金を探すだけなら、まず ミラサポ plus と J-Net21 (どちらも中小機構運営・無料・公的) が出発点になります。

一方で、AI・顧問先管理・CRM・監査メモへ組み込むなら、HTML を読むより jpcite の API/MCP が向いています。一次資料URL、取得時刻、採択事例、行政処分、法令メタ、known gaps を同じ形式で返せるため、会社フォルダや相談前プレ診断を ¥3/課金単位で自動化できます。

税理士 / 行政書士 / 中小企業診断士

顧客横断で扱うなら一次資料連結が必須

顧問先 10 社以上を扱う士業の方は、二次情報サイトの古い金額や終了済み制度をそのまま使うと事故になります。一次資料 URL と取得時刻を確認できる仕組みが必要です。

jpcite は、主要公開レコードの出典 URL + 取得時刻、e-Gov 法令へのリンク、行政処分 1,185 件を API/MCP から参照できます。Claude Desktop / Cursor / Cline は MCP、ChatGPT Custom GPT は OpenAPI Actions で同じ根拠データを取得できます。

開発者 / AI Agent 構築

API/MCP を重視する場合

主要な補助金検索サイト・支援ポータルの多くは、公開 API や MCP サーバーを提供していません (2026-04-26 時点)。AI から呼ぶには HTML スクレイピング + 個別 TOS 確認が必要です。

jpcite は REST + MCP + OpenAPI 仕様書 + 139 機能を公開しています。匿名 3 リクエスト/日で評価でき、本番は ¥3/課金単位 完全従量、SDK 不要です。

更新ポリシー

  • 更新頻度: 月次レビュー。各社の公開ページを再取得し、変更があれば即反映します。
  • 追記方針: 比較対象外のサービスでも「ユーザーが Google から辿り着くもの」は段階的に追加します (例: jGrants, J-LIS, J-Startup 等)。
  • 誤り報告: 各社担当者・利用者からの修正提案を歓迎します。info@bookyou.net 直行 (代表)。出典 URL を添えていただければ確認します。
  • 誇張禁止: 当社についても「業界最大」「No.1」等の主張はしません。jpcite の数字は about.html の数字 および定期的なデータ検証で裏取りしています。
