コスト削減試算 (β)

LLM モデル × 月次クエリ数 × ドメイン構成 から、 jpcite を使った場合の (1) verify 時間節約 (h/月)、 (2) 監査可能引用の lift (citation_ok)、 (3) 月次 net 損益 (¥) の 3 軸を試算します。 token saving は raw figure として参考表示するのみ。 ベンチマークは JCRB-v1 50 問、 エンドポイントは /v1/calculator/savings。

※ 参考指標は verify 時間の推定短縮と citation_ok lift です。 JCRB-v1 の測定条件に基づく試算であり、個別の削減時間・費用対効果を保証しません。 raw token math だけでは jpcite ¥3/billable unit を償却できないケースがあります。 生 token spend 削減を目当てに使うサービスではありません。 詳細は context savings の説明 をご参照ください。

入力

JCRB-v1 で実測した 5 モデル。 sonnet / 4o は family fallback。
LLM への問い合わせ回数 / 月。 1 顧問先 × 月次 確認 件数 などで算出。
補助金 / 税務 / 法令 のクエリ比率。 ±5% 程度の影響。
JPY 換算用。 treasury レートに合わせて上書きしてください。
