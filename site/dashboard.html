⚡ 5 分で最初の API 呼び出し

下のいずれか 1 つを選んでコピペするだけ。あなたの APIキー (もしくは匿名 3 リクエスト/日) で即座に動きます。詳細手順は各 統合ガイド を参照。 

curl -H "X-API-Key: am_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
 "https://api.jpcite.com/v1/programs/search?q=IT%E5%B0%8E%E5%85%A5&limit=3"

APIキー を省くと匿名 3 リクエスト/日 per IP (JST 翌日リセット) で動作。

1. ターミナルで pip install autonomath-mcp · 2. 下を ~/Library/Application Support/Claude/claude_desktop_config.json (macOS) に貼り付け · 3. Claude Desktop 再起動。

{

    📋 法令改正・制度新設アラート

    法令改正・制度新設・行政処分などの差分を Webhook 通知 (HTTPS のみ) または メール で 先回り通知します。通知配信は ¥3/リクエスト (税込 ¥3.30) の完全従量です。 filter_type は tool / law_id / program_id / industry_jsic / all から選択、min_severity は critical (制度終了・適用条件 変更) / important (金額・対象変更) / info (出典再取得など) の 3 段階。詳しくは アラートガイド。

    🔔 保存条件 + 自動配信ダイジェスト

    検索条件 (q / 都道府県 / 対象事業者 / 金額帯など) を保存しておくと、 毎日 06:00 JST に最新の制度差分を自動スキャンし、 新着があった場合のみ ダイジェストメール を配信します。 各配信は ¥3 (税込 ¥3.30) の従量課金 です (登録自体は無料)。 税理士法 §52 / 弁護士法 §72 該当の業務はおこないません — 配信内容は 公開情報の検索結果であり、税務助言・法律判断は資格者にご確認ください。

    🔔 Webhook 通知 (成功配信ごと ¥3)

    自社の HTTPS エンドポイントに、 program.created / program.amended / enforcement.added / tax_ruleset.amended / invoice_registrant.matched の各イベントを 署名検証 (SHA256) 付きで POST します。 1 回成功配信 = ¥3/リクエスト 課金 です。失敗時の再試行は課金対象外です。 5 回連続で失敗すると、意図しない課金を防ぐため自動で無効化します。 詳細は Webhook 通知 docs。

    本サービスは公開情報の集約 API であり、 税務助言・法律相談ではありません (§52)。 配信される 送信内容 は政府一次資料と公開データに基づく機械集約結果です。
