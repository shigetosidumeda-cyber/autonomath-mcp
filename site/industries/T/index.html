分類不能の産業 (JSIC T): 関連制度一覧

更新: 2026-04-29 / Bookyou株式会社 (梅田茂利)

TL;DR

JSIC T (分類不能の産業 / Industries unable to classify) に該当する公的支援制度は、現時点で税務会計AIが業種要件・名称キーワード・出典ドメインの3軸でクロスリファレンスを抽出した結果、該当 0 件でした。これは「制度が存在しない」という意味ではなく、「業種コード T に直接マッピングできる証拠を持つ制度が AutonoMath の集約データ内に確認されていない」という意味です。

本業種で該当が無い理由

  • JSIC T に対応する業種要件を含む公募要領が、税務会計AIが収集対象としている中央省庁・自治体・公庫の公開ページに見当たらない。
  • 該当する業種の事業者は、業種横断的な制度 (例: 中小企業向けの IT 導入補助金、ものづくり補助金、事業再構築補助金等) で支援を受けられる可能性があります。
  • 分類不能の産業 領域の制度は、業界団体・特殊法人・公益財団法人を経由した補助・委託の形態を取ることが多く、JSIC 業種要件として明文化されないことがあります。

代替の探索方法

  • 都道府県別インデックス — 事業所所在地の都道府県から制度を探す
  • 対象者別インデックス — 法人格・規模 (中小企業・個人事業主・スタートアップ等) で絞り込む
  • 制度一覧 — 全 10,953 件の制度名キーワード検索
  • ダッシュボード検索 — 自然言語クエリで業種横断検索

API / MCP で取得

業種絞り込みは API / MCP の industry_jsic パラメータで実行できます。該当 0 件の業種では空配列が返ります。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.zeimu-kaikei.ai/v1/programs?industry_jsic=T"

無料 50 req/月。料金体系 ・ API キー発行
