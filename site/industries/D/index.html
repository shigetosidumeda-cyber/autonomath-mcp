JSIC D 建設業 補助金・税制 主要20件

JSIC D (建設業) に該当する事業者向けの 主要制度を、補助金額の大きい順に上位20件まで集約しました。 各制度は AutonoMath の制度詳細ページに、出典は一次資料 URL に直接リンクしています。

  • 建築物等のZEB化・省CO2化普及加速事業 tier A ・ 補助金・交付金 ・ 最大 50,000万円 一次資料を開く
  • サステナブル建築物等先導事業（省CO2先導型） tier A ・ 補助金・交付金 ・ 最大 50,000万円 一次資料を開く

  • 移住者の住宅取得及びリフォーム費用支援 tier A ・ 補助金・交付金 ・ 最大 100万円 一次資料を開く
  • 用地造成事業 tier A ・ 公的支援制度 ・ 金額非公表 一次資料を開く
  • 令和7年度熊本県見守り付き住宅確保要配慮者専用賃貸住宅改修事業補助金 tier A ・ 補助金・交付金 ・ 金額非公表 一次資料を開く
