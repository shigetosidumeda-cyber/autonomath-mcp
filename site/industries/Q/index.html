複合サービス事業 (JSIC Q): 関連制度一覧 (15 件)

更新: 2026-04-29 / EN: Programs for Compound services (JSIC Q) / Bookyou株式会社 (梅田茂利)

TL;DR

JSIC Q (複合サービス事業 / Compound services) 事業者向けの日本の公的支援制度を、税務会計AI が業種要件・名称キーワード・出典ドメインの3軸でクロスリファレンス抽出しました。本ページには 15 件 の制度を Tier (S / A / B) 別にソートして掲載しています。各制度ページから一次資料 URL に直接アクセスできます。

業種コード Q の概要

JSIC (日本標準産業分類) における大分類 Q は「複合サービス事業」です (英語: Compound services)。この業種に直接マッピングされる制度のほか、業種横断的な中小企業向け制度 (IT 導入補助金、ものづくり補助金、事業再構築補助金等) も並行して活用できます。本ページの掲載順は Tier (信頼度) → 提供主体 → 制度名のアルファベット順です。

Tier A (信頼度中高: 要件確認済) (4 件)

  1. JAアグリローン（農業機械・施設取得資金）
    融資 (政策金融) / 最大 5,000万円 / 出典: jabank.org
  2. 福知山市有機JAS認証新規取得支援事業補助金
    補助金・交付金 / 地方自治体 (京都府) / 最大 4万円 / 出典: city.fukuchiyama.lg.jp
  3. 広島県信用漁業協同組合連合会「かき養殖経営安定緊急対策資金」
    融資 (政策金融) / 地方自治体 (広島県) / 最大 600万円 / 出典: city.kure.lg.jp
  4. 八尾市農業後継者補助事業（JAとの共同事業）
    補助金・交付金 / 市町村 / 最大 40万円 / 出典: city.yao.osaka.jp

API / MCP で取得

JSIC Q (複合サービス事業) の制度一覧は API / MCP からも取得できます。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.zeimu-kaikei.ai/v1/programs?industry_jsic=Q&tier=S,A&limit=50"

MCP クライアントからは search_programs(industry_jsic="Q", tier=["S","A"]) で呼べます。詳細は API reference および MCP tools 参照。
