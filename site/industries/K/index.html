不動産業、物品賃貸業 (JSIC K): 関連制度一覧 (9 件)

更新: 2026-04-29 / EN: Programs for Real estate and goods rental and leasing (JSIC K) / Bookyou株式会社 (梅田茂利)

TL;DR

JSIC K (不動産業、物品賃貸業 / Real estate and goods rental and leasing) 事業者向けの日本の公的支援制度を、税務会計AI が業種要件・名称キーワード・出典ドメインの3軸でクロスリファレンス抽出しました。本ページには 9 件 の制度を Tier (S / A / B) 別にソートして掲載しています。各制度ページから一次資料 URL に直接アクセスできます。

業種コード K の概要

JSIC (日本標準産業分類) における大分類 K は「不動産業、物品賃貸業」です (英語: Real estate and goods rental and leasing)。この業種に直接マッピングされる制度のほか、業種横断的な中小企業向け制度 (IT 導入補助金、ものづくり補助金、事業再構築補助金等) も並行して活用できます。本ページの掲載順は Tier (信頼度) → 提供主体 → 制度名のアルファベット順です。

Tier A (信頼度中高: 要件確認済) (3 件)

  1. 地域貢献中小企業支援事業（エコ型住宅建設促進事業）補助金
    補助金・交付金 / 北海道別海町 (北海道) / 最大 100万円 / 出典: betsukai.jp
  2. 令和7年度熊本県見守り付き住宅確保要配慮者専用賃貸住宅改修事業補助金
    補助金・交付金 / 熊本県 (熊本県) / 出典: pref.kumamoto.jp
  3. 熊本県見守り付き住宅確保用配慮者専用賃貸住宅改修事業補助金
    補助金・交付金 / 熊本県 (熊本県) / 出典: pref.kumamoto.jp

Tier B (信頼度中: 要件部分確認) (6 件)

  1. 南砺市_民間賃貸住宅家賃支援事業補助金（南砺市農業人材活躍促進重点事業）
    補助金・交付金 / 最大 24万円 / 出典: city.nanto.toyama.jp
  2. 森林づくり木造住宅建築促進事業費補助金
    補助金・交付金 / 最大 100万円 / 出典: city.ozu.ehime.jp
  3. 民間賃貸住宅家賃助成事業
    補助金・交付金 / 北海道小清水町 (北海道) / 最大 2万円 / 出典: town.koshimizu.hokkaido.jp
  4. 更別村住宅建設等助成金
    補助金・交付金 / 北海道更別村 (北海道) / 最大 50万円 / 出典: sarabetsu.jp
  5. 定住促進住宅建設奨励制度
    補助金・交付金 / 北海道鹿追町 (北海道) / 最大 100万円 / 出典: town.shikaoi.lg.jp
  6. 民間賃貸住宅借上制度による住宅提供（賃貸型応急住宅・みなし仮設）
    補助金・交付金 / 地方自治体 / 最大 100万円 / 出典: city.tamana.lg.jp

API / MCP で取得

JSIC K (不動産業、物品賃貸業) の制度一覧は API / MCP からも取得できます。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.zeimu-kaikei.ai/v1/programs?industry_jsic=K&tier=S,A&limit=50"

MCP クライアントからは search_programs(industry_jsic="K", tier=["S","A"]) で呼べます。詳細は API reference および MCP tools 参照。
