業種別 (JSIC) 補助金・融資・税制・行政処分

更新: 2026-04-29 / Bookyou株式会社 (梅田茂利)

税務会計AI は、JSIC (日本標準産業分類) 大分類 A〜T 単位で日本の公的支援制度を集約しています。事業の主たる業種コードを選択すると、その業種に該当する制度・採択事例・出典 URL を一次資料リンク付きで一覧できます。現在、合計 1027 件の業種×制度クロスリファレンスを公開しています。

出典は中央省庁・都道府県・市区町村・日本政策金融公庫等の公式ページに限定しています。集約サイトサイト (noukaweb / hojyokin-portal / biz.stayway 等) は対象外です。業種コードと制度の適合性は、制度の対象者要件・名称キーワード・出典ドメイン等から税務会計AI が推定したものです。最終的な申請可否は公募要領をご確認ください。

API / MCP で取得

業種絞り込みは API / MCP の industry_jsic パラメータで実行できます。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.zeimu-kaikei.ai/v1/programs?industry_jsic=A&tier=S,A"

MCP クライアント (Claude Desktop, Cursor, ChatGPT) からは search_programs(industry_jsic="A") で呼べます。詳細は API reference および MCP tools 参照。
