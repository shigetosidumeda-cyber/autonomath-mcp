JSIC E 製造業 (中小企業) 補助金・税制 主要20件

JSIC E (製造業 中小企業特化) に該当する事業者向けの 主要制度を、補助金額の大きい順に上位20件まで集約しました。 各制度は jpcite の制度詳細ページに、出典は一次情報の URL に直接リンクしています。

主要制度（tier S/A/B）

  • 小規模事業者持続化補助金 一般型 通常枠 (第19回) tier S ・ 補助金・交付金 ・ 最大 50万円 出典ページを開く
  • 小規模事業者持続化補助金（一般型・通常枠）商工会地区分（第19回） tier S ・ 補助金・交付金 ・ 金額非公表 出典ページを開く
  • 小規模事業者持続化補助金（創業型）商工会地区分（第3回） tier S ・ 補助金・交付金 ・ 金額非公表 出典ページを開く
  • 令和7年度補正予算 省エネ・非化石転換補助金（工場・事業場型） tier S ・ 補助金・交付金 ・ 金額非公表 出典ページを開く
  • SHIFT事業（工場・事業場における先導的な脱炭素化取組推進事業） tier A ・ 補助金・交付金 ・ 最大 50,000万円 出典ページを開く
  • 脱炭素技術等による工場・事業場の省CO2化加速事業（SHIFT事業） tier A ・ 補助金・交付金 ・ 200万円 〜 50,000万円 出典ページを開く
  • Go-Tech 事業 (成長型中小企業等研究開発支援事業, 旧サポイン) tier A ・ 公的支援制度 ・ 最大 9,750万円 出典ページを開く
  • ものづくり・商業・サービス生産性向上促進補助金 (ものづくり補助金) tier A ・ 補助金・交付金 ・ 最大 8,000万円 出典ページを開く
  • ものづくり・商業・サービス生産性向上促進補助金 tier A ・ 補助金・交付金 ・ 最大 1,250万円 出典ページを開く
  • 【産業振興課】ものづくり産業脱炭素化促進事業補助金について更新しました tier A ・ 補助金・交付金 ・ 金額非公表 出典ページを開く
  • 令和4年度補正和歌山県ものづくり生産力高度化事業費補助金 tier A ・ 補助金・交付金 ・ 金額非公表 出典ページを開く
  • 新事業進出補助金(旧事業再構築補助金後継) tier B ・ 補助金・交付金 ・ 1,500万円 〜 9,000万円 出典ページを開く
  • 成長型中小企業等研究開発支援事業（Go-Tech事業） tier B ・ 奨励・インセンティブ制度 ・ 最大 4,500万円 出典ページを開く
  • JAPANブランド育成支援等事業（ものづくり補助金グローバル枠） tier B ・ 補助金・交付金 ・ 最大 4,000万円 出典ページを開く
  • 中小企業投資促進税制 tier B ・ 公的支援制度 ・ 50万円 〜 3,000万円 出典ページを開く
  • ものづくり・商業・サービス生産性向上補助金 tier B ・ 補助金・交付金 ・ 750万円 〜 3,000万円 出典ページを開く
  • ものづくり補助金（グローバル枠） tier B ・ 補助金・交付金 ・ 最大 3,000万円 出典ページを開く
  • 宮城県食品製造業省エネ機器等導入促進支援事業費補助金 tier B ・ 補助金・交付金 ・ 最大 2,000万円 出典ページを開く
  • 中小企業省力化投資補助金 tier B ・ 補助金・交付金 ・ 200万円 〜 1,500万円 出典ページを開く
