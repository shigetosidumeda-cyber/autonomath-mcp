更新: 2026-04-29 / EN: Programs for Wholesale and retail trade (JSIC I) / Bookyou株式会社 (梅田茂利)

TL;DR

JSIC I (卸売業、小売業 / Wholesale and retail trade) 事業者向けの日本の公的支援制度を、税務会計AI が業種要件・名称キーワード・出典ドメインの3軸でクロスリファレンス抽出しました。本ページには 26 件 の制度を Tier (S / A / B) 別にソートして掲載しています。各制度ページから一次資料 URL に直接アクセスできます。

業種コード I の概要

JSIC (日本標準産業分類) における大分類 I は「卸売業、小売業」です (英語: Wholesale and retail trade)。この業種に直接マッピングされる制度のほか、業種横断的な中小企業向け制度 (IT 導入補助金、ものづくり補助金、事業再構築補助金等) も並行して活用できます。本ページの掲載順は Tier (信頼度) → 提供主体 → 制度名のアルファベット順です。

API / MCP で取得

JSIC I (卸売業、小売業) の制度一覧は API / MCP からも取得できます。

curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.zeimu-kaikei.ai/v1/programs?industry_jsic=I&tier=S,A&limit=50"

MCP クライアントからは search_programs(industry_jsic="I", tier=["S","A"]) で呼べます。詳細は API reference および MCP tools 参照。
