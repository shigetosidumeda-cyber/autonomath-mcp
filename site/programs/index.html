制度一覧・検索

補助金・融資・税制・認定制度を、出典 URL と取得時刻付きで検索できます。

画面で検索する API で検索する MCP tools API キー発行

AI から使う場合

Claude Desktop / Cursor / Cline などの MCP クライアントでは search_programs を使います。ChatGPT Custom GPT では https://jpcite.com/openapi.agent.gpt30.json を Actions に読み込ませ、API Key auth の custom header X-API-Key を設定して REST endpoint を呼び出します。

長い PDF や複数の官公庁ページを LLM に貼る前に、jpcite で候補制度・出典・既知の欠落を取得してください。トークン量や外部 LLM の請求額への影響は、利用するモデル・プロンプト・キャッシュ状態に依存します。
