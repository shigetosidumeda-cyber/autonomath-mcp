お知らせ — 制度変更検出

jpcite が制度改正履歴 (追記専用ログ) から自動検出した、日本の公的制度 (補助金・税制・認定・法令) の変更履歴です。 直近 50 件を月別に表示します。 RSS は /rss.xml で配信。

カテゴリで絞り込み

  • すべて
  • 補助金
  • 税制
  • 認定
  • 法令

URL クエリパラメータ ?category=subsidy 等で同等のフィルタが可能です。 ページネーションは ?page=2 形式。

直近の変更検出 (最大 50 件)

現在、直近 7 日間に検出された変更はありません。 制度変更が検出され次第、 自動でこのページに反映されます。

直近 7 日間に検出された変更がない場合は、この案内のみを表示します。

API / MCP で取得

同じ差分を機械可読 (JSON) で取得するには get_amendment_diff を呼び出してください。

curl -H "X-API-Key: YOUR_API_KEY" \
 "https://api.jpcite.com/v1/am/amendments?since=2026-04-22T00:00:00Z"

Claude Desktop / Cursor / Cline などの MCP クライアントでは get_provenance(entity_id="...") 経由で同じ差分ログにアクセスできます。ChatGPT Custom GPT では OpenAPI Actions 経由で同等の REST endpoint を使えます。

本変更検出は制度改正履歴 追記専用ログから自動で行われており、 公式発表との差異がある場合は公式情報をご確認ください。 §52 税理士法に基づき、 個別の税務判断は税理士にご相談ください。 本ページは法的助言・税務助言・申請代行を構成するものではありません。
