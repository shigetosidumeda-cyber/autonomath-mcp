Gemini CLI と連携する

インストール

uvx autonomath-mcp

設定ファイル

~/.gemini/settings.json に以下を追記します。

{
 "mcpServers": {
 "jpcite": {
 "command": "uvx",
 "args": ["autonomath-mcp"]
 }
 }
}

本番運用は "env": {"AUTONOMATH_API_KEY": "sk_..."} を追加します。

動作確認

新しいターミナルで gemini を起動し、ツールが認識されれば接続完了です。
