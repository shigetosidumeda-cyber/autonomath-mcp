Integrations / Windsurf

Windsurf で 税務会計AI を使う

5 分で動く統合ガイド (登録不要 / Free 50 req/月)

所要時間: 5 分以内。必要なもの: Windsurf 1.0 以降 (Codeium 公式 IDE) + Python 3.10 以降 (pip または uvx)。API キー不要 — 50 req/月 per IP は匿名で動きます。stdio 接続、protocol 2025-06-18 対応。
ボリューム: プログラム 11,684 件 (検索可) / 全 14,472 件 / 89 MCP tools / autonomath-mcp v0.3.1。Windsurf Cascade は MCP の stdio をネイティブサポートするので JSON 1 個で 89 tools を Cascade から直接呼べます。

Step 1 — API キーを取得 (任意)

50 req/月 までは認証不要 (anonymous tier、IP 単位、JST 月初 00:00 リセット)。本番運用は Dashboard から API キー (sk_...) を発行 → Stripe metered で ¥3/req 自動課金 (税込 ¥3.30、最低料金・年契約・tier なし)。signup form は トップページ または Dashboard。

Step 2 — autonomath-mcp をインストール

Windsurf 内蔵ターミナル (Ctrl+`) で:

pip install autonomath-mcp

または uvx で ephemeral 実行 (PATH 汚染なし、自動アップデート):

uvx autonomath-mcp

動作確認:

autonomath-mcp --version
# → autonomath-mcp 0.3.1

Step 3 — mcp_config.json を編集

Windsurf の Cascade は MCP 設定を mcp_config.json から読みます。2 つの編集経路:

経路 A — UI から (推奨):

  1. Windsurf 右上 Settings アイコン → Cascade → MCP Servers セクション
  2. + Add new MCP server または Edit raw config をクリック

経路 B — ファイルを直接編集:

OS mcp_config.json
macOS~/.codeium/windsurf/mcp_config.json
Windows%USERPROFILE%\.codeium\windsurf\mcp_config.json
Linux~/.codeium/windsurf/mcp_config.json

ファイルが無ければ新規作成。中身は以下を貼り付け (既存があれば "zeimu-kaikei" キーだけ追加):

{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp"
    }
  }
}
API キー入りバージョン (50 req/月 を超える運用) 
{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp",
      "env": {
        "ZEIMU_KAIKEI_API_KEY": "sk_xxxxxxxxxxxx"
      }
    }
  }
}
uvx 版 (PATH 解決の trouble を回避) 
{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "uvx",
      "args": ["autonomath-mcp"]
    }
  }
}

JSON 保存後、Cascade パネルの Refresh アイコン (円形矢印) をクリック、または Windsurf を再起動。MCP Servers 一覧に zeimu-kaikei (89 tools) と表示されれば成功。

Step 5 — Troubleshoot

1. MCP Servers 一覧に表示されない / "Failed to start"

  • Cascade パネル → MCP Servers 該当行 → Show Logs でスタックトレース確認
  • 大抵は autonomath-mcp: command not found → which autonomath-mcp 結果を絶対パスで指定
  • あるいは "command": "uvx", "args": ["autonomath-mcp"] に切替
  • Refresh ボタンで再ロード

2. JSON syntax error

  • python -m json.tool ~/.codeium/windsurf/mcp_config.json で valid 確認 (macOS)
  • カンマ抜け / 余分なカンマで詰まりがち。VS Code 互換 (Ctrl+Shift+P → JSON: Validate)

3. Cascade がツールを呼んでくれない

  • プロンプトに zeimu-kaikei MCP の search_programs を使って のように tool 名を明示
  • Cascade の Mode が Write になっていることを確認 (Read-only モードはツール承認をスキップしない)

4. 50 req/月 の残量を確認

  • Cascade に get_usage_status を呼んで残クォータを教えて と指示。このツール呼び出し自体はクォータを消費しません (probe-only)
  • JST 月初 00:00 に自動リセット
