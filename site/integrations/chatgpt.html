Integrations / ChatGPT (GPTs Action)

ChatGPT GPTs に 税務会計AI を統合

5 分で動く統合ガイド (登録不要 / Free 50 req/月)

所要時間: 5 分以内。必要なもの: ChatGPT Plus / Team / Enterprise (Custom GPT 作成権限)。API キー不要 — anonymous tier 50 req/月 per IP は認証なしで動きます。
仕組み: ChatGPT GPTs は MCP プロトコルを直接話せませんが、OpenAPI 3.x スペックを Import すると Actions として REST 経由で呼べます。MCP 版 (Claude Desktop / Cursor / Cline) と同じ 11,684 件のプログラム + 9,484 件の法令 + 13,801 件の適格事業者 + 2,065 件の判例 + 50 件の税制ルールセットにアクセス。MCP 89 tools と REST 表面はデータソース共通。

本ページは ChatGPT 専用ガイドです。 stdio 直結 (MCP) 版は Claude Desktop / Cursor / Cline を参照。

Step 1 — API キーを取得 (任意)

50 req/月 までは認証なしで動作します (anonymous tier、IP 単位、JST 月初 00:00 リセット)。本番運用や複数ユーザー公開の Custom GPT は Dashboard から API キー (sk_...) を発行してください。Stripe metered で ¥3/req 自動課金 (税込 ¥3.30、最低料金・年契約・tier なし)。signup form は トップページ または Dashboard から。

Step 2 — Custom GPT を新規作成 + OpenAPI Import

  1. ChatGPT (chatgpt.com) を開いてログイン
  2. 左サイドバー Explore GPTs → + Create (右上)、または直接 chatgpt.com/gpts/editor
  3. 上部タブを Configure に切り替え (Create タブのウィザードはスキップ)
  4. Configure タブの一番下に Actions セクション → + Create new action
  5. Action 編集画面の Schema セクションで Import from URL をクリック → 以下を貼り付け:
https://api.zeimu-kaikei.ai/v1/openapi.json

Import 失敗時は手動 fetch: curl -s https://api.zeimu-kaikei.ai/v1/openapi.json > openapi.json → そのテキストを Schema 欄に貼り付け。

Instructions (必須): Configure タブの Instructions に以下をペースト (Action から呼ばずに訓練データで答えるのを防ぎます):

あなたは日本の公的制度 (補助金・融資・税制優遇・認定制度) の検索アシスタントです。

ルール:
1. 必ず Actions の 税務会計AI API を呼んでから回答する。記憶や訓練データから補助金額・締切を答えない。
2. 各回答に必ず source_url + source_fetched_at (出典取得日) を併記する。
3. 個別の税務・法務助言は税理士法 §52 により禁止。『上記は情報検索結果です。具体的な助成判断は税理士・行政書士へ』を末尾に必ず付ける。
4. 検索のヒット件数が 0 のときは『該当なし』と正直に答える。類似制度をでっち上げない。
5. 適格請求書発行事業者 (T-番号) は search_invoice_registrants で必ず実検索。
6. 排他ルール (補助金併用可否) は check_exclusions で必ず判定する。

レスポンス形式:
- search 系は {total, limit, offset, results[]} 形式
- 各 row は source_url + source_fetched_at を持つ → 引用形式で示す
- _disclaimer フィールドが付いていたらそのまま末尾に表示

Authentication: Anonymous は None、API キーは API Key → Bearer で sk_... をそのまま貼り付け。

Privacy policy URL (公開時に必須):

https://zeimu-kaikei.ai/privacy.html

Step 3 — 動作確認クエリ

Custom GPT のプレビュー欄に貼り付け:

search_programs を呼んで prefecture=東京都 + target_type=中小企業 + keyword=省エネ で 3 件取得。primary source URL もつけて。

ChatGPT が search_programs Action 呼び出しの承認ダイアログを出します (初回のみ)。Always Allow をクリックすると以後自動で呼びます。

Step 5 — Troubleshoot

1. 401 Unauthorized

  • API キー (sk_...) を Bearer で送るとき、頭の Bearer プレフィックスは ChatGPT が自動で付与。キー本体だけを貼る
  • Anonymous tier で動かしているのに 401 が出るときは、ChatGPT 側が前回のキャッシュを送っている → Action を一度削除して再 Import

2. ChatGPT が Action を呼ばずに training data から答える

  • Instructions 冒頭の "必ず Actions の 税務会計AI API を呼んでから回答する" を強化
  • "検索結果なしで答えるな" / "If you cannot call the API, respond '検索失敗' and stop" を追記
