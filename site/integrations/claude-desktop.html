Integrations / Claude Desktop

Claude Desktop で 税務会計AI を使う

5 分で動く統合ガイド (登録不要 / Free 50 リクエスト/月)

所要時間: 5 分以内 (実測中央値 3 分)。必要なもの: Claude Desktop 1.0 以降 + Python 3.10 以降 (pip または uvx)。API キー不要 — 50 リクエスト/月 per IP は匿名で動きます。

Step 1 — autonomath-mcp をインストール

ターミナルを開いて 1 行:

pip install autonomath-mcp

または毎回最新版を取ってくる ephemeral run (pip install を残したくない場合):

uvx autonomath-mcp

動作確認:

autonomath-mcp --version
# → autonomath-mcp 0.3.1

Step 2 — claude_desktop_config.json を編集

OS ごとに設定ファイルの場所が違います:

OS 設定ファイルパス
macOS~/Library/Application Support/Claude/claude_desktop_config.json
Windows%APPDATA%\Claude\claude_desktop_config.json
Linux~/.config/Claude/claude_desktop_config.json

ファイルが無ければ新規作成。中身は以下を貼り付け (既に他の MCP サーバを使っている場合は mcpServers オブジェクトに "zeimu-kaikei" キーだけ追加):

{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp"
    }
  }
}

API キーを使う場合 (50 リクエスト/月 を超える運用): "env": {"ZEIMU_KAIKEI_API_KEY": "sk_..."} を追加するだけ。詳細は Dashboard から発行。

API キー入りバージョン (option) 
{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp",
      "env": {
        "ZEIMU_KAIKEI_API_KEY": "sk_xxxxxxxxxxxx"
      }
    }
  }
}

Step 3 — Claude Desktop を再起動

ファイル保存後、Claude Desktop を完全終了 (Cmd+Q / Windows: タスクバーから終了) して再起動。

チャット入力欄の左下に ハンマーアイコン が出ていれば成功。クリックすると zeimu-kaikei 配下に 89 MCP ツール (search_programs / get_program / prescreen_programs / smb_starter_pack / search_laws / search_invoice_registrants / 他) が並びます。

Step 4 — 最初のプロンプト

新しいチャットで以下を貼り付け:

中小企業向けの省エネ補助金を3件探して。primary source URL もつけて。

Claude が自動で search_programs ツールを呼び、JSON レスポンス (補助金 3 件 + source_url + fetched_at) を取得して整形して返します。

Troubleshooting

1. ハンマーアイコンが出ない

  • Claude Desktop を完全終了 (タスクバー / メニューバー常駐も含めて) してから再起動
  • JSON が valid か確認: python -m json.tool ~/Library/Application\ Support/Claude/claude_desktop_config.json (macOS)
  • カンマ抜け / 余分なカンマで詰まりがち。VS Code などで開いて syntax error を確認

2. ツール呼び出しがタイムアウト / "command not found: autonomath-mcp"

  • Claude Desktop は ログインシェルの PATH を継承しない。which autonomath-mcp でフルパスを取って "command": "/Users/you/.local/bin/autonomath-mcp" のように絶対パスで指定
  • あるいは "command": "uvx", "args": ["autonomath-mcp"] に切り替え (uvx がインストール済みなら PATH 解決が楽)

3. "Server zeimu-kaikei failed to start"

  • Claude Desktop の Developer → Open Logs から mcp-server-zeimu-kaikei.log を確認
  • 大抵は Python バージョン (3.9 以下は不可、3.10+ 必須) か pip インストールミス

4. レスポンスが日本語で文字化け (Windows)

  • Windows は環境変数 "env": {"PYTHONIOENCODING": "utf-8"} を JSON に追加

5. 50 リクエスト/月 の残量を確認したい

  • Claude に「get_usage_status を呼んで残クォータを教えて」と指示。このツール呼び出し自体はクォータを消費しません (probe-only)
  • JST 月初 00:00 に自動リセット
