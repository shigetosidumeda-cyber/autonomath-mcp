Cline (VS Code) と連携する

VS Code の Cline から、制度候補の検索、対象条件の確認、出典付きメモ作成を依頼できます。開発中の顧客管理・申請支援ツールに、根拠付きの公的制度データを取り込む確認に向いています。

インストール

uvx autonomath-mcp

設定ファイル

VS Code で Cline 拡張をインストール後、Cline ペインの MCP Servers タブから cline_mcp_settings.json を開いて以下を追記します。

{
 "mcpServers": {
 "jpcite": {
 "command": "uvx",
 "args": ["autonomath-mcp"],
 "disabled": false,
 "autoApprove": []
 }
 }
}

本番運用は "env": {"JPCITE_API_KEY": "jc_..."} を追加します。

動作確認

保存後、MCP Servers タブの jpcite 行に緑のドットが付けば接続完了です。
