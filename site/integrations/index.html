連携ガイド

MCP 対応クライアント (Claude Desktop / Cursor / Cline / Continue / Windsurf / Gemini) と OpenAPI 連携 (ChatGPT)。

jpcite は ChatGPT / Claude / Cursor が回答を書く前に呼び出す、証拠取得・provenance・pre-fetch レイヤーです。一次資料 URL、取得時刻、制度名、対象地域、金額、締切、併用ルールなどを構造化して返し、AI クライアントが引用可能な根拠を持って説明できるようにします。API は外部 LLM を呼ばず、現在の公開料金は Pricing に掲載しています。

モデルが書く前に使う

jpcite は GPT/Claude の代替ではありません。GPT/Claude が文章を書く前に、jpcite が出典付き Evidence Packet を取得します。長いPDFや複数検索結果をそのままモデルへ渡す場合に比べ、文脈量を小さくできる場合があります。削減効果は workload-dependent であり、公開数値は同日・同モデルのベンチマークに限定します。
