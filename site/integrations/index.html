Integrations — MCP-native AI agent 統合 + SDK

我々の API は 89 MCP tools として AI agent から直接呼べます。Claude Desktop / Cursor / Cline / Continue.dev / Windsurf / Gemini CLI 全部 stdio 接続、protocol 2025-06-18 対応。
ChatGPT GPTs は OpenAPI Action 経由。autonomath-mcp v0.3.1 / pip install 1 行 + JSON 1 行で動きます。

5 分の約束: 6 つの AI agent (Claude Desktop / Cursor / Cline / Continue / Windsurf / Gemini) は MCP stdio をネイティブサポート。pip install autonomath-mcp + JSON / YAML 1 行 → restart でハンマー / Cascade / Continue Chat 配下に 89 tools がそのまま並びます。
volume: プログラム 11,684 件 (検索可) / 全 14,472 件 / 9,484 法令 / 13,801 適格事業者 / 2,065 判例 / 50 税制ルールセット。
Billing: 50 req/月 per IP は登録不要・認証不要 (匿名 anonymous tier)。本番運用は API キーで ¥3/req metered Stripe (税込 ¥3.30) 完全従量、tier・最低料金・年契約なし、signup 後 Stripe が自動課金。
注: 本 API は情報検索のみを提供します。税理士法 §52 により、個別の税務助言は税理士にご相談ください。

API ベース URL + 認証

  • Base URL: https://api.zeimu-kaikei.ai/v1
  • OpenAPI: /v1/openapi.json
  • Auth: X-API-Key: sk_... ヘッダー (Bearer も受理)
  • Anonymous: 50 リクエスト/月 per IP, JST 月初 00:00 リセット (no key required)
