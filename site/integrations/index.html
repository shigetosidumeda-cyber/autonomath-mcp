AI クライアント統合 (5 分で動く)

MCP サーバ または OpenAPI Actions として直接 AI ツールから 税務会計AI を呼べる、ステップバイステップ統合ガイド。登録不要 / 50 リクエスト/月 無料。

Claude Desktop

Anthropic 公式デスクトップアプリに 89 MCP ツールを追加。pip install + JSON 1 行。

所要 3 分

Cursor

Cursor IDE の MCP Servers タブから設定。uvx または .mcpb URL で 89 ツール。

所要 4 分

Cline (VS Code)

VS Code 拡張 Cline / Roo Code に MCP サーバ追加。read-only ツールは auto-approve 可。

所要 5 分

OpenAI Custom GPT

ChatGPT Custom GPT に /v1/openapi.json を Import。111 エンドポイントが Actions として動作。

所要 5 分

すべて ZEIMU_KAIKEI_API_KEY 環境変数を読みます。未設定なら anonymous tier (50 リクエスト/月 per IP, JST 月初リセット) で動作します。

API ベース URL + 認証

  • Base URL: https://api.zeimu-kaikei.ai/v1
  • OpenAPI: /v1/openapi.json
  • Auth: X-API-Key: sk_... ヘッダー (Bearer も受理)
  • Anonymous: 50 リクエスト/月 per IP, JST 月初 00:00 リセット (no key required)
  • Authenticated: ¥3 / req 完全従量 (税込 ¥3.30) / Pricing

エラー処理 (全サンプル共通)

HTTP 意味 対処
401Auth 失敗ZEIMU_KAIKEI_API_KEY を確認
403Key 失効 / quota 枯渇Dashboard で確認
429Rate limitRetry-After 秒待機 (サンプルは自動リトライ)
5xxServer error指数バックオフで再試行 (サンプルは 0.5s/1s/2s)
