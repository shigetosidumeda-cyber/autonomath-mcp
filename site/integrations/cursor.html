Cursor で 税務会計AI を使う

5 分で動く統合ガイド (登録不要 / Free 50 リクエスト/月)

所要時間: 4 分以内。必要なもの: Cursor 0.45 以降 (MCP 対応版) + Python 3.10 以降 (pip または uvx)。API キー不要 — 50 リクエスト/月 per IP は匿名で動きます。

Step 1 — autonomath-mcp をインストール

ターミナルで:

pip install autonomath-mcp

あるいは uvx で ephemeral 実行 (PATH 汚染なし、自動アップデート):

uvx autonomath-mcp

Step 2 — Cursor の MCP 設定を開く

Cursor を起動し、以下の手順で MCP 設定画面を開きます:

  1. Cursor Settings を開く (Cmd+, / Windows: Ctrl+,)
  2. 左サイドバーから Features → MCP Servers を選択
  3. + Add new MCP サーバ をクリック

Step 3 — JSON 設定を貼り付け

Cursor は ~/.cursor/mcp.json (User-level) または .cursor/mcp.json (Project-level) に MCP 設定を保存します。Settings UI からの追加はこのファイルに書き込まれます。

Option A — pip で入れたパッケージを使う (最速)

{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp"
    }
  }
}

Option B — uvx で毎回最新を取る (PATH 解決が確実)

{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "uvx",
      "args": ["autonomath-mcp"]
    }
  }
}

Option C — .mcpb バンドル URL (DXT-style 1-click インストール、Cursor 0.50+ 対応版)

https://zeimu-kaikei.ai/downloads/autonomath-mcp.mcpb

Cursor の "Install from MCPB" メニューにこの URL を貼り付け。Python ランタイムを Cursor が自動で解決します。

API キー入りバージョン (50 リクエスト/月 を超える運用) 
{
  "mcpServers": {
    "zeimu-kaikei": {
      "command": "autonomath-mcp",
      "env": {
        "ZEIMU_KAIKEI_API_KEY": "sk_xxxxxxxxxxxx"
      }
    }
  }
}

API キーは Dashboard で発行 (¥3/リクエスト 完全従量, 税込 ¥3.30, no minimum)。

Step 4 — 設定を反映

JSON 保存後、MCP Servers 画面で zeimu-kaikei 行の Refresh アイコン (円形矢印) をクリック。緑のドットが付いて 89 tools と表示されれば接続完了。

反映されない場合は Cursor を完全終了 (Cmd+Q) して再起動。

Step 5 — Cursor チャットで試す

Cursor のチャット (Cmd+L / Ctrl+L) を開き、以下を貼り付け:

@zeimu-kaikei 中小企業向けの省エネ補助金を3件探して。primary source URL もつけて。

Cursor が search_programs ツールを呼び、JSON レスポンスを整形して返します。@zeimu-kaikei プレフィックスは省略可ですが、付けるとツールの選択範囲が明確になります。

Troubleshooting

1. MCP Servers 画面に追加しても緑のドットが付かない

  • autonomath-mcp が PATH に通っているか確認: which autonomath-mcp
  • 絶対パスで指定: "command": "/usr/local/bin/autonomath-mcp" (or which の出力)
  • あるいは "command": "uvx", "args": ["autonomath-mcp"] に切り替え

2. ツールが 0 tools と表示される

  • Cursor を完全再起動 (Cmd+Q / Windows 通知領域から終了)
  • Cursor の Output → MCP パネルでサーバ起動ログを確認 (Python トレースバックが出ていれば pip 再インストール)

3. autonomath-mcp: command not found

  • pip がユーザー領域にインストール: ~/.local/bin/autonomath-mcp (Linux/macOS) や %APPDATA%\Python\Scripts\autonomath-mcp.exe (Windows) を command に直接指定
  • python -m jpintel_mcp.mcp.server でも起動可: "command": "python", "args": ["-m", "jpintel_mcp.mcp.server"]

4. Cursor チャットがツールを呼んでくれない

  • プロンプトに @zeimu-kaikei を付けて指名
  • "using zeimu-kaikei tools, ..." のように 英語で命令しても効く
  • Auto モードよりも Agent モードのほうがツール使用に積極的

5. 50 リクエスト/月 の残量を確認

  • @zeimu-kaikei get_usage_status を呼んで残クォータを教えて — このツール呼び出し自体はクォータを消費しません (probe-only)
  • JST 月初 00:00 に自動リセット
