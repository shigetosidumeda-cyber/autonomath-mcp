よくある質問 (FAQ)

API キーはどこで発行できますか?

3 経路あります: 1) Pricing で有料 API キーを発行 (Stripe Checkout、 ¥3.30/unit 完全従量、 即時発行) / 2) メールでトライアル (14 日 / 200 req、 マジックリンク認証、 カード不要) / 3) 登録不要の匿名 3 リクエスト/日/IP (curl / Playground でその場で利用、 JST 翌日 00:00 リセット)。

解約方法は?

ダッシュボード から Stripe Customer Portal を開き、 サブスクリプションをキャンセルしてください。 キャンセル後も当月末までは API アクセス可能で、 当月利用分のみ ¥3/billable unit 従量請求が発生します。 次月以降の課金は停止します。 解約違約金・最低利用期間はありません。

API キーをなくしました。 再発行できますか?

ダッシュボード の「キーを再発行」ボタンから rotate できます。 古いキーは即時無効化されるため、 クライアント側を更新してから実行してください。

請求書 (適格請求書) のダウンロード方法は?

Stripe Customer Portal から PDF ダウンロード可能です。 jpcite の適格請求書発行事業者番号は T8010001213708 (Bookyou株式会社) です。

API レートリミット / 利用上限の確認方法は?

各レスポンスの X-RateLimit-Remaining ヘッダ、 もしくは ダッシュボード の使用量グラフで確認できます。 月次予算 cap は ダッシュボードから任意で設定可能。 cap 超過時は HTTP 503 + cap_reached:true で停止します。

MCP の自動認証 (go.html) と通常の API キー発行は何が違いますか?

go.html は Claude Desktop など MCP クライアントが提示する確認コード (例 ABCD-1234) を受け取って Stripe Checkout に紐付け、 自動でキーを MCP サーバーへ配るための専用フローです。 通常の API キー (curl / Cursor / ChatGPT Custom GPT 等) を取得したい場合は Pricing を使ってください。

Claude Desktop / Cursor / Cline / ChatGPT Custom GPT への設定方法は?

統合ガイドを各クライアント向けに用意しています: Claude Desktop · Cursor · Cline · ChatGPT Custom GPT。 支払い完了後の success ページ でも同じ設定ブロックをコピー&貼り付けできます。

電話 / Slack Connect / Zoom サポートは利用できますか?

提供していません。サポートはメール (info@bookyou.net) と GitHub Issues に限定しています。 個別導入会議、専用チャット、個別契約交渉は受け付けていません。 法人購買・法的方針は 法人購買 と 法的姿勢 をご覧ください。

税理士法 §52 / 弁護士法 §72 該当の業務はおこないますか?

おこないません。 jpcite は日本の制度情報の検索インデックス API です。 税務助言・法律相談・申請代行・社労士業務は対象外で、 出力結果は士業 (税理士・弁護士・社労士・行政書士) による確認の代替にはなりません。

外部 LLM の請求額削減は保証されますか?

保証しません。 jpcite は Evidence Packet (出典 URL + 取得時刻 + content_hash + known_gaps) を返す検索 API で、 入力文脈量の参考削減見込みは compression.input_context_reduction_rate として返しますが、 これは入力文脈量だけの参考推定で、 出力 tokens / reasoning tokens / cache / provider tool / search 料金 / 為替を含みません。

データ更新頻度は?

主要公開行に source_url + source_fetched_at + content_hash を付与しています。 優先度の高い出典の再取得中央値は約 7 日で、 鮮度の per-source 内訳は データ鮮度 ページで公開しています。 法令改正の差分は日次走査、 出典 URL の死活監視は毎晩自動実行しています。

個人情報・データの取り扱い / GDPR / APPI 開示請求は?

プライバシーポリシー をご覧ください。 APPI に基づく開示・削除請求は info@bookyou.net へメールでお願いします。
