士業 案件紹介

認定支援機関 / 税理士 / 社労士 / 中小企業診断士 / 行政書士 / 弁護士 向けの案件紹介プログラム。jpcite を利用する 中小企業 が 専門家 を探す際、地域・業種・専門分野の一致度でランキング表示されます。

登録から支払いまで

  1. 登録 — 下のフォームで事務所名・法人番号 (13 桁)・専門分野・成約単価を送信。
  2. 法人番号 確認 — 国税庁 適格請求書事業者 13,801 社の公開データと照合。
  3. Stripe 振込先登録 (簡易版) — 振込先口座を登録。jpcite は Stripe 経由で支払いを中継します。
  4. 公開 — 検索結果に表示開始。ランキングは「専門性一致 → 実績数 → ID 昇順」の順 (広告料金による改変なし、景表法対応)。
  5. 成約報告 — 受任が決まったら /v1/advisors/report-conversion に referral_token と受任額を送信。
  6. 月末支払い — 前月の成約分を、翌月 末日に Stripe 振込で自動送金。

ランキング開示 (景表法対応)

検索結果は、以下の基準で順位付けされます。広告料金によるランキング操作は一切ありません。

  1. 専門分野 (specialty) の一致度 — 中小企業 のクエリから抽出した制度カテゴリと 専門家 の登録専門分野の重複数。
  2. 業種 の一致度 — 中小企業 の業種 (農業 / 製造 / IT / サービス / 建設 / 小売) と 専門家 の得意業種の一致。
  3. 成約実績数 — これまでに jpcite 経由で成約した件数。
  4. ID 昇順 (同点時の決定基準) — 登録順。

本リストは広告ではありません。専門性と実績のマッチングで並び替えています (景品表示法 対応)。

士業法コンプライアンス

jpcite は案件紹介の場として動作します。専門家と中小企業の契約は直接締結していただき、jpcite は契約の当事者・公式紹介者にはなりません。

  • 税理士法 §33-2 — 名義貸しの禁止。専門家 は自らの資格で業務を受任し、jpcite の名義では業務を行えません。
  • 社会保険労務士法 §27 — 独占業務 (申請書類作成) は社労士のみ。それ以外の助言は 専門家 の判断です。
  • 弁護士法 §72 — 報酬を得る法律事務は弁護士のみ。専門家 間で業務範囲を適正に仕分けてください。
  • 景表法 — ランキング方式は上記のとおり開示。広告料金による操作なし。
  • APPI — 法人番号は公開情報のため取得可。個人番号は取得しません。IP は salted SHA256 で hash 保存のみ。
