登録から支払いまで

  1. 登録 — 下のフォームで事務所名・法人番号 (13 桁)・専門分野・成約単価を送信。
  2. 法人番号 verify — 国税庁 適格請求書事業者 13,801 社の公開データと照合。
  3. Stripe Connect Express — 振込先口座を登録。AutonoMath は Stripe 経由で支払いを中継します。
  4. 公開 — 検索結果に表示開始。ランキングは「専門性一致 → 実績数 → ID 昇順」の順 (広告料金による改変なし、景表法対応)。
  5. 成約報告 — 受任が決まったら /v1/advisors/report-conversion に referral_token と受任額を送信。
  6. 月末支払い — 前月の成約分を、翌月 末日に Stripe Transfer で自動振込。

ランキング開示 (景表法対応)

検索結果は、以下の基準で順位付けされます。広告料金によるランキング操作は一切ありません。

  1. 専門分野 (specialty) の一致度 — SME のクエリから抽出した制度カテゴリと advisor の登録専門分野の重複数。
  2. 業種 (industry) の一致度 — SME の業種 (agri / manufacture / it / service / construction / retail) と advisor の得意業種の一致。
  3. 成約実績数 (success_count) — これまでに AutonoMath 経由で成約した件数。
  4. ID 昇順 (tie-breaker) — 登録順。

本リストは広告ではありません。専門性と実績のマッチングで並び替えています (景品表示法 対応)。

士業法コンプライアンス

AutonoMath は matching platform として動作します。advisor と SME の契約は直接締結していただき、AutonoMath は introducer-of-record にはなりません。

  • 税理士法 §33-2 — 名義貸しの禁止。advisor は自らの資格で業務を受任し、AutonoMath の名義では業務を行えません。
  • 社会保険労務士法 §27 — 独占業務 (申請書類作成) は社労士のみ。それ以外の助言は advisor の判断です。
  • 弁護士法 §72 — 報酬を得る法律事務は弁護士のみ。advisor 間で業務範囲を適正に仕分けてください。
  • 景表法 — ランキング方式は上記のとおり開示。広告料金による操作なし。
  • APPI — 法人番号は公開情報のため取得可。個人番号は取得しません。IP は salted SHA256 で hash 保存のみ。
