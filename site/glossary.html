用語集 — 公的支援 5 用語

補助金・助成金・融資・税制優遇・認定制度。日本の公的支援を理解する基礎用語の定義と違いを 1 ページで整理しました。各用語に対応する jpcite の API 経路も併記しています。

用語の詳細

補助金 (ほじょきん / Subsidy)

国・自治体が政策目的 (産業振興・地域活性化・脱炭素・DX 推進など) を達成するため、事業者に給付する資金 です。原則として返還不要ですが、成果報告 (実績報告書) は必須で、目的外使用や虚偽申請が判明した場合は返還命令の対象になります。原資は税金のため、競争審査 (公募 + 採択) を伴うのが標準で、採択倍率は制度により 30-80% と幅広い。代表例は事業再構築補助金 (経産省、最大 1.5 億円)・ものづくり補助金 (中小機構、最大 4,000 万円)・IT導入補助金 (中小企業庁、最大 450 万円)。jpcite の検索対象 11,684 件のうち約 80% がこのカテゴリです。

助成金との違い: 助成金は要件充足主義 (要件を満たせば原則受給可)、補助金は競争審査 (採択枠内のみ)。
融資との違い: 融資は返還 + 利息あり、補助金は返還不要 (但し精算払いのため一時的に立替が必要)。
API での絞り込み: 種別「補助金」で検索条件を絞れます (詳細は API リファレンス)。

助成金 (じょせいきん / Grant)

補助金とほぼ同義ですが、特に厚生労働省管轄の雇用・労働関連で使われる用語です。要件を満たせば原則受給可であり、補助金のような競争審査・採択枠の概念がありません (予算枠を超えると年度末に締め切り)。原資は雇用保険二事業 (事業主負担分) です。代表例はキャリアアップ助成金 (非正規→正規転換、最大 80 万円/人)・人材開発支援助成金 (従業員研修費)・両立支援等助成金 (育児/介護休業)・業務改善助成金 (賃上げ + 設備投資)。社会保険料の納付遅延がある事業者は受給不可など、固有要件があります。

補助金との違い: 助成金は要件充足主義 (採択倍率の概念なし)、補助金は競争審査。
申請窓口: 都道府県労働局 / ハローワーク (補助金は経産局や事務局)。
API での絞り込み: 種別「助成金」+ 所管「厚生労働省」で絞れます (詳細は API リファレンス)。

融資 (ゆうし / Loan)

公的金融機関 (日本政策金融公庫・商工中金・信用保証協会保証付き民間融資) からの貸付制度です。返還必須 + 利息有りですが、民間プロパー融資より低利 (年 0.5%-2% 程度)・長期 (5-20 年)・無担保枠あり等、優遇された条件で借りられます。代表例はマル経融資 (商工会議所推薦、無担保無保証、最大 2,000 万円)・新規開業資金 (公庫、最大 7,200 万円)・農業経営基盤強化資金 (スーパーL資金) (公庫、最大 3 億円)・セーフティネット保証 (信用保証協会)。jpcite は担保 / 個人保証人 / 第三者保証人の 3 軸で 108 件の融資制度を分解しています。

補助金との違い: 補助金は返還不要だが精算払い、融資は返還必須だが事前入金 → 資金繰り 補完目的で併用が標準。
審査軸: 信用 (税金滞納なし)・事業計画・担保 / 保証人。
対応 API 経路: GET /v1/loan-programs/search

税制優遇 (ぜいせいゆうぐう / Tax Incentive)

法人税 / 所得税 / 固定資産税 / 消費税の軽減・特別償却・税額控除を行う制度群です。代表例は中小企業投資促進税制 (機械装置 30% 特別償却 or 7% 税額控除)・研究開発税制 (試験研究費の 6-14% 税額控除)・賃上げ促進税制 (給与増加額の最大 40% 税額控除)・固定資産税の特例 (経営力向上計画認定で 1/2 軽減)。減免額が即 資金繰り 改善に直結し、補助金と異なり申請不採択リスクが無い (要件充足主義)。多くは事前認定 (認定制度) + 確定申告で適用、の 2 段階構造です。

補助金との違い: 補助金は 現金給付、税制優遇は納税額減 (赤字企業は効果なし)。
認定制度との関係: 中小企業投資促進税制等は経営力向上計画認定が前提 → 認定制度を先に取得。
対応 API 経路: GET /v1/tax_rulesets/search

認定制度 (にんていせいど / Certification)

行政が事業者を「認定」する制度群です。代表例は経営革新計画 (都道府県認定)・地域未来牽引企業 (経産省選定)・健康経営優良法人 (経産省 + 日本健康会議)・経営力向上計画 (主務大臣認定)・事業継続力強化計画 (経産局認定)。直接の現金給付は無いが、補助金加点 (採択倍率を引き上げる) / 融資条件優遇 (公庫の利率引下げ) / 公的入札加点 (総合評価落札方式) / 税制優遇の前提 等、 補助金・融資・税制を有利に進めるための土台になります。認定取得自体は無料 (印紙代のみ) で、計画書の質が審査基準。jpcite は 66 件の認定制度を構造化しています。

補助金との違い: 補助金は 現金、認定は資格 (補助金/融資/税制の前提)。
取得順序: 認定 → 補助金/税制申請、が標準フロー。
対応 API 経路: GET /v1/am/certifications

よくある質問 (FAQ)

Q. 補助金と助成金は複数併用できますか?
原則として併用可能ですが、同一経費の重複計上は禁止です (「経費の二重計上」は不正受給扱い)。さらに制度ごとに「他の補助金との併用不可」条項が設定されている場合があります。jpcite は 181 本の排他ルール (排他 125 + 前提 17 + 絶対 15 + その他 24) で衝突を機械判定するため、 制度の組合せを投げれば即座に判定が返ります。
Q. 5 用語の優先順位 (申請順) はありますか?
原則: 認定制度 → 補助金/助成金 → 融資 → 税制優遇の順で検討します。認定制度は補助金加点や融資条件優遇の前提となるため最初に取得、補助金/助成金で初期投資の自己負担を圧縮、不足分を融資で調達、最後に税制優遇で適正な節税を行うのが標準フローです。 jpcite に法人プロフィールを投げると、 認定 → 補助金 → 融資の順で受付中の制度を並べます。
Q. 補助金の採択率はどのくらいですか?
制度により大きく異なります。事業再構築補助金は 30-50%、ものづくり補助金は 40-60%、IT導入補助金は 60-80% 程度。jpcite は2,286 件の実採択事例 を保持し、GET /v1/case-studies/search で 制度 ID 別の採択企業一覧を返します。採択率は jpcite が独自試算するのではなく、事務局公表の採択結果から逆算する原則です。
Q. 融資と補助金、どちらが先ですか?
補助金は「精算払い」(事業完了後に支払われる)のため、事業実施期間中の 資金繰り を融資で繋ぐのが標準です。具体的には: (1) 補助金交付決定通知を受領、(2) 公庫マル経 / 信用保証協会保証付き融資で事業資金を借入、(3) 事業実施・経費支払い、(4) 補助金請求・入金、(5) 融資の繰上返済 (任意) という流れ。 jpcite は融資制度と補助金/助成金を 1 リクエストで横断検索できます。
Q. 税制優遇は申請が必要ですか?
税制優遇の多くは「事前認定 + 確定申告で適用」の 2 段階です。例えば中小企業投資促進税制は経営力向上計画の認定 (認定制度) → 設備投資 → 確定申告書に明細書を添付、という流れ。研究開発税制のように事前認定不要のものもあります。jpcite の GET /v1/tax_rulesets/search は要件・申請手続・対象設備を構造化した 50 件の 税制ルール を返します。
