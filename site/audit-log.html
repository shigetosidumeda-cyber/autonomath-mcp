制度変更検出フィード (Audit Log)

公的機関データの差分を毎日 cron で検出。検出のみで個別判断は行いません。 各行は entity_id / field_name / prev_hash / new_hash / detected_at / source_url を含む append-only 記録です。差分メタデータは CC-BY-4.0、原典は各 source_url の公的機関ライセンスに従います。

本サービスは税務助言ではありません

税務会計AI は公的機関が公表する制度・税制・法令情報を検索・整理して提供するサービスで、税理士法 §52 に基づき個別具体的な税務判断・申告書作成代行は行いません。弁護士法 §72 に基づき法律事務も行いません。個別案件は資格を有する税理士・弁護士に必ずご相談ください。詳細は 利用規約 をご確認ください。

使い方 (API + RSS)

同じデータは GET https://api.zeimu-kaikei.ai/v1/am/audit-log で JSON 取得、/audit-log.rss で RSS 取得できます。匿名 50 リクエスト/月 per IP まで無料、認証付きキーは ¥3/リクエスト 税別 (税込 ¥3.30) で従量課金。

curl 'https://api.zeimu-kaikei.ai/v1/am/audit-log?since=2026-04-01&limit=50'

# 認証付き (¥3/req, 税込 ¥3.30)
curl -H 'X-API-Key: sk_live_...' \
     'https://api.zeimu-kaikei.ai/v1/am/audit-log?entity_id=program_xxx_2026'
