LINE bot

スマホ 1 台で、補助金・融資・税制・行政処分を日本語で検索。jpcite の REST API を LINE Messaging API で包んだ bot です。

公開準備中 (2026-05-06 予定)。早期アクセスをメールでお知らせします。

何ができる?

自然文で検索

「東京都の飲食店で使える補助金」

業種・都道府県・制度種別を自動抽出。補助金 11,684 件 + 採択事例 2,286 件 + 融資 108 + 行政処分 1,185 + 税制 50 から、公式 URL 付きで返答。

締切リマインド

応募回 1,106 と連動

気になる制度を「お気に入り」に追加すると、応募開始・締切 3 日前に LINE 通知。JST で時刻を換算。

写真で領収書 OCR (後日)

電帳法 / インボイス連携

領収書を撮影すると適格請求書事業者 (国税庁 13,801 社登録) と照合。T 番号検索も可能。
