jpcite Embed Widget — Demo

本ページは widget の動作確認用です。実際の運用では 製品ページ から申し込んで取得した wgt_live_... キーをサイトに埋め込みます。

Light theme (default)

東京都 / 飲食店 / 設備投資 一次資料 URL、取得時刻、締切候補、相談前に確認する質問を表示する想定です。

このページは静的な表示例です。実 API へのリクエストは発生しません。 実サイトでは発行済み key と許可 origin が一致した場合だけ結果が返ります。

Dark theme

製造業 / 省エネ投資 ダークテーマでも同じ根拠付き候補カードを表示する想定です。

Prefecture filter のみ (コンパクト版)

都道府県だけで絞り込み 小さな相談ページに置くコンパクト表示の例です。

English UI (language=en)

English UI English labels can be used while preserving Japanese primary-source links.

Programmatic (JS API)

Programmatic mount point example. Paste the code below into your own allowlisted origin. 
new Jpcite.Widget({
  container: '#programmatic-target',
  key: 'wgt_live_YOUR_KEY',
  filters: ['industry', 'prefecture'],
  theme: 'light',
  language: 'ja',
  limit: 5,
  onResult: function (p) { console.log('matched:', p.unified_id); }
});

埋め込みスニペット (2 行)

<script src="https://jpcite.com/widget/jpcite.js"></script>
<div data-jpcite-widget data-key="wgt_live_YOUR_KEY"></div>

スニペット 1 枚で検索 UI + 結果リスト + 一次資料リンク + 「powered by jpcite」 フッターが自動で組まれます。
